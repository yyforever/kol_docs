```

└────────────────┬─────────────────────────────┘
                 │ 调用
    ┌────────────┴────────────┐
    │                         │
    ▼                         ▼
┌────────────────┐  ┌────────────────────┐
│ kol-agent      │  │ kol-creator        │

│ $0-199/月 PLG  │  │ 免费+订阅→抽佣      │
└────────────────┘  └────────────────────┘
```
