## 共享底层能力

三条线复用的核心能力（不单独建目录，在各产品线PRD中互相引用）：

- **KOL数据库**（亿级档案、65K niche标签）

- **建联系统**（批量邮件、多渠道触达）
- **AI引擎**（LLM集成、Agent工作流、多语言）
