> 上游文档：[01_达人核心价值网络调研.md](01_达人核心价值网络调研.md)
> 目的：把“帮达人赚钱”拆成不同人群、赚钱路径、链路断点和可介入点。

---

## 1. 结论先行

当前不能再用“达人”作为一个整体人群讨论。至少要分成两条主链路：

1. TikTok Shop Affiliate / 商品带货佣金链路。
2. 品牌内容创作者 / 商单内容采购链路。

这两条链路的共同点是都依赖内容能力，但赚钱机制完全不同。

| 维度 | TikTok Shop Affiliate | 品牌内容创作者 / 商单内容创作者 |
|---|---|---|
| 谁付钱 | 消费者购买商品后产生佣金，或卖家/品牌给补贴 | 品牌、agency 或商家直接采购内容/授权/发布合作 |
| 达人卖的东西 | 内容带来的商品销售可能性 | 内容生产能力、广告素材、授权使用、低风险交付 |
| 关键判断 | 这个商品机会能不能卖，能不能结算 | 品牌为什么选我，这单价格/权利/付款是否合理 |
| 失败位置 | 商品不匹配、listing 不可信、内容不转化、佣金/退款/归因/违规 | 品牌不回复、不选中、低价高工作量、rights 被低估、付款不稳 |
| 第一单价值 | 首单销售、样品资格、账号信号、内容反馈 | 现金/样品收益、真实交付经验、可展示样片、评价、报价依据、联系人 |
| 聚星可能介入 | 机会筛选、匹配解释、风险提示、结算追踪、样品/补贴背书 | AI 商单处理、机会质量解释、品牌筛选解释、报价/rights/付款保护 |

## 2. Segment A：TikTok Shop Affiliate 达人

### 2.1 赚钱链路

```text
满足 creator / affiliate 资格
-> 找到商品、样品或 seller invite
-> 判断是否值得做
-> 拿到样品、自购或直接挂链
-> 制作内容 / live / shoppable video
-> 获得曝光、点击和购买
-> 订单确认、非退款、非违规、正确归因
-> 佣金、样品款或补贴到账
```

### 2.2 当前看到的主要断点

| 断点 | 解释 | 证据状态 |
|---|---|---|
| 样品和首单冷启动 | 无 GMV / 无销量时更难拿样品；没有样品又难做真实内容 | L3 |
| 商品是否值得做 | 高佣金、高播放或高点击都不保证销售 | L3-L4 |
| 商品页是否承接得住 | 价格、评价、物流、listing 信任和视频承诺不一致会吞掉转化 | L3 |
| invite 噪音 | seller/agency invite 很多，但群发、不匹配、低质量 | L3 |
| 机会动态失效 | product/seller eligibility、listing 变更、佣金规则、品牌内容规则会让机会失效 | L2-L3 |
| 到账不确定 | 退款、partial refund、违规追溯、佣金冻结、归因口径、数据异常 | L3 |

### 2.3 判断边界

不能说 TikTok Shop Affiliate 单纯“缺商品机会”。官方 Product Marketplace、Collab Invites、样品机制、seller invite 和外部社区机会都存在。

当前判断：

```text
机会很多，但达人很难判断哪些机会值得投入，
并且机会会在内容制作、购买转化、订单确认、退款和归因阶段动态失效。
```

### 2.4 可能的可介入点

待验证，不是产品方案：

- 过滤低质量 seller invite。
- 解释某商品为什么适合某类 creator。
- 标记商品/listing/物流/评价/饱和度风险。
- 解释样品、refundable sample、PPS、eligibility、退款和归因规则。
- 帮助 creator 记录收益、异常、样品和争议材料。
- 给冷启动商品加固定补贴或真实样品背书，降低 creator 试错成本。

### 2.5 待补证问题

- 高播放不出单的主因到底是内容、商品、listing、受众还是归因。
- creator 拒绝 seller invite 的一手理由。
- 样品/商品机会失效发生在哪个阶段。
- payout / eligibility / attribution 风险是否高频到足以影响中小 creator 持续投入。

## 3. Segment B：品牌内容创作者 / 商单内容创作者

### 3.1 赚钱链路

```text
建立样片、portfolio、定位和报价
-> 通过平台、社区、DM、email、agency 或 inbound 获得品牌需求
-> 品牌判断是否采购
-> 协商 deliverables、价格、rights、whitelisting、exclusivity、revision、deadline
-> 制作和交付内容
-> 品牌验收、付款、投放或发布
-> 沉淀样片、评价、联系人、复购、retainer 或报价依据
```

### 3.2 当前看到的主要断点

| 断点 | 解释 | 证据状态 |
|---|---|---|
| 被品牌选中难 | 品牌收到大量 pitch，需要快速判断 creator 是否靠谱 | L3 |
| 第一单不确定 | 新手缺真实样片、评价、报价依据和品牌背书 | L3 |
| 低价任务边界不清 | 低价不一定错，问题是 scope、rights、revision、付款和后续价值不清 | L3-L4 |
| 报价和 rights 被低估 | usage rights、whitelisting、exclusivity、raw files、retainer 等常被低估 | L3-L4 |
| 付款和 scam 风险 | 不付款、付款慢、合同不清、追款成本高 | L3 |
| 长期关系不足 | 从一次性项目转复购/retainer 的机制还不清楚 | L2-L3 |

### 3.3 判断边界

不能说品牌内容创作者单纯“缺项目”。平台、社区、招募帖、cold outreach 和直接品牌机会都存在。

当前判断：

```text
品牌内容创作者更可能缺的是：
可持续获得、可信、匹配、报酬合理、
能成交并能顺利完成的品牌合作来源，
以及判断这单是否值得继续、是否可能被品牌选中、
是否能顺利交付和到账的依据。
```

样片、定位、pitch、报价、授权、付款、交付可靠性和后续关系都是其中变量，不是单独主痛点。

### 3.4 低价平台和社区的正确位置

低价平台 / 社区不是天然坏，也不是天然好。

可能价值：

- 新手拿第一单。
- 获得真实 brief。
- 练习沟通和交付。
- 获得样片、评价和平台记录。

可能问题：

- 申请成本高。
- 低价高工作量。
- 修改和重拍不清。
- 授权过宽。
- 付款慢或依赖平台状态。
- 不一定带来更好客户。

判断标准不是“价格低不低”，而是：

```text
这单是否钱和样品值得、工作量可控、付款可靠，
并且最好能留下可复用商业证据。
```

### 3.5 可能的可介入点

待验证，不是产品方案：

- 帮新手判断低价任务是否值得做。
- 通过内部商单、Gmail 邀约和外部公开 Offer，验证是否能形成更稳定的待处理商单来源。
- 帮 creator 判断真实合作是否适合自己、是否值得继续沟通。
- 帮 creator 判断已有 Gmail 商单邀约是否真实、是否值得回复、该追问什么。
- 帮 creator 起草回复、报价和条款追问，但必须验证是否真的降低沟通成本。
- 在真实机会前提下，帮 creator 补齐品牌判断所需的必要资料。
- 标记 rights、whitelisting、exclusivity、raw files、revision、付款周期等风险。
- 提供付款/争议/合同/验收的标准化流程。
- 在合作完成后，帮 creator 保留样片、评价、联系人、复购提醒和报价依据。

### 3.6 待补证问题

- 品牌具体如何筛选 creator。
- 第一单来自哪里，做完留下什么。
- 低价任务在哪些条件下是训练场，哪些条件下是陷阱。
- 成熟 creator 如何获得 premium brand deal、retainer、agency roster。
- 自有商单、Gmail 邀约、外部公开 Offer 哪种来源更能产生高质量行动；主动外联后续单独验证。
- creator 是否愿意社媒登录、授权账号数据、导入 Gmail，或使用 AI 回复/报价辅助。
- 品牌内容创作者是否愿意为“获得更靠谱合作”“提高被品牌选中概率”和“保护价格/rights/付款”投入时间或使用工具。

## 4. Segment C：泛小网红 / micro creator

### 4.1 当前定位

暂不作为独立第一主线。

原因：

- 赚钱路径太分散。
- 可能同时包含 TikTok Shop、品牌内容、affiliate、sponsorship、平台流量分成。
- 如果不继续细分，容易得到“达人需要变现工具”这种无用结论。

### 4.2 当前处理方式

把泛小网红作为两条主链路的外延样本：

- 如果主要靠 TikTok Shop 商品佣金，放入 Segment A。
- 如果主要靠品牌采购内容或商单，放入 Segment B。
- 如果靠曝光 sponsorship 或平台分成，需要后续另拆。

## 5. Segment D：MCN / 经纪人

### 5.1 当前定位

不是 `kol-creator` 第一版目标用户，只保留为间接观察视角。

MCN / 经纪人的赚钱逻辑可能是：

```text
管理达人数
× 每个达人可变现机会
× 平均成交率
× 抽佣/服务费
- 机会筛选成本
- 匹配成本
- 沟通成本
- 履约管理成本
- 纠纷成本
```

### 5.2 调研价值

MCN / 经纪人可以帮助我们反向理解网红端问题：

- 达人为什么被品牌选中。
- 哪些任务必须人工服务。
- 报价、合同、追款、复购中哪些环节最难。
- 哪些功能可以工具化，哪些只能靠关系网络。

## 6. 两条主链路的本质差异

### 6.1 购买决策者不同

TikTok Shop Affiliate 的最终购买决策者是消费者。品牌内容创作者的购买决策者是品牌或 agency。

这会导致完全不同的问题：

- TikTok：内容看起来有效，但消费者不买。
- 品牌内容：creator 看起来会拍，但品牌不敢买。

### 6.2 风险来源不同

| 风险 | TikTok Shop Affiliate | 品牌内容创作者 |
|---|---|---|
| 平台规则 | 高 | 中 |
| 佣金/归因 | 高 | 低 |
| 退款/退货 | 高 | 低 |
| rights/授权 | 中 | 高 |
| 不付款 | 中 | 高 |
| scope creep | 低-中 | 高 |
| 品牌选择 | 中 | 高 |

### 6.3 数据价值不同

TikTok Shop Affiliate 需要的数据更偏：

- 商品。
- listing。
- 物流。
- 佣金。
- 价格。
- 历史销量。
- 内容转化。
- 退款/eligibility 风险。

品牌内容创作者需要的数据更偏：

- 品牌预算。
- brief。
- 交付物。
- 使用权。
- 付款周期。
- 同类 creator 案例。
- 样片、报价、过往成交和品牌筛选证据。

## 7. 新手 / 成熟阶段差异

### 7.1 新手

新手的核心通常不是抽象“稳定收入”，而是低风险获得第一批值得做、能完成、能到账的真实合作。

| 链路 | 新手关键问题 |
|---|---|
| TikTok Shop Affiliate | 首单、样品、商品是否值得做、内容是否能转化、invite 是否可信 |
| 品牌内容创作者 | 第一批靠谱合作、价格/样品是否值得、低价任务是否是台阶、品牌为什么选我 |

### 7.2 成熟

成熟 creator 更可能卡在交易质量、复购、授权定价、风险和运营效率。

| 链路 | 成熟关键问题 |
|---|---|
| TikTok Shop Affiliate | 稳定可卖商品、seller/listing 质量、退款/冻结/归因、收入波动 |
| 品牌内容创作者 | 直接品牌/agency、复购/retainer、rights 定价、付款、客户进度管理 |
