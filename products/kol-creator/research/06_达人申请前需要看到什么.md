# 附录：达人申请前需要看到什么

> 创建时间：2026-05-11
> 重构时间：2026-05-13
> 状态：附录 / 暂停扩写
> 上游文档：[03_调研推进计划与净收益公式.md](03_调研推进计划与净收益公式.md)
> 目的：保留单次机会质量判断的信息结构；不作为当前主线，也不用于证明产品形态。

---

## 1. 定位

本文件只回答一个窄问题：

```text
如果未来要展示一个合作机会，
达人至少需要看到哪些信息，才能判断这单是否值得继续了解？
```

它不能回答：

- 第一版产品应该是什么。
- 是否应该做 Offer Board。
- 是否应该做任务市场。
- 达人最核心需求是不是“更多机会”。
- 字段完整是否足以保护达人。

字段只有在能帮助判断以下三件事时才有价值：

1. `选择问题`：这个机会是否适合我，品牌/卖家为什么会选我。
2. `净收益问题`：做完这单后，现金、佣金、样品、权利、工作量、付款和风险综合后是否值得。
3. `复利问题`：这单是否留下样片、评价、品牌背书、联系人、复购或报价依据。

---

## 2. 单次机会判断公式

```text
单次机会净收益 =
    机会可信度
  × 匹配度
  × 转化 / 成交概率
  × 单次真实收益
  × 到账概率
  - 创作成本
  - 沟通成本
  - 风险成本
```

这只是长期收入系统的一环。它不能解释：

- 机会是否稳定出现。
- 是否能形成长期合作。
- 收入是否可预测。
- 达人是否有更高价值机会来源。
- 品牌、卖家或消费者为什么选择这个达人。
- 这次合作是否留下下一次可复用的商业证据。

## 3. TikTok Shop Affiliate：商品 / 样品 / 佣金机会

候选字段，不是定稿。

| 信息 | 判断什么 | 说明 |
|---|---|---|
| 商品 / 品牌名称 | 机会识别 / 信任 | 推广对象是谁 |
| 平台 | 机会类型 | TikTok Shop / Amazon / 其他 |
| 合作类型 | 收益和成本 | 样品 + 佣金、纯佣金、fixed fee + 佣金 |
| 佣金率 / 单件佣金 | 名义收益 | 不能单独决定价值 |
| 固定补贴 / bonus | 风险缓冲 | 用于降低纯佣金不确定性 |
| 样品条件 | 创作成本 / 现金流 | 免费样品、refundable sample、自购、是否必须发内容 |
| 内容要求 | 创作成本 | 视频数量、live、发布时间、claim 限制 |
| 商品价格 / 类目 | 匹配和转化 | 判断受众是否可能购买 |
| Listing 信号 | 转化风险 | 评价、价格、物流、退货、是否过度饱和 |
| 历史销量 / GMV / 转化线索 | 转化概率 | 需要标注来源和可信度 |
| 目标达人画像 | 匹配度 | niche、平台、粉丝量、内容形式 |
| 结算方式 | 到账概率 | 平台佣金、品牌补贴、人工结算 |
| 退款 / partial refund 影响 | 到账风险 | 是否会导致佣金 ineligible |
| eligibility 风险 | 机会稳定性 | product/seller eligibility、VoC/CEI/SPS 等 |
| 申请资格 | 沟通成本 | 粉丝、GMV、内容类目、地区 |
| 反馈时效 | 沟通成本 | 申请后多久有结果 |

### 不能只看

- 高佣金。
- 高播放。
- 商品热榜。
- 样品价值。

这些都不能单独证明机会值得做。

## 4. 品牌内容创作者 / 商单内容创作者：品牌项目机会

候选字段，不是定稿。

| 信息 | 判断什么 | 说明 |
|---|---|---|
| 品牌 / 广告主 | 信任 / 匹配 | 是否真实，是否适合内容定位 |
| 项目目标 | 品牌购买意图 | 买内容、买发布、买广告素材、买长期合作还是买转化 |
| 预算 / fixed fee | 名义收益 | 是否明确付费，金额是否合理 |
| 样品 / 产品价值 | 收益和成本 | gifted 是否足够补偿 |
| Deliverables | 创作成本 | 几条视频/图文/Story/直播 |
| Usage rights | 收益和风险 | 使用多久、用在哪里 |
| Whitelisting / ad usage | 收益和风险 | 是否允许品牌投广告 |
| Exclusivity | 收益和风险 | 独家品类、期限 |
| Raw files / derivative works | 权利范围 | 是否要原始素材、二创、剪辑权 |
| NIL / personality / likeness rights | 人格权风险 | 是否涉及肖像、人格权、AI likeness |
| Revision 次数 | 创作成本 | 修改范围和次数 |
| Scope change 规则 | 成本控制 | 超出 brief 后是否加价 |
| Deadline | 创作成本 | 交付周期 |
| 付款方式 / 周期 | 到账概率 | 预付、里程碑、Net 7/15/30/60 |
| 合同 / 条款状态 | 风险控制 | 是否有合同、模板条款、签署方式 |
| 申请资格 | 匹配和成本 | 平台、粉丝、地区、内容类目 |
| 争议处理 | 到账和风险 | 不付款、返工、取消合作时怎么办 |
| 是否可公开展示 | 可复用证据 | 能否作为 portfolio/case |
| 是否有评价/复购可能 | 复利 | 是否能降低下一次成交阻力 |

### 不能只看

- 有预算。
- 品牌名看起来不错。
- 任务看起来简单。
- 样品免费。

这些都不能证明净收益为正。

---

## 5. 可信度信息

单次机会是否可信，至少要回答：

- 合作方是否真实。
- 预算或佣金是否明确。
- 样品或产品是否真实可得。
- 结算规则是否明确。
- 是否清楚说明达人义务。
- 是否清楚说明品牌使用权。
- 是否有历史成交或其他达人反馈。
- 是否有平台或人工运营背书。
- 是否有明确申请反馈时效。
- 是否有争议处理方式。

---

## 6. 使用边界

这些字段只能降低单次机会判断成本。

它们不能直接证明：

- 达人最缺的是合作机会页面。
- 达人愿意注册一个 Offer Board。
- 达人愿意为字段完整的合作机会付费。
- 单次合作判断比稳定机会来源更重要。
- 字段越多，达人判断越好。
- 信息完整本身足以降低付款、授权、转化或复购风险。
