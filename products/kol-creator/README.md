## 定位

帮网红更好地赚钱。当前阶段先做价值定位调研，不预设产品形态。

更准确的当前研究问题是：

```text
哪一类达人，在哪一种赚钱链路里，缺少什么关键条件，
导致他们无法把内容能力和受众资产稳定转成正收益？
```

## ⚠️ 注意

- 产品定位不等于“达人版 Jungle Scout”，数据只能作为辅助。
- 当前不预设 Offer Board、任务市场、Agent、SaaS 订阅工具或达人版数据分析产品。
- 可以复用 kol-agent / Openclaw 类底层能力，但这不是当前阶段的产品定位前提。
- 品牌上倾向子品牌定位；NoxInfluencer 可作为母品牌背书，不要求完全避开。
