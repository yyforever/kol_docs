## 结论

聚星选星助手定位为聚星在“浏览器插件选号工具”这一层的标准入口，目标是承接个人卖家/小团队在 YouTube、TikTok、Instagram 上的“边刷边选”工作流，并把数据、列表、外联行为回流聚星体系。

当前按 `products/` 下独立目录管理，与 `kol-xxl` 同级；这次迁移不把它并入消消乐，也不预设为新的战略产品线。
