## 背景说明

品牌输入 Campaign brief → Agent 自主完成找人→邀约→谈判→履约→追踪。这条线当前仍是聚星主站中的重要产品方向，但本目录中的旧文档并不自动代表当前 truth。

下面的“核心决策”更多反映早期讨论状态，已经存在一部分与当前方案冲突或过时的假设。阅读时请优先把它们当背景，而不是现行边界。

## 历史决策快照（仅作背景）

| 决策 | 结论 |
|------|------|
| 品牌 | NoxInfluencer / 聚星（不换） |
| 产品形态 | **嵌入聚星平台**（非独立站） |
| 界面 | Dashboard + Chat 混合 |
| 计费 | **纯订阅制**（包 API），$0/~29/~99/定制 |
| Free 层 | 14 天试用 + 1 个完整 Campaign 配额 |
| P0 用户 | 双线并列：中国出海（存量维稳）+ 海外本土（增量优先） |
| MVP 谈判 | AI 起草 + 人工审核 → 快速放开全自动 |
| 部署 | 云端优先，Enterprise 用 AgentPod 独立容器 |
| 与 kol-api | 并行开发，共享 Core 层 |

## 历史业务确认（仅作背景）

| 项 | 确认 |
|-----|------|
| 邮件规模 | ~100 万/月（建联+谈判+合作管理），行业头部 |
| AI 建联 | 做到与人工一致可行，响应率更多取决于"找谁"而非"怎么写" |
| Agent 架构 | **自主 Agent**（类 OpenClaw/Claude Code），无编排框架，LLM + Tools + Context |
| 后端架构 | **独立微服务**，与聚星 Java 后端 API 互调 |
| 邮件基础设施 | 走聚星现有 |
| 嵌入可行性 | ✅ 已确认（不是大问题） |
| 谈判 PoC 数据 | ✅ 历史数据充足 |

## 历史差异化判断（仅作背景）

- **定价基准**：10 年合作数据 → AI 谈判定价基准比竞品更精准
- **客户隔离部署**：AgentPod 多租户，5 个 Agent 竞品零覆盖
- **$29 起步**：比 Stormy $80 便宜 64%
- **双市场**：中国出海 + 海外本土
- **40 万存量**：边际获客成本→0
