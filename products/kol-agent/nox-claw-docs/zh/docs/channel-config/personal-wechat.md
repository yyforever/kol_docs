# 个人微信

本文档说明如何把个人微信接入 NoxClaw。接入完成后，你可以在微信中向 NoxClaw 发送消息，NoxClaw 会把消息同步到当前席位的微信渠道会话中，并把站内回复投递回对应的微信会话。

当前 NoxClaw 个人微信渠道使用 **扫码绑定** 模式。你不需要准备 App ID、App Secret、Token、Encoding-AESKey 或回调 URL；只需要在 NoxClaw 中生成二维码，并使用手机微信扫码即可完成绑定。

## 配置前准备

开始前，请确认你已经具备以下条件：

- 可以登录 NoxClaw，并且当前账号有使用渠道配置的权限。
- 手机上已安装并登录微信。
- 当前网络可以正常访问 NoxClaw 和微信服务。
- 已确认要绑定的微信账号。建议一个微信账号只绑定一个 NoxClaw 席位，避免多个席位共享同一个微信账号导致消息归属混乱。

个人微信绑定依赖二维码扫码确认，请在二维码有效期内完成扫码。如果二维码过期，可以在 NoxClaw 中刷新二维码后重新扫码。

## 一、打开微信 ClawBot 配置

1. 打开 NoxClaw，进入「渠道配置」。
2. 找到「微信 ClawBot集成」卡片。
3. 点击「配置」。
4. NoxClaw 会打开微信配置弹窗，并开始生成用于绑定的二维码。

> 图片占位：NoxClaw 渠道配置页面中的微信 ClawBot集成卡片。

## 二、扫码完成绑定

二维码生成后，打开手机微信并扫描弹窗中的二维码。

1. 使用需要绑定的微信账号扫码。
2. 按微信页面提示确认登录或授权。
3. 回到 NoxClaw，等待渠道状态同步。

扫码成功后，NoxClaw 会记录当前席位与该微信账号的绑定关系。此时渠道可能会先进入「已扫码，等待激活」状态，这是正常状态。

> 图片占位：NoxClaw 微信配置弹窗中的二维码。

## 三、发送首条消息激活会话

扫码成功后，NoxClaw 还需要等到真实微信会话就绪。请在微信中向绑定后的 NoxClaw 入口发送一条任意测试消息，例如：

```text
你好
```

首条消息到达后，NoxClaw 会自动激活当前微信渠道会话，并在渠道会话页面中展示该会话。之后你就可以在 NoxClaw 中查看微信消息并回复。

如果扫码完成后页面一直提示「已扫码，等待激活」，通常表示 NoxClaw 还没有收到来自微信的第一条消息。请先在微信中发送一条消息触发会话创建。

> 图片占位：微信中发送首条测试消息并在 NoxClaw 中生成渠道会话。

## 四、二维码过期和刷新

微信绑定二维码有时效限制。二维码过期后，NoxClaw 弹窗会显示「二维码已过期」。

如果二维码过期或扫码失败，请按以下方式处理：

1. 点击弹窗中的「刷新」。
2. 等待 NoxClaw 生成新的二维码。
3. 使用微信重新扫码。

请不要保存旧二维码反复使用。旧二维码过期后无法继续完成绑定。

> 图片占位：二维码过期后的刷新入口。

## 五、开始使用

绑定并激活会话后，你可以通过微信和 NoxClaw 进行消息往来：

1. 在微信中向已绑定的 NoxClaw 入口发送消息。
2. 回到 NoxClaw 的渠道会话页面，查看微信渠道会话。
3. 在 NoxClaw 中回复该会话。
4. NoxClaw 会把回复投递回对应的微信会话。

个人微信渠道绑定成功只代表微信账号已经和当前席位建立绑定。首次外部消息到达之前，NoxClaw 可能显示已扫码或已连接但暂无会话，这是正常状态。

## 六、重新绑定和解绑

- 如果需要更换微信账号，请先点击「解绑」，再使用新的微信账号重新扫码绑定。
- 如果二维码过期但尚未完成绑定，可以直接刷新二维码，不需要先解绑。
- 如果微信渠道进入异常状态，可以点击「重新绑定」并重新扫码。NoxClaw 会在重新绑定流程中更新当前席位的微信绑定关系。
- 解绑后，NoxClaw 会移除当前席位与该微信账号的绑定关系。后续消息不再进入当前席位的微信渠道会话。

## 七、常见问题

### 1. NoxClaw 需要填写 App ID、App Secret 或回调 URL 吗？

不需要。当前 NoxClaw 个人微信渠道使用扫码绑定模式，不需要开发者凭证，也不需要配置 URL、Token 或 Encoding-AESKey。

### 2. 二维码过期了怎么办？

点击配置弹窗中的「刷新」，生成新的二维码后重新扫码。二维码有时效限制，过期后不能继续使用。

### 3. 扫码后为什么还提示等待激活？

扫码成功只代表微信账号已经完成绑定确认。NoxClaw 还需要收到第一条微信消息才能创建真实渠道会话。请在微信中发送一条任意测试消息，等待 NoxClaw 自动激活会话。

### 4. 微信里发消息没有响应怎么办？

请依次检查：

1. NoxClaw 渠道卡片是否仍处于已绑定、已扫码或已连接状态。
2. 是否使用了扫码时绑定的同一个微信账号发送消息。
3. 二维码是否已经过期，或者绑定流程是否被中断。
4. 是否已经发送首条消息激活会话。
5. 如仍无法恢复，可以先解绑，再重新扫码绑定。

### 5. 可以多个微信账号绑定同一个 NoxClaw 席位吗？

当前建议按一个微信账号绑定一个 NoxClaw 席位使用。如果需要切换微信账号，请先解绑当前账号，再使用新账号扫码绑定。
