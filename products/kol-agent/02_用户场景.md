## 一、品牌的达人营销现状

### 1.1 全链路人工成本分布

品牌激活 50 个达人的完整流程需 **101-175 小时**人力（Influencer Hero）。各环节成本差异巨大：

```
搜索 → 筛选 → 评估 → 邀约 → 谈判 → 执行管理 → 效果追踪
 8%     12%     8%    22%    33%     10%        7%
 └─── 28% ───┘        └── 55% ──┘
  信息密集型              沟通密集型（最烧人力）
```

| 环节 | 占比 | 痛点 | 量化数据 | 来源 |
|------|:----:|------|---------|------|
| 搜索 | ~8% | 跨平台手动搜索，无结构化对比 | 人工搜索 3-5 天/轮 | 行业调研 |
| 筛选 | ~12% | 假粉泛滥，肉眼无法判断 | **49% IG 达人存在某种程度欺诈** | HypeAuditor |
| 评估 | ~8% | 受众画像需多源交叉验证 | 79% 营销人员认为 ROI 归因是最大挑战 | eMarketer |
| **邀约** | **~22%** | 响应率低，联系方式难找 | 需发 **333+ 封邮件**激活 50 人，邮箱覆盖率仅 25-40% | Influencer Hero |
| **谈判** | **~33%** | 无标准定价，报价不透明 | **87.9%** 营销人员说谈判越来越难，**58%** 达人要求混合报酬，平均每个达人 **6 轮邮件往返** | Modash 2025 |
| 执行管理 | ~10% | 物流/审稿/合规/付款分散 | 各环节分散在不同工具和表格 | 聚星客户反馈 |
| 效果追踪 | ~7% | ROI 未知 + 曝光量不足 | — | 聚星客户反馈 |

**核心发现**：邀约+谈判占 **55%** 人工时间，是品牌最大的成本黑洞——也是 Agent 的核心战场。

### 1.2 两条产品线的视角差异

同样的全链路痛点，kol-api 和 kol-agent 从**完全不同的视角**去解决：

| 维度 | kol-api（能力层） | kol-agent（本项目） |
|------|-------------------|-------------------|
| **本质** | 被其他 Agent 调用的 Tool | 品牌直接使用的 AI 员工 |
| **用户** | 开发者 / 其他 Agent 平台 | 品牌广告主 |
| **交互** | API 调用 / MCP / Skill | Dashboard + Chat + 审批流 |
| **体验** | 品牌通过 Agent 间接使用 | 品牌在聚星平台直接使用 |
| **上手成本** | 需要 Agent 环境或 API Key | 聚星账号即可，零技术门槛 |
| **Agent 自主性** | 取决于第三方 Agent 平台 | **我们定义 Agent 能力和权限** |
| **审批** | Agent 平台决定交互方式 | **我们设计 Human-in-the-loop** |
| **关系** | 共享 Core 层 | 共享 Core 层 |

**关键洞察**：kol-api 把"怎么用"交给了第三方 Agent 平台——品牌体验不受我们控制。kol-agent 把全流程攥在自己手里——从发现到付费的每一步都由我们设计。

### 1.3 中国出海品牌 vs 海外本土品牌

> **优先级（2026-02-25 确认）**：双线并列，海外本土品牌优先级稍高。中国出海客户基本已是聚星存量，增长空间有限，正常服务即可；海外本土是主要增量方向。

两类 P0 客户的痛点差异决定了 Agent 的差异化交互设计：

| 维度 | 中国出海品牌 | 海外本土品牌 |
|------|------------|------------|
| **团队** | 2-5 人，海外 0-3 人 | 1-3 人，创始人兼做营销 |
| **核心痛点** | 跨文化沟通障碍、语言不自信、达人不信任中国品牌邮件 | 时间不够、全手动 3-5 天找人、假粉泛滥 |
| **偏好** | 效果导向（CPA/ROI）、铺量 | 品牌建设、长期关系、调性匹配 |
| **平台** | TikTok Shop 为主 | Instagram/YouTube/TikTok 均衡 |
| **Agency 依赖** | 高（缺本地资源） | 低（趋势 in-house） |
| **决策速度** | 快（ROI 导向） | 中等（需要"感觉对"） |
| **Agent 接受度** | ⭐⭐⭐⭐⭐（已习惯工具化运营） | ⭐⭐⭐⭐（开放但需建立信任） |
| **获客成本** | **趋近于零**（聚星 40 万存量） | 需通过聚星平台注册引流 |
| **Agent 特别需求** | 多语言邀约模板、跨文化谈判策略、本地化内容审核 | 品牌调性匹配、个性化邀约、简单易用 |

> 数据来源：R4 场景画像、用户画像研究

## 二、用户旅程（从发现到付费）

### 2.1 发现路径

kol-agent 嵌入聚星平台，发现路径分两条线：

#### 路径 A：存量客户转化（P0，最优先）

```
聚星 40 万存量客户 → 登录聚星平台 → 看到 Agent 新栏目（首页 Banner + 侧边栏入口）
  → 点击进入 → 看到 "创建你的第一个 AI Campaign（免费）" 引导
  → 10 秒内创建第一个 Campaign → 体验全流程 → 转化付费
```

优势：
- 零获客成本
- 用户已有聚星账号、历史数据、支付方式
- 可直接复用聚星已有的达人收藏夹、合作历史
- **平滑过渡**：品牌可以在"手动操作"和"Agent 自动"之间无缝切换

#### 路径 B：海外新客户注册（P0）

```
Google/社媒内容 → 聚星官网（NoxInfluencer.com）→ 注册（邮箱/Google SSO）
  → 进入聚星平台 → Agent 栏目引导 → 创建第一个免费 Campaign
```

聚星平台对海外品牌注册无门槛，Agent 是平台功能之一——新用户可以同时使用聚星传统功能和 Agent。

#### 路径 C：口碑与内容营销（P1）

```
看到案例/教程/推荐 → 搜索 "NoxInfluencer Agent" → 官网注册 → 使用
```

### 2.3 付费转化时刻

品牌在以下时刻感受到最强付费动力：

| 时刻 | 触发 | 心理 |
|------|------|------|
| **第一个 Campaign 完成** | 看到全流程效果：找人→邀约→谈判→达成合作 | "这比手动快 10 倍，下一次也要用" |
| **Free 配额用完** | 想创建第二个 Campaign | "才 $29/月，比实习生便宜 100 倍" |
| **谈判省钱** | AI 把 $1,200 谈到 $850，比要价降 29% | "AI 一次谈判省的钱就超过月费" |
| **规模化需求** | 从 10 个达人扩展到 50-200 个 | "需要更多配额" |

**定价锚点**（01 决策 D8）：

| 替代方案 | 品牌当前成本 | 聚星 Agent |
|---------|-----------|-----------|
| 专职达人营销经理 | $83K-120K/年 | **$29-199/月** |
| 传统 SaaS（GRIN 等） | $5K-30K/年 + 大量人力 | Agent 自主执行，人只审批 |
| Stormy AI | $170/月，不支持 IG | 全平台 + 亿级数据 + 更低价 |
| Aha | 按交付付费，无 PLG | PLG 自助 + Enterprise 客户隔离 |

### 2.4 留存驱动力

| 驱动力 | 说明 | 量化预期 |
|--------|------|---------|
| **全链路省时** | 从 101-175 小时 → 数小时（人工仅做审批） | 节省 90%+ 人力 |
| **数据信任** | 亿级达人库 + 假粉检测 + 受众画像 + 定价基准 | 基于 10 年数据飞轮 |
| **越用越准** | Agent 记住品牌偏好、合作历史、达人黑白名单 | 第 3 次 Campaign 起明显 |
| **不可替代性** | 大规模邮件邀约 + AI 自动谈判在市场上极度稀缺 | 有验证的直接竞品仅 Stormy/Aha/AMT 三家（图灵和 Scrumball 尚无产品验证） |
| **切换成本** | 达人关系数据、合作历史、谈判记录沉淀在平台 | 使用 3 个月后切换成本高 |

## 三、5 个核心使用场景

### 场景 1：小品牌首次铺量（P0）

> **触发条件**：新品上市或新市场拓展，1-3 人团队，无达人营销经验，预算 $5K-20K/Campaign

#### 品牌画像

- 年收入 $50 万-$500 万的 DTC 品牌
- 创始人/运营负责人兼做营销
- 没有达人资源积累，不知道从哪开始
- 中国出海 or 海外本土均适用

#### 节点 6 + 7：谈判策略确认 + 谈判执行

```
交互方式：Dashboard 谈判面板 + Chat 实时同步
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

策略确认（节点 6）：
- Agent 展示：达人要价、市场基准、建议策略、预算范围
- 品牌设定：预算上限、可接受的合作形式、底线
- 一旦确认，Agent 开始谈判

谈判执行（节点 7）MVP 模式：
- Agent 起草每轮回复 → 品牌审核 → Agent 发送
- Dashboard 展示谈判进程时间线
- 品牌可随时干预（修改回复内容、调整策略、终止谈判）

谈判执行 成熟模式：
- 品牌设定预算规则后，Agent 全自动执行
- 仅在以下情况暂停请求人工介入：
  · 达人要价超预算上限 30%+
  · 达人提出非标准条款（独家、使用权等）
  · 谈判超过 5 轮未达成一致
  · 品牌主动设定的特殊触发条件

超时策略：达人回复后 4 小时内未审核 → Chat 紧急提醒
         8 小时未审核 → Agent 发送"稍后回复"的临时邮件（维持沟通温度）
         24 小时未审核 → 通知品牌可能错失机会
```

### 4.3 审批超时策略总览

| 审批节点 | 首次提醒 | 二次提醒 | 超时处理 |
|---------|:-------:|:-------:|---------|
| 达人列表确认 | 24h | 48h | 72h Campaign 暂停 |
| 邮件发送审批 | 12h | 24h | 48h 保存草稿，Campaign 暂停 |
| 谈判策略确认 | 4h | 8h（发临时邮件） | 24h 通知错失风险 |
| 谈判回复审核（MVP） | 4h | 8h（发临时邮件） | 24h 通知错失风险 |
| 续约审批 | 7 天 | 14 天 | 到期自动标记"未续约" |

**设计原则**：
- **不替品牌做决定**——超时后暂停，不自动执行
- **维持沟通温度**——在达人侧发送"稍后回复"的临时邮件，避免品牌因审批延迟而失去达人
- **多渠道提醒**——Chat 内通知 → 平台通知 → 邮件

### 4.4 渐进式放权机制

**核心理念**：从"每次审核"逐步过渡到"规则授权自动执行"，信任通过实际使用建立。

```
Level 0：全审核模式（新用户默认）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
每个操作都需要品牌确认：
达人列表 → ✋审核 → 邮件 → ✋审核 → 谈判策略 → ✋审核 → 每轮回复 → ✋审核

Level 1：模板授权模式（使用 2-3 个 Campaign 后解锁）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
品牌审核邮件模板 → Agent 按模板批量发送（无需逐封审核）
达人列表 → ✋审核 → 邮件模板 → ✋审核 → 批量发送 → 自动

Level 2：规则授权模式（使用 5+ 个 Campaign 后解锁）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
品牌设定规则：
- "评分 > 85 的达人自动加入列表"
- "预算内的邀约自动发送"
- "谈判在预算范围内自动执行，超预算再问我"

达人搜索 → 自动 → 邮件发送 → 自动 → 谈判 → 自动（预算内）/ ✋审核（超预算）

Level 3：全自动模式（Enterprise / 高信任用户）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Agent 完全自主运行，品牌只看报告和处理异常：
全流程自动 → 品牌收到周报/异常告警 → 只在异常时介入
```

**解锁条件**（非自动，品牌手动开启）：

| Level | 解锁条件 | 设置位置 |
|:-----:|---------|---------|
| 0 | 默认 | — |
| 1 | 完成 2+ 个 Campaign + 品牌手动开启 | Dashboard 设置 → Agent 权限 |
| 2 | 完成 5+ 个 Campaign + 品牌设定规则 | Dashboard 设置 → 自动化规则 |
| 3 | Enterprise 层 + 品牌书面确认 | 客户成功团队协助 |

### 5.4 我们的差异化定位

```
Stormy = "AI 辅助的达人营销工具"（5 个 AI Agent 帮你做事）
  → 偏 Agent 集合，品牌参与度较高

Aha = "全链路 AI 员工"（品牌只做 3 件事）
  → 最激进的自动化，但无 PLG、客群收窄至"Built for AI companies"

聚星 Agent = "嵌入聚星的全流程 AI 员工 + 渐进式放权"
  → 品牌从全审核开始，逐步放权到全自动
  → 既服务信任度低的新用户（Level 0），也服务高信任度的老客户（Level 3）
  → 嵌入已有平台 = 存量客户零获客成本
  → 亿级数据 + 客户隔离 = 竞品不可复制的壁垒
```

**核心差异总结**：

1. **渐进式信任**（vs Aha 的一步到位）：不强制品牌信任 AI，而是让品牌自己决定放权节奏
2. **嵌入式分发**（vs 独立产品）：40 万存量客户直接转化，边际获客成本→0
3. **Dashboard + Chat 混合**（vs 纯 Chat 或纯 SaaS）：Chat 发起指令，Dashboard 展示数据和审批
4. **Enterprise 客户隔离**（竞品零覆盖）：AgentPod 独立容器，MCN/Agency 刚需
5. **定价碾压**：$29 vs Stormy $80（便宜 64%），且功能更全
