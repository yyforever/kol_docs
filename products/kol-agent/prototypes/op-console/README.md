# OpenClaw Control Plane Demo

这是一个零依赖的静态网页 demo，用来演示“官方 OpenClaw + 自研控制面”的后台管理系统形态。

## 打开方式

直接双击打开 `index.html` 即可。

如果你希望按站点方式访问，也可以在目录里启动一个本地静态服务，例如：

```powershell
cd C:\Users\Nox306\openclaw-control-plane-demo
python -m http.server 8080
```

然后访问 `http://127.0.0.1:8080/`。

## 可演示的交互

- 在 `租户管理` 中创建租户，查看租户详情抽屉
- 在 `实例管理` 中创建实例、重启实例、销毁实例、加入升级篮
- 在 `批量升级中心` 中创建升级批次，推进或完成批次
- 在 `Token 策略` 中调整租户的 token 上限
- 在 `迁移中心` 中创建迁移任务，并推进或完成迁移
- 在 `控制面总览` 中通过告警卡片快速跳转到对应页面

## 说明

- 所有数据都存放在浏览器内存中，刷新页面会恢复为初始 mock 数据
- 这是一个领导演示版原型，不包含真实后端、不落库、不调用 OpenClaw API
