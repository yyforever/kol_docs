## 一、订阅制核心理念

### 1.1 三个关键词

| 关键词 | 含义 |
|--------|------|
| **即开即用** | 客户付订阅费用，获得一个立即可用的 AI 员工。不需要接入、部署、配置任何技术细节 |
| **隐藏复杂性** | 客户不需要知道 Agent 跑在哪、用什么模型、数据存在哪里、邮件系统怎么对接。这些都是我们的事 |
| **按需扩展** | 配额用完不断服务——提醒、降速、升级或买扩展包，客户自己选择 |

### 1.2 设计原则

1. **功能完全一致**：所有付费层级（Starter/Pro/Enterprise）提供**完全相同的功能**——搜索、邀约、谈判、CRM、报表、续约管理等。没有"功能分层"，只有配额差异
2. **技术细节对用户透明**：部署模式（共享/独立容器）、模型选型、Core 层架构等不暴露给用户
3. **配额是柔性上限**：正常使用足够，超出后提醒 → 降速 → 暂停，不是硬切断
4. **订阅包含 API 使用**：kol-agent 用户内部调用 Core 层（kol-api）不额外计费
5. **简单可预期**：月付/年付固定费用，品牌可以预算

## 二、层级设计

### 2.1 层级总览

| 层级 | 月费（待校准） | 年付（待校准） | 目标用户 | 核心定位 |
|------|:----:|:----:|---------|---------|
| **Free 试用** | $0 | — | 首次体验 | 给够跑完 1 个完整 Campaign，14 天试用期 |
| **Starter** | ~$29 | ~$24/月（省 ~17%） | 小品牌 / DTC / 个人卖家 | 完整功能 + 基础配额，满足日常使用 |
| **Pro** | ~$99 | ~$79/月（省 ~20%） | 中型品牌 / 成长期电商 | 更大配额 + 更多并发 Campaign |
| **Enterprise** | 定制 | 年约 | MCN / Agency / 大品牌 | 高配额 + 独立数据隔离 + SLA + 定制 |

> **所有价格标注"待校准"**——需要 Beta 期真实使用数据来确定最终定价。当前价格是基于竞品对标和成本分析的锚定参考。

### 2.2 Free 试用策略

| 维度 | 设计 |
|------|------|
| **配额** | 足够跑完 **1 个完整 Campaign** 的全部流程（搜索 + 筛选 + 邀约 + 谈判） |
| **时间限制** | **14 天** |
| **为什么不永久免费** | 避免资源空转（LLM 调用、邮件发送均有成本），限时更能驱动决策 |
| **转化设计** | 第 1 个 Campaign 跑通 → 品牌看到全流程效果 → 自然付费 |
| **到期处理** | 数据保留 30 天（可随时激活付费恢复），不删除品牌记忆和 CRM 数据 |

> 01 D9 决策：让品牌走完全流程是最强转化。试用不是"试一下功能"，而是"完整跑一遍"。

### 2.3 Starter（~$29/月）

| 维度 | 设计 |
|------|------|
| **目标** | 小品牌 / DTC 卖家，1-3 人团队 |
| **核心价值** | "~$29/月替代 3-5 天手动找人 + 10-20 小时邮件邀约" |
| **功能** | **完整功能**——搜索、邀约、谈判、CRM、报表、续约管理，全部可用 |
| **配额级别** | 基础——满足每月 1-3 个 Campaign 的正常运营 |
| **竞品锚点** | Stormy $80（便宜 64%）、Euka $199（便宜 85%） |

### 2.4 Pro（~$99/月）

| 维度 | 设计 |
|------|------|
| **目标** | 中型品牌，2-5 人营销团队 |
| **核心价值** | "~$99/月替代 $2-8K/月 Agency 或 $83K/年专职经理" |
| **功能** | **完整功能**（与 Starter 完全一致） |
| **配额级别** | 放大——更多并发 Campaign（~10 个）、更多邮件、更多达人搜索 |
| **竞品锚点** | Euka $199（便宜 50%）、Stormy $170（便宜 42%） |

### 2.5 Enterprise（定制）

| 维度 | 设计 |
|------|------|
| **目标** | MCN / Agency / 大品牌，服务 5-20 品牌客户 |
| **核心价值** | "独立 Agent 空间替代每个品牌独立的 SaaS 订阅（$5K-30K/年/品牌）" |
| **功能** | **完整功能** + 以下增值 |
| **增值** | 独立数据隔离（AgentPod）、SLA 保障、定制 Agent 人格、私有化部署选项、品牌白标、专属客户成功 |
| **定价模式** | 年约，定制报价（Sales 主导） |
| **最低预期** | ~$500+/月（私有化更高） |
| **竞品锚点** | GRIN $2,000+/月（聚星 Enterprise 定价显著更低） |

## 三、配额维度设计

### 3.1 配额维度

所有层级共享以下配额维度，区别在于各维度的具体数值：

| 配额维度 | 说明 | 设计逻辑 |
|---------|------|---------|
| **并发 Campaign 数** | 同时运行的 Campaign 上限 | 核心限制维度——直接决定使用规模 |
| **月达人搜索次数** | 每月可执行的搜索请求数 | 每次搜索消耗 LLM + 数据查询资源 |
| **月邀约邮件量** | 每月可发送的邀约邮件数（含 follow-up） | 邮件发送有硬成本（聚星邮件基础设施） |
| **月 AI 谈判轮次** | 每月可执行的 AI 谈判轮次 | 每轮谈判消耗 LLM 推理 + 邮件 |
| **CRM 达人容量** | CRM 达人池中可管理的达人总数 | 存储和检索成本 |
| **月 Agent 对话轮次** | 每月品牌与 Agent 的对话轮次 | LLM 推理成本 |
| **月效果报表** | 每月可生成的效果分析报表数 | LLM + 数据分析 |
| **存储空间** | 品牌记忆 + Campaign 数据的存储量 | 存储成本 |

### 3.2 配额参考范围（待校准）

| 维度 | Free 试用 | Starter | Pro | Enterprise |
|------|:--------:|:-------:|:---:|:----------:|
| 并发 Campaign | 1 | ~3 | ~10 | 定制（50+） |
| 月达人搜索 | ~10 次 | ~50 次 | ~200 次 | 定制 |
| 月邀约邮件 | ~30 封 | ~200 封 | ~1,000 封 | 定制 |
| 月 AI 谈判轮次 | ~20 轮 | ~100 轮 | ~500 轮 | 定制 |
| CRM 达人容量 | ~50 | ~500 | ~5,000 | 定制 |
| 月 Agent 对话 | ~50 轮 | ~500 轮 | ~2,000 轮 | 定制 |
| 月效果报表 | 1 | ~5 | ~20 | 定制 |
| 存储空间 | 100MB | 1GB | 10GB | 定制 |

> ⚠️ **所有数值标注"待校准"**——以上为初始设计参考，需要 Beta 期真实使用数据确定最终值。设计逻辑：Free 试用够跑完 1 个 Campaign，Starter 够日常小品牌使用，Pro 覆盖中型品牌规模化运营。

### 3.3 配额计算逻辑

一个典型的小品牌 Campaign 流程消耗估算：

| 环节 | 动作 | 配额消耗 |
|------|------|---------|
| 搜索 | 1-3 次搜索请求 | 搜索 1-3 次 |
| 筛选 | Agent 分析 15-30 个达人 | Agent 对话 5-10 轮 |
| 邀约 | 发送 10-15 封邮件 + 5-10 封 follow-up | 邮件 15-25 封 |
| 谈判 | 5-8 个达人，每个 3-5 轮 | 谈判 15-40 轮 |
| 管理 | 合作确认、进度跟踪 | Agent 对话 10-20 轮 |
| **单 Campaign 总计** | — | **搜索 ~3 + 邮件 ~20 + 谈判 ~25 + 对话 ~20** |

Starter 配额（~3 并发 Campaign）可支撑 **每月 3 个完整 Campaign**，对小品牌日常运营足够。

## 四、计费模式

### 4.1 月付 / 年付

| 模式 | Starter | Pro | Enterprise |
|------|:-------:|:---:|:----------:|
| **月付** | ~$29/月 | ~$99/月 | — |
| **年付** | ~$24/月（省 ~17%） | ~$79/月（省 ~20%） | 年约定制 |
| **支付方式** | 聚星现有支付系统 | 同左 | 线下合同 |

> 年付折扣吸引长期客户，提升 LTV。Enterprise 默认年约。

### 4.2 超出配额处理

分三阶段渐进处理，不硬切断：

```
阶段 1：提醒（配额使用 80%）
━━━━━━━━━━━━━━━━━━━━━━━━━━━
Agent：你本月的邀约邮件配额已使用 80%（160/200 封）。
      还有 40 封可用，预计可支撑 2 个新 Campaign。
      需要升级到 Pro 获得更多配额吗？

阶段 2：降速（配额使用 100%）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Agent：你本月的邀约邮件配额已用完。
      当前 Campaign 中已发送的邮件照常跟踪回复和谈判。
      新的邀约邮件将暂停，直到配额刷新（下月 1 日）或升级/购买扩展包。

      [🔼 升级到 Pro] [📦 购买扩展包] [⏳ 等待下月]

阶段 3：暂停新操作（所有关键配额用完）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Agent：你本月的配额已全部用完。进行中的 Campaign 照常跟踪。
      新的搜索、邀约和谈判操作已暂停。
```

**原则**：进行中的合作不受影响（已发邮件继续跟踪、正在谈判的继续完成）。只暂停**新操作**的发起。

### 4.3 按需扩展包

对于不想升级订阅等级、但偶尔需要额外配额的用户：

| 扩展包 | 内容 | 价格（待校准） |
|--------|------|:----:|
| **邮件加量包** | +200 封邀约邮件 | ~$10 |
| **Campaign 加量包** | +3 个并发 Campaign（本月有效） | ~$15 |
| **谈判加量包** | +100 轮 AI 谈判 | ~$10 |

> 扩展包按需购买，当月有效，未用完不退不滚。价格待校准。

## 五、与 kol-api 的计费关系

### 5.1 内部调用不计费

```
kol-agent 用户（品牌）
    │
    ├── 在聚星平台内使用 Agent → 消耗 kol-agent 订阅配额
    │                           → 内部调用 Core 层（kol-api）
    │                           → ⚠️ 不消耗 kol-api credit
    │
    └── kol-api 独立用户（开发者/第三方 Agent 平台）
                                → 消耗 kol-api credit（独立计费）
```

### 5.2 设计理由

- **用户体验**：品牌不需要理解"API credit"的概念，一个订阅费搞定一切
- **简单可预期**：月费固定，不会因为 Agent 多调了几次 API 而产生意外费用
- **不挤压 kol-api**：kol-agent 的 Core 层调用是内部成本，不走 kol-api 的 credit 体系
- 两条线独立计费，独立核算

## 七、Enterprise 定价设计

### 7.1 增值服务

Enterprise 在**完整功能**基础上提供以下增值：

| 增值项 | 说明 | 成本驱动 |
|--------|------|---------|
| **独立数据隔离** | AgentPod 独立容器，品牌间物理隔离 | ~$5-15/月/品牌（服务器成本） |
| **SLA 保障** | 99.5%+ 可用性，响应时间承诺 | 运维成本 |
| **定制 Agent 人格** | 自定义 SOUL.md——Agent 的沟通风格、判断标准 | 配置成本低 |
| **私有化部署** | 容器导出到客户自有服务器 | 部署 + 支持成本 |
| **品牌白标** | 去掉"NoxInfluencer"品牌，替换为客户品牌 | 前端改动 |
| **专属客户成功** | 专人对接，定期 review | 人力成本 |
| **API 集成** | 对接品牌内部系统（审批、ERP、CRM 等） | 开发成本 |

## 八、Free 试用详设

### 8.1 试用配额

| 维度 | 配额 | 理由 |
|------|------|------|
| 持续时间 | **14 天** | 足够跑完 1 个 Campaign 全流程（搜索→邀约→等回复→谈判→确认） |
| 并发 Campaign | **1** | 体验完整流程 |
| 搜索次数 | **~10 次** | 搜索 + 调整条件 |
| 邀约邮件 | **~30 封** | 邀约 10-15 人 + follow-up |
| AI 谈判 | **~20 轮** | 与 3-5 个达人完成谈判 |
| CRM 达人容量 | **~50** | 体验 CRM 功能 |
| Agent 对话 | **~50 轮** | 充分互动 |

### 8.2 试用到期流程

```
Day 1:  注册 → 创建 Campaign → 开始使用
Day 10: Agent 提醒："试用还剩 4 天，你的 Campaign 进展很好！"
Day 13: Agent 提醒："明天试用到期。要继续使用吗？升级到 Starter 仅需 ~$29/月"
Day 14: 试用到期 → 新操作暂停，进行中的 Campaign 照常跟踪
Day 14+: 品牌数据保留 30 天，可随时激活付费恢复
Day 44: 数据过期（通知后删除）
```

### 8.3 转化优化

| 转化时刻 | 触发 | 品牌心理 |
|---------|------|---------|
| **第一个 Campaign 跑通** | 看到全流程效果 | "这比手动快 10 倍" |
| **AI 谈判省钱** | 把 $1,200 谈到 $850 | "AI 一次省的钱就超月费" |
| **试用即将到期** | Agent 提醒 + 展示已积累的数据 | "不续费就白费了这些工作" |
| **配额用完** | 想做更多但被限制 | 自然升级 |

## 九、成本结构分析

### 9.1 主要成本项

| 成本项 | 估算 | 说明 |
|--------|------|------|
| **LLM 推理** | ~$0.01-0.05/轮对话 | Claude/GPT-4 级别，随模型降价持续下降 |
| **邮件发送** | ~$0.001-0.005/封 | 聚星现有基础设施边际成本极低 |
| **达人数据查询** | 已有成本（聚星核心资产） | 不额外增加 |
| **存储** | ~$0.02-0.05/GB/月 | 云存储 |
| **服务器（共享）** | ~$1-3/品牌/月 | Starter/Pro 共享实例分摊 |
| **服务器（独立）** | ~$5-15/品牌/月 | Enterprise AgentPod 容器 |

### 9.2 单用户月成本估算（Starter 典型使用）

| 项目 | 月用量 | 单价 | 月成本 |
|------|--------|------|:------:|
| Agent 对话 | ~300 轮 | ~$0.02 | ~$6 |
| 邮件发送 | ~100 封 | ~$0.003 | ~$0.3 |
| 搜索请求 | ~30 次 | ~$0.05 | ~$1.5 |
| 存储 | ~500MB | ~$0.03 | ~$0.02 |
| 服务器分摊 | — | — | ~$2 |
| **合计** | — | — | **~$10** |

**Starter 月费 ~$29，毛利率 ~65%**。Pro 和 Enterprise 毛利率更高（配额放大但成本不等比增加）。

### 9.3 规模效应

| 指标 | 100 用户 | 1,000 用户 | 10,000 用户 |
|------|:--------:|:----------:|:-----------:|
| 服务器成本/用户 | ~$3 | ~$2 | ~$1.5 |
| LLM 成本/用户 | ~$8 | ~$6（批量折扣） | ~$4 |
| 毛利率 | ~60% | ~70% | ~80% |
