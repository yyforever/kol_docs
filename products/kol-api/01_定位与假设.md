## 一、定位

### 一句话

**NoxInfluencer 是 AI 驱动的达人营销全链路自动化——让 Agent 替品牌完成从找人到谈判到管理的完整流程。**

### 品类

不是"达人数据开发者 API"，是 **AI Agent 达人营销自动化平台**。

### 类比

Janney AI / Stormy AI 之于达人营销 = NoxInfluencer 之于 AI Agent 生态的达人营销能力。

- 在各 Agent 平台上可被自动发现和调用，不做自己的 Agent 平台
- 覆盖全链路：搜索→评估→邀约→谈判→管理→追踪
- 品牌用户能感知到"用了 vs 没用"的效率差异（手动 3-5 天 vs 30 秒）
- 多平台分发，不绑定任何一家

### 定位翻转

| 维度 | 旧定位（占位版） | 新定位 | 翻转原因 |
|------|----------------|--------|---------|
| **客户** | 开发者 | 品牌广告主 | 品牌有营销预算，付费意愿和能力远高于开发者 |
| **交付方式** | REST API + SDK | Agent 平台 Skill/MCP | 品牌通过 Agent 使用能力，不需要写代码 |
| **体验标准** | DX（文档/SDK/错误码） | 全链路效率（找人→谈判→管理） | 品牌在意"帮我全搞定"，不在意 API 设计 |
| **竞品** | Modash API / CreatorDB | 人工调研 / MCN / SaaS / Janney AI / Stormy AI | 品牌的替代方案不是别的 API，是别的工作方式；AI 邀约+谈判赛道有 3 家新玩家 |
| **分发** | RapidAPI / MCP 目录 | Agent 自动发现 + Agent 商店 | 品牌不逛商店，靠 Agent 自动发现并调用 |

## 二、市场判断

定位基于三个判断。前两个有数据支撑，第三个是趋势推断——也是最大的不确定性。

### 判断 1：AI Agent 正在成为新的办公入口

置信度：**高**

事实依据：
- OpenClaw 2025-11 启动，2026-01 48 小时内 9K→106K Stars，目前 157K+
- ClawHub 10 天内从 0 增至 5,700+ Skill，日增 500+
- Manus $100M ARR，Meta $2B 收购进行中
- Claude Code、ChatGPT、Copilot 均在构建 Agent 商店/插件体系
- 非技术岗位（营销、运营、HR）开始使用 Agent 处理日常工作

### 判断 2：达人营销品类在 Agent 生态中是空白

置信度：**高**

事实依据：
- 6 个超级 Agent 商店（OpenClaw/Claude/ChatGPT/Copilot/Manus/Google），达人营销工具数均为 **零**
- MCP 目录 17,000+ server 中，Marketing 仅 9 个，达人数据专业工具 0 个生产级可用
- ClawHub Marketing 品类 700+ Skill，达人数据方向几乎空白
- 综合缺口率 > 99%

> 详见 `竞品分类图谱.md` 第六节

### 判断 3：品牌广告主会通过 Agent 做达人营销

置信度：**中**——这是本文最关键也最不确定的判断。

支撑逻辑：
- 达人营销全链路人工成本极高：激活 50 个达人需 101-175 小时人力
- 信息密集型（搜索+筛选+评估）占 28%，**沟通密集型（邀约+谈判）占 55%**——后者才是品牌最大的成本黑洞
- 这两类工作都适合 AI 处理：信息密集型靠数据检索，沟通密集型靠 LLM 生成+自动邮件
- Agent 是解决这些痛点的自然界面：一句话描述需求 → 即时获得结构化结果 → 一键批量邀约 → AI 代谈

反面论据：
- 目前没有公开数据显示品牌广告主正在用 Agent 做达人筛选
- Agent 办公习惯尚在早期，非技术岗位的渗透率未知
- 达人营销决策往往涉及主观判断（品牌调性、内容风格），AI 能否胜任存疑

**结论：这是一个趋势预判，不是已验证的需求。NoxInfluencer 是先发卡位，不是追随需求。**

## 三、客户

### 核心客户

使用 AI Agent 办公的**品牌广告主 / 营销团队**。

画像：
- 有达人营销预算（年 10 万-数百万）
- 已在使用 OpenClaw / Claude / ChatGPT 等 Agent 平台
- 日常需要搜索、筛选、评估达人
- 对 MCN 推荐的中立性有疑虑，或不愿为 SaaS 仪表盘付年费

### 核心场景

| 优先级 | 场景 | 用户对 Agent 说 | NoxInfluencer 提供 |
|:------:|------|---------------|-------------|
| P0 | 达人发现 | "帮我找 10 个美国美妆类 TikTok 达人，粉丝 5-20 万" | 秒级结构化达人清单 + 假粉检测 + 受众画像 |
| P0 | 邀约触达 | "帮我邀约这 12 位达人，推广我们新产品" | 大规模个性化邮件发送 + 响应追踪 + 自动跟进（两阶段确认：预览→执行） |
| P0 | 谈判协商 | "这个达人要价 $1,200，帮我谈到 $800" | AI 全自动代谈（在预算范围内邮件往返）+ 市场定价基准（两阶段确认：策略预览→启动谈判） |
| P1 | CRM 管理 | "我之前合作过哪些健身达人？" | 合作历史 + 达人白/黑名单 + 主动提醒 |
| P1 | 合作管理 | "这个达人的合作进展怎么样了？" | 全流程状态追踪（邀约→谈判→合同→发货→发布→结算） |
| P1 | 竞品对标 | "竞品最近合作了哪些达人？效果如何？" | 品牌合作历史 + 赞助内容识别 |
| P2 | 效果追踪 | "这轮 Campaign 的 ROI 是多少？" | 实时数据 + 效果汇总 |

> 场景优先级依据见 `02_用户场景.md` 第一节——邀约+谈判占全链路 55% 人工成本，是付费级差异化的核心

### v1 不服务

| 需求 | 原因 | 替代方案 |
|------|------|---------|
| 纯中国市场（抖音/小红书/B 站） | v1 优先国际市场验证 | 后续版本 |
| 创作者授权数据（收入/私信） | 需要 Creator OAuth，技术路径不同 | Phyllo |
| 支付/结算 | 涉及金融合规，v1 不碰 | 品牌自有财务流程 |

## 四、价值主张

### 核心价值

品牌当前做达人筛选的方式：
- 人工搜索：3-5 天 + 1 个人力
- SaaS 仪表盘：$5K-30K/年，需 Sales call，需学习操作
- MCN 推荐：免费但有利益偏差

用了 NoxInfluencer 的 Agent：**一句话，30 秒，数据驱动，无偏差。**

这不是"更便宜的 Modash"——是完全不同的交互方式。

### 凭什么是我们

| 优势 | 说明 | 置信度 |
|------|------|:------:|
| **数据资产** | 聚星 10 年达人数据，6 平台覆盖（YT/TikTok/IG/FB/X/NaverBlog），含合作效果数据、频道价值估算、假粉检测和受众画像（三平台）。H4 验证：覆盖率/新鲜度/准确度均为行业 top | 高（H4 ✅ 已验证） |
| **成本结构** | 中国团队，运营成本是 Modash/HypeAuditor 的 1/3-1/5 | 高 |

| **先发时机** | 99%+ 供给缺口 + 窗口期 6-12 个月 | 高 |
| **已有 API** | 聚星已有 6 平台 API endpoint（搜索/频道/视频/评论/受众/品牌分析），非从零开始 | 高 |

## 六、分发策略

### 核心原则

**多平台分发，不绑定任何一家。** 分发策略分两个轴线：

- **技术分发**（让 Agent 平台能调用 NoxInfluencer）：OpenClaw、Claude Code 等开发者生态
- **品牌获客**（让品牌广告主实际用上 NoxInfluencer）：Agent 自动发现 + ChatGPT Apps

Agent 平台格局变化极快（OpenClaw 两个月内三次改名），绑定单一平台风险极高。

### 分发矩阵

| 平台 | 格式 | 技术分发 | 品牌获客 | 理由 |
|------|------|:------:|:------:|------|
| **Agent 自动发现** | 工具元数据注册 | — | **P0** | 非技术用户不逛商店，靠 Agent 自动匹配意图并调用 |
| **ChatGPT App Store** | GPT Action | P1 | **P0** | 3 亿+ MAU，对非技术用户最友好，品牌获客最大入口 |
| **OpenClaw / ClawHub** | SKILL.md | **P0** | P2 | 157K Stars，技术社区热度最高 |
| **Claude Code / Anthropic** | MCP Server | **P0** | P2 | 企业 Agent 生态最成熟 |
| **Copilot Agent Store** | Agent | P1 | P1 | M365 企业入口 |
| **Glama / Smithery / mcp.so** | MCP Server | P1 | — | MCP 信任三角，影响专业用户决策 |
| **SkillsMP / Skills.sh** | SKILL.md | P2 | — | 跨平台开放标准目录 |

> 关键区分：技术分发 P0（OpenClaw/Claude）确保 NoxInfluencer 能被 Agent 调用；品牌获客 P0（Agent 自动发现/ChatGPT）确保品牌广告主真正用上。两者缺一不可。

### 技术架构

一套核心 API，两个接入层：

```
品牌广告主 → Agent 平台（OpenClaw / Claude / ChatGPT ...）
                ├── SKILL.md 接入层 → OpenClaw / Claude Code / Codex CLI
                └── MCP Server 接入层 → Claude / Cursor / Glama
                         ↓
                    NoxInfluencer 核心服务（REST API）
                         ↓
                    聚星数据后端
```

MCP 不替代 REST，是在 REST 之上加 Agent 接入层。核心 API 只建一次。

## 七、商业模式

### 收入模型：Credit 额度制（借鉴 Firecrawl + Apify）

所有成功的 MCP 产品都用同一套变现模式：**MCP 免费分发获客 → 官网 API Key 注册 → Stripe 付费**。目前无任何成功产品在 Agent 内完成付费。

| 模式 | 说明 | 依据 |
|------|------|------|
| **Credit 额度制** | 不同操作消耗不同 credit（搜索 1、评估 2、邀约 3、谈判 5） | Firecrawl: 一次性 500 credits → $1.5M→$22M ARR（12 个月 15x） |
| **一次性免费额度** | 200 credits（用完即止，非每月重置） | Firecrawl 用一次性免费→高转化；Context7 每月免费→被迫砍 92% 才能逼转化 |
| **订阅分层** | Free→$29→$99→$199→Enterprise | 对标 Firecrawl $16-599、Tavily $30-500 |
| **官网 Stripe 付费** | 在 NoxInfluencer 官网完成注册和付费 | 100% 成功竞品（Firecrawl/Tavily/Context7/Apify）均走官网+Stripe |

> 竞品标杆：Firecrawl $22M ARR、Tavily $275M 被收购（成立仅 1 年）、Apify $596K/月创作者分成

### 定价锚点

| 对比对象 | 品牌当前付出 | NoxInfluencer 目标价 |
|---------|-----------|--------------|
| SaaS 仪表盘（Modash 平台） | $135-1,350/月 | $29-199/月（便宜 3-7x） |
| 人工调研（1 个实习生） | ~$2,000/月工资 | 替代，非价格竞争 |
| MCN 推荐 | "免费"但有隐性成本 | 提供中立数据的溢价 |
| AI 邀约+谈判人力 | $83K-120K/年（专职经理） | 自动化替代大部分 55% 的沟通人力 |

> 具体定价策略和 credit 消耗细节见 `02_用户场景.md` 2.4 节，完整方案见 `04_定价与商业模式.md`。

1. 覆盖率：欧美头部达人覆盖率高于 70% 阈值，行业 top 水平

**结论：H4 通过验证，可以开工。数据壁垒是真实的竞争优势。**

## 十、风险

按严重度排序：

| # | 风险 | 严重度 | 发生概率 | 应对 |
|---|------|:------:|:------:|------|
| 1 | **需求不存在** — 品牌还没准备好用 Agent 做达人营销 | 致命 | 中 | 最小版本先测需求，不投重资源 |
| 2 | **数据不达标** — 聚星欧美数据 vs 竞品差距过大 | 致命 | 中 | 开工前抽样验证 H4 |
| 3 | **平台变局** — Agent 平台格局剧变，今天的热门明天衰落 | 高 | 高 | 多平台分发，一套 API 多个接入层 |
| 4 | **巨头入场** — Modash/GRIN 快速推出 MCP/Skill | 高 | 低 | 抢先发 + PLG 定价 + 多平台网络效应 |
| 5 | **AI 搜索质量差** — 自然语言→达人结果不准确、不可靠 | 高 | 中 | 技术 PoC 先行，降低对 AI 搜索的依赖（支持结构化筛选） |

### 退路

如果 H1（品牌用 Agent 做达人营销）12 个月内未验证，有多条备选路径：

| Plan | 方向 | 所需调整 | 可复用资产 |
|------|------|---------|----------|
| **B1** | 聚焦技术用户——回退为开发者 API | 强化 DX（SDK/文档），弱化邮件/谈判功能 | Core 层 + REST API + MCP Server + 数据层 100% 复用 |
| **B2** | B2B 数据供应商——向 MCN/Agency 批量供数据 | 增加批量导出能力，调整定价（按数据量计费） | 数据层 + 假粉检测 + 受众画像 100% 复用 |
| **B3** | 亚洲市场优先——利用聚星在亚洲的数据优势 | 增加中国平台支持（抖音/小红书/B 站），调整定位 | 四层架构 + 全部 Tool 逻辑复用，仅需增加数据源适配 |

回退成本低：四层架构（Harness/Shell/Core/Services）确保 Core 层业务逻辑与接入方式解耦，任何退路都能复用 80%+ 已建成基础设施。
