## 一句话定位

**开发者用来构建达人营销产品的数据API。**

类比：Brave Search API之于搜索，Stripe之于支付——我们之于达人数据。

---

## 核心问题

### 谁买？

**主要买家（按优先级）**：

| 优先级 | 买家 | 场景 | 付费意愿 | 规模 |
|:------:|------|------|:--------:|------|
| P0 | **构建营销Agent的团队** | Agent需要查询/匹配/验证达人 | 💰💰💰 | 快速增长（2025爆发） |
| P0 | **达人营销SaaS构建者** | 白标搜索/分析能力 | 💰💰💰💰 | 稳定 |
| P1 | **Agency技术团队** | 内部工具集成达人数据 | 💰💰💰 | 稳定 |
| P2 | **品牌增长团队** | 直接调用API做自动化 | 💰💰 | 大但分散 |

**不服务**：个人开发者玩票（免费层可用但不优化）、需要中国平台数据的（v1不做）

### 为什么买？

现有替代方案全部有致命缺陷：

| 替代方案 | 致命缺陷 |
|---------|---------|
| Modash API ($10K+/年) | 太贵、需Sales、年付锁定 |
| HypeAuditor API ($30K+/年) | 更贵、更封闭 |
| 自建爬虫 | 维护成本高、合规风险、不稳定 |
| 官方平台API | 数据极有限、需Creator授权 |
| 不用达人数据 | AI幻觉率40%+（ChatGPT达人推荐准确率仅59.7%） |

**核心痛点**（用户原话）：
> "utilizing their APIs is costly for me as I'm operating on a bootstrap budget" — Reddit
> "modash and hypeauditor are too expensive" — Reddit
> "需要Sales conversation才能看到定价和文档" — 多个开发者

### 凭什么是我们？

| 优势 | 说明 |
|------|------|
| **数据规模** | 1亿+达人档案，覆盖YT/TikTok/IG/Twitch |
| **10年数据飞轮** | 合作效果数据（哪个达人×哪个品类=什么效果）——竞品没有 |
| **成本结构** | 中国团队，成本是Modash/HypeAuditor的1/3-1/5 |
| **MCP原生** | 从Day 1支持MCP协议，竞品全部缺失 |
| **NoxGPT** | 自然语言达人搜索，可直接暴露为API/MCP能力 |
| **PLG定价** | $0起步，月付，无需Sales——市场唯一 |

### 凭什么不是我们？（诚实面对）

| 劣势 | 严重度 | 应对 |
|------|:------:|------|
| 数据量不是最大（1亿 vs Modash 3.5亿） | 🟡 | 够用，质量>数量 |
| 无假粉检测 | 🔴 | 集成第三方（Phyllo $199/月）或自研 |
| NoxInfluencer品牌污名 | 🔴 | 新品牌发布 |
| 海外开发者零认知 | 🟠 | MCP目录+开发者社区冷启动 |
| 数据来自scraping非官方API | 🟡 | 行业共性（Modash同样） |

## 战略价值（超越产品本身）

这个产品不只是一个API，它是聚星的**战略基础设施**：

1. **海外客户入口**：聚星历史上100%国内客户，API/MCP是首次切入海外的通道
2. **kol-agent的底层**：路线2的Agent产品直接调用这套API
3. **生态卡位**：成为Agent时代的"达人数据标准接口"
4. **防御价值**：如果别人做了这个API，聚星的Agent产品就要依赖竞品数据
