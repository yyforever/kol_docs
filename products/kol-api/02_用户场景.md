> 更新：2026-02-12

### ⚠️ 定位演进说明

本文在调研和场景分析中发现，01_定位中"v1 不服务 CRM / 合同 / 支付"的边界需要修正。品牌的本质痛点是**人工成本高——全链路每个环节都费劲**，仅切入搜索→筛选→评估（占 28% 人工时间）不足以构建付费级价值。

kol-api 的定位从"达人数据能力"演进为 **AI 驱动的达人营销全链路自动化**——让 Agent 替品牌完成从找人到管理的完整流程。

## 一、品牌广告主的达人营销现状

### 1.1 通用工作流与人工成本分布

品牌做达人营销的核心链路一致，但**各环节的人工成本差异巨大**：

```
搜索 → 筛选 → 评估 → 邀约 → 谈判 → 执行管理 → 效果追踪
 8%     12%     8%    22%    33%     10%        7%
 └─── 28% ───┘        └── 55% ──┘
  信息密集型              沟通密集型（最烧人力）
```

> 数据来源：Influencer Hero 研究——激活 50 个达人的完整流程，人工需 101-175 小时

**核心发现**：搜索+筛选+评估只占 28% 的人工时间。邀约+谈判占 55%，才是品牌最大的成本黑洞。效果追踪虽然只占 7% 的操作时间，但 ROI 未知和曝光量不足是两个**本质痛点需求**。

### 1.2 各环节痛点量化

| 环节 | 占人工时间 | 痛点 | 量化数据 | 来源 |
|------|:---------:|------|---------|------|
| 搜索 | ~8% | 跨平台手动搜索，无结构化对比 | 人工搜索 3-5 天/轮 | 行业调研 |
| 筛选 | ~12% | 假粉泛滥，肉眼无法判断 | 49% IG 达人存在某种程度欺诈 | HypeAuditor 报告 |
| 评估 | ~8% | 受众画像需多源交叉验证 | 79% 营销人员认为 ROI 归因是最大挑战 | eMarketer |
| **邀约** | **~22%** | 响应率低，联系方式难找 | 需发 333+ 封邮件激活 50 人（响应率 8-20%），邮箱覆盖率仅 25-40% | Influencer Hero |
| **谈判** | **~33%** | 无标准定价，87.9% 营销人员说越来越难 | 平均每个达人 6 轮邮件往返，58% 达人要求混合报酬 | Modash 2025 调查 |
| 执行管理 | ~10% | 物流/审稿/合规/付款全靠人盯 | 各环节分散在不同工具和表格 | 聚星客户反馈 |
| 效果追踪 | ~7% | ROI 未知 + 曝光量不足 | 这是两个本质痛点需求 | 聚星客户反馈 |

### 1.3 EU/US 品牌 vs 国内品牌的关键差异

| 维度 | EU/US 品牌 | 国内品牌（聚星客户） |
|------|-----------|-------------------|
| **主要工作方式** | SaaS 仪表盘 + 手动搜索 | SaaS 工具自助（聚星平台） |
| **工具支出** | 自助 $49-299/月，中型 $478-2,000/月 | 平台年费制为主 |
| **最痛环节** | 假粉/数据真实性 + 邀约谈判耗时 | 人工成本高（全链路都费劲）+ ROI 未知 |
| **合规压力** | FTC 单条违规 $53K+，GDPR 上限收入 4% | 相对宽松，但趋严 |
| **AI 采用** | 82% 使用 AI 工具，仅 35% 看到显著效果 | 起步阶段 |
| **人力成本** | 专职经理 $83K-120K/年，高级 $215K/年 | 相对较低但也在增长 |

v1 优先服务 EU/US 品牌：Agent 渗透率高、工具支出意愿已验证、人力成本压力大（自动化 ROI 更明显）。

## 二、用户旅程（从发现到付费）

### 2.1 发现路径：Agent 自动发现是第一优先级

**必须正视的现实**：非技术用户不会主动逛 Skill 商店。ClawHub 安装需要 CLI（`npx clawdhub@latest install`），对品牌广告主是硬门槛。

因此，**Agent 自动发现和调用 kol-api 是最关键的分发策略**——品牌不需要知道 kol-api 的存在，只需要在 Agent 对话中提出达人营销需求，Agent 自动发现并调用。

| # | 路径 | 优先级 | 说明 |
|---|------|:------:|------|
| A | **Agent 自动发现并调用** | **最高** | 品牌在 Agent 中提需求 → Agent 自动发现 kol-api → 直接调用并返回结果。类似 ChatGPT 自动调用 Brave Search，用户无需安装。需要在各平台注册为"达人营销"品类的首选工具 |
| B | **代理商/IT 预配置** | 高 | 有人帮品牌在 Agent 环境中配好 kol-api，品牌通过聊天界面使用 |
| C | **内容营销引流** | 中 | 品牌从行业文章/视频/案例看到效果 → 让技术人员或 Agency 协助配置 |
| D | **品牌自助安装** | 低 | 仅在技术门槛降低后可行（ChatGPT Apps 免代码安装 / OpenClawd.ai 托管方案） |

**落地要求**：
- 各 Agent 平台的"达人营销"语义触发词注册（influencer / KOL / creator marketing）
- ChatGPT Apps 上架（对非技术用户最友好，3 亿+ MAU）— 品牌获客最重要的入口
- MCP Server 元数据完善（工具描述精准，让 Agent 能自动匹配用户意图）

### 2.3 留存驱动力

| 驱动力 | 说明 | 对应场景 |
|--------|------|---------|
| **全链路省人** | 从找人到谈判到管理，每个环节都省时间和人力 | 全部 |
| **数据信任** | 假粉检测 + 受众画像 + 定价基准让品牌觉得"靠谱" | 评估+谈判 |
| **越用越准** | Agent 记住品牌偏好、合作历史、达人黑白名单 | CRM+记忆 |
| **不可替代性** | 大规模邮件邀约 + AI 自动谈判是市场稀缺能力 | 邀约+谈判 |

### 2.4 付费模式：借鉴已验证的成功路径

调研发现，**所有成功的 MCP 产品都用同一套变现模式**——MCP 免费分发获客，官网 API Key 完成付费转化：

```
品牌在 Agent 中使用 → 需要 API Key → 去 kol-api 官网注册 →
免费额度体验 → 额度用完/需要高级能力 → 官网升级付费（Stripe）
```

> 目前没有任何成功产品在 Agent 内完成付费。Stripe + Cloudflare 正在开发 MCP 内付费能力（2025 年 5 月发布 SDK），值得持续跟踪。

#### 成功竞品变现细节

| 产品 | 收入 | 免费层 | 付费起点 | 核心策略 |
|------|------|--------|---------|---------|
| **Firecrawl** | $1.5M→**~$22M ARR**（12 个月 15x） | 500 次**一次性**（非每月） | $16/月 | 一次性免费额度造紧迫感，用完即需付费 |
| **Tavily** | **$275M 被收购**（成立仅 1 年） | 1,000 次/月 | $30/月 | 做 LangChain 默认搜索工具，100% 收入来自 Agent 用量 |
| **Context7** | 396K 周下载 | 1,000 次/月（原免费层砍 92%） | $10/月/人 | 先用海量免费用户建品类心智，再大幅削减免费额度逼转化 |
| **Apify** | **$596K/月**创作者分成 | $5/月平台额度 | 按事件计费 $0.005-0.50 | PPE 模型：按操作计费，创作者拿 80%，平台拿 20% |
| **21st.dev** | ~$500/月 MRR | 10 次/月 | $16/月 | 纯 MCP 原生付费，规模仍小 |

> 来源：Firecrawl Series A 公告、Tavily/Nebius 收购报道（Bloomberg 2026-02-10）、Context7 Dev Genius 报道、Apify 官方开发者页面

#### kol-api 变现设计（借鉴 Firecrawl + Apify 模型）

**核心机制**：Credit 额度制（类似 Firecrawl），不同操作消耗不同 credit 数。

| 操作 | Credit 消耗 | 说明 |
|------|:----------:|------|
| 达人搜索 | 1 credit | 一次搜索返回最多 50 条结果 |
| 达人详情+受众 | 2 credits | 含假粉检测、受众画像 |
| 邮件邀约 | 3 credits | 含联系方式查询+邮件发送 |
| AI 谈判（每轮） | 5 credits | 代品牌与达人邮件往返一轮 |
| 竞品对标 | 5 credits | 品牌合作历史分析 |

**定价阶梯**（对标 Firecrawl $16-599、Tavily $30-500）：

| 层级 | 月费 | Credits/月 | 目标用户 | 对标 |
|------|:----:|:---------:|---------|------|
| **Free** | $0 | 200（一次性） | 首次体验 | Firecrawl 一次性免费 |
| **Starter** | $29 | 2,000 | SMB/DTC | CreatorDB $79（便宜 2.7x） |
| **Pro** | $99 | 10,000 | 中型品牌 | Modash $135（便宜 1.4x） |
| **Growth** | $199 | 30,000 | 重度用户/小型 Agency | HypeAuditor $250+ |
| **Enterprise** | 定制 | 不限 | 大型品牌/Agency | Sales 对接 |

**关键设计决策**（来自竞品经验）：

| 决策 | 选择 | 竞品依据 |
|------|------|---------|
| 免费额度一次性 vs 每月 | **一次性**（用完即止） | Firecrawl 一次性 500 credits → 15x 增长；Context7 每月免费→被迫砍 92% 才能逼转化 |
| 付费墙放在哪 | **大规模邀约 + AI 谈判** | 搜索免费建心智，高价值操作付费。Tavily: 基础搜索便宜，高级搜索 2x credit |
| 超额计费 | **按 credit 单价** | Firecrawl: 超额 $9/1K credits；避免硬性阻断 |
| 支付方式 | **Stripe（官网）** | 100% 成功竞品都用官网+Stripe，无一例 Agent 内付费 |

#### 付费转化漏斗

```
Agent 自动发现 kol-api → 首次搜索（消耗 1 credit）
  → 觉得好用，继续搜索+评估（消耗 ~50 credits）
  → 200 免费 credits 用完 → Agent 提示"注册 kol-api 获取更多额度"
  → 品牌去官网注册 → 免费 credit 已用完，直接看到付费方案
  → $29/月起步（2,000 credits ≈ 每月找+评估 ~200 个达人，或邀约 ~60 个）
  → 深度用户升级 $99-199/月（AI 谈判 + CRM + 大规模邀约）
```

> 假设 H9：Freemium 转付费率 > 3%。行业基准 3-5%，Firecrawl 用一次性免费层实现了更高转化（具体数据未公开）。

## 三、核心使用场景

### 3.1 P0：达人发现（搜索+筛选+评估）

**痛点**：人工搜索 3-5 天，假粉泛滥（49% IG 达人有欺诈），受众画像需多源验证。

**交互示例**：

```
用户：帮我找欧洲健身类 Instagram 达人，粉丝 1-10 万，最好经常发 Reels

Agent：找到 15 位符合条件的达人，按互动率排序：

 #  达人            粉丝     互动率   Reels占比   国家    可邀约
 1  @fitvibes_de    8.2万    6.3%    78%        德国    ✅
 2  @strongandsara  5.1万    5.8%    65%        英国    ✅
 3  @move_with_max  3.4万    5.2%    82%        荷兰    ❌ 无邮箱
 ...

用户：分析一下 @fitvibes_de 的数据真实性

Agent：
  数据真实性：✅ 可信（87/100）
  - 粉丝增长自然，无突增  - 92% 真实互动  - 评论质量高
  受众：女性 72%, 25-34 岁 48%, 德国 45%
  品牌契合度：与你目标受众重合约 62%，高度推荐
```

**kol-api 提供**：`search_creators`、`get_creator_profile`、`check_authenticity`、`get_audience`

### 3.2 P0：邀约触达

**痛点**：占 22% 人工时间。需发 333+ 封邮件才能激活 50 个达人。联系方式覆盖率仅 25-40%，手动找邮箱极耗时。**大规模邮件触达能力在市场上稀缺**，且需要数据支撑——哪些达人邮箱已验证、历史响应率如何、是否有合作意向。

少量邀约可走品牌自己邮箱，大规模邀约必须通过 kol-api。

**交互示例**：

```
用户：帮我邀约刚才找到的 12 位可联系达人，推广我们新出的蛋白粉

Agent：已准备邀约方案：

  可直接邀约：12 位（邮箱已验证）
  预估响应率：15-25%（基于历史数据和达人活跃度）
  预计回复：2-3 位

  邮件内容预览（已根据每位达人定制）：
  ──────────────────────────
  To: @fitvibes_de
  Subject: Collaboration with [Brand] — Protein Powder Launch 🏋️

  Hi [Name],
  Love your Reels about home workouts — especially the 30-day challenge
  series (2.1M views!). We're launching a plant-based protein powder
  for the EU market and think your audience is a perfect fit.

  Would you be interested in a gifted collaboration + paid Reel?
  Budget range: €600-900 per Reel.

  Looking forward to hearing from you!
  ──────────────────────────

  确认发送吗？还是需要调整邮件内容？

用户：发送吧

Agent：已发送 12 封邀约邮件。我会跟踪回复情况：
  - 送达确认后通知你
  - 有回复时立即告知
  - 3 天无回复的自动发 follow-up（你可以关闭）
```

**kol-api 提供**：`get_contacts`（联系方式+可达性评分）、`send_outreach`（大规模个性化邮件发送）、`track_responses`（响应追踪+自动跟进）

### 3.3 P0：谈判协商

**痛点**：占 33% 人工时间——全链路最大的单项成本。平均每个达人 6 轮邮件往返，87.9% 营销人员说谈判越来越难，58% 达人要求混合报酬（底价+分成）。目前**全市场仅 3 家新公司**（Aha/Janney AI/Stormy AI）在做自动谈判，极度稀缺。

**交互示例**：

```
用户：@fitvibes_de 回复了，要价 €1,200 一条 Reels，帮我谈

Agent：
  分析：
  - 市场基准：同量级欧洲健身达人 Reels 中位数 €750
  - 该达人要价高于中位数 60%，但互动率也高于中位数 1.4x
  - 合理区间：€700-900
  - 建议策略：先提 €700 + 产品赠送，目标成交 €800-850

  我来代你谈判吗？预算上限设多少？

用户：上限 €900，尽量谈到 €800

Agent：收到，开始谈判。我会：
  1. 回复达人，提出 €700 + 产品赠送的初始报价
  2. 根据达人回复在 €700-900 区间内灵活协商
  3. 每轮谈判进展实时同步给你
  4. 达成一致后发合作确认邮件（你审核后发出）

  ───── 谈判进行中 ─────
  Round 1: 我方出价 €700 + 产品 → 达人还价 €1,000
  Round 2: 我方 €800 + 产品 + Story 加推 → 达人还价 €900
  Round 3: 我方 €850 含 1 Reel + 2 Stories → 达人接受 ✅

  谈判完成：€850（含 1 条 Reel + 2 条 Story），比要价降 29%
  需要我发送合作确认邮件吗？
```

**kol-api 提供**：`get_pricing_benchmark`（市场定价基准）、`negotiate`（AI 自动谈判，在预算范围内代品牌与达人邮件往返）、`get_negotiation_history`（谈判记录）

**竞品对标**：Janney AI 案例——AI 将 $3,500 谈到 $2,000（4 封邮件，降幅 43%）

### 3.4 P1：CRM 与达人关系管理

**痛点**：合作过的达人散落在邮件和表格里，无法系统追踪。

**交互示例**：

```
用户：我之前合作过哪些健身达人？效果怎么样？

Agent：
  你的健身达人合作历史（共 8 位）：

  复投推荐（效果好 + 关系良好）：
  ✅ @fitvibes_de — 合作 3 次, 平均互动率 5.8%, 最近合作 2 周前
  ✅ @strongandsara — 合作 2 次, 平均互动率 4.2%, 回复速度快

  观望：
  ⚠️ @gym_daily — 合作 1 次, 互动率 2.1%（低于预期）

  黑名单：
  ❌ @fake_fit — 合作后发现假粉率 38%，已标记

  需要对推荐达人发起新一轮邀约吗？
```

**kol-api 提供**：`get_collaboration_history`（合作历史）、`update_creator_status`（达人白名单/黑名单/备注）

### 3.5 P1：主动提醒与监控

**痛点**：关键节点靠人记，容易漏。

**能力清单**：

| 提醒类型 | 场景 | 触发条件 |
|---------|------|---------|
| 合作到期 | 合同快到期，是否续约 | 合同到期前 7 天 |
| 内容发布 | 达人发布了合作内容 | 检测到赞助内容上线 |
| 数据异动 | 达人粉丝骤降或互动率异常 | 偏离基线 > 30% |
| 邀约跟进 | 邮件无回复 | 发送后 3 天无响应 |
| 竞品动态 | 竞品与特定达人开始合作 | 检测到新赞助内容 |

**kol-api 提供**：`set_alert`（配置提醒规则）、`get_alerts`（获取触发的提醒列表）

### 3.6 P1：合作管理

**痛点**：合同、物流、审稿、合规、付款各自分散，每个环节都费劲。

**交互示例**：

```
用户：@fitvibes_de 的合作进展怎么样了？

Agent：
  @fitvibes_de — 蛋白粉推广合作
```

## 四、不同品牌规模的差异化路径

### 4.1 SMB/DTC（$29-99/月）

- **现状**：Excel + 手动搜索 + 逐个发邮件
- **发现路径**：Agent 自动推荐（路径 A）
- **核心价值**：从全手动到全自动的跨越——搜索+邀约+谈判一条龙
- **典型用法**：每月找 10-30 个达人，批量邀约，AI 帮谈价

### 4.2 中型品牌（$99-199/月）

- **现状**：已用 SaaS 工具（Modash/AspireIQ），但邀约和谈判仍靠人
- **发现路径**：内容营销（路径 C）或代理商推荐（路径 B）
- **核心价值**：比 SaaS 便宜 3-7x + 邀约谈判自动化（竞品没有）
- **典型用法**：CRM 管理达人池，批量邀约+自动谈判，主动提醒跟进

### 4.3 大型品牌/Agency（定制方案）

- **现状**：GRIN/Impact 全套 + 专职团队
- **发现路径**：IT/开发者配置（路径 B）
- **核心价值**：AI 谈判层作为现有工具的增强，替代部分人力
- **典型用法**：API 集成到内部系统，大规模邀约+谈判自动化

## 五、关键假设与风险

### 5.1 定位演进风险

01_定位中 kol-api 是"达人数据能力"，本文扩展为"全链路自动化"。这意味着：
- **技术复杂度大幅上升**：需要邮件发送基础设施、AI 谈判引擎、CRM 存储
- **与聚星平台的关系变化**：从"互补"变为部分"重叠"
- **竞品集合变化**：不只对标 Modash API，还对标 Janney AI / Stormy AI / Aha

需要在 01_定位中同步更新，或在 03_API能力设计中明确阶段划分。

### 5.2 技术门槛假设

品牌大概率不能自助安装，因此 **Agent 自动发现是唯一可规模化的品牌获客路径**。

落地条件：
- ChatGPT Apps 上架后被推荐系统覆盖
- MCP 工具描述精准匹配"influencer/KOL/creator marketing"意图
- 各平台达人营销品类注册为首选工具

### 5.3 邀约能力假设

v1 自建大规模邮件发送，需要评估：
- 邮件投递率和反垃圾信誉（冷启动风险）
- 聚星已有 YT/TikTok/IG 联系方式端点，覆盖率待验证
- influencers.club 有 340M 创作者中 100M+ 验证邮箱（约 29% 覆盖率），可作为数据补充参考
- 合规要求：CAN-SPAM / GDPR 对商业邮件的限制

### 5.4 AI 谈判假设

全自动代谈是最高难度的 AI 应用，风险包括：
- 谈判话术的质量和文化适应性
- 品牌对"AI 代表我谈"的信任度
- 达人对 AI 邮件的接受度（是否反感）
- 兜底机制：谈判僵局时如何回退到人工

**验证方式**：先在内部测试 50 个真实谈判案例，评估成交率和降价幅度。

### 5.5 平台演化假设

不押注任何一家 Agent 平台。一套核心服务，多个接入层：

| 平台 | 品牌友好度 | 自动发现支持 | 优先级 |
|------|:---------:|:----------:|:-----:|
| ChatGPT Apps | 最高 | 推荐系统 | 品牌获客 P0 |
| OpenClaw/ClawHub | 低 | Skill 自动匹配 | 技术分发 P0 |
| Claude Code | 低 | MCP 自动发现 | 技术分发 P0 |
