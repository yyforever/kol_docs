# 达人表现监控

监控类能力不是发现的替代品，而是 shortlist 和合作后的延续。

## 适合的场景

- 观察 shortlist 中谁值得继续追踪
- 跟踪合作对象在一段时间内的变化
- 为 campaign 复盘准备更连续的上下文

## 价值

- 让团队不再只看单次快照
- 能更早发现异常和趋势变化
- 有助于把决策从“主观印象”转向“持续观察”
