# AI 代理驱动的达人营销

AI Agent 对达人营销最直接的价值，不是替代全部人工，而是把关键环节串成更短的工作流。

## 更适合被压缩的环节

- 候选达人发现
- 值不值得继续投入的判断
- 已合作对象的持续观察

## 为什么公开文档要分成 Help 和 Use Cases

因为帮助文档解决“怎么用”，而场景页解决“为什么值得用”。这两类内容面向的搜索意图不同，但都应该使用同一套事实来源。
