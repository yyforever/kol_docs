# 触达前达人评估

许多团队的低效，不是找不到达人，而是过早进入联络。

## 常见问题

- 受众看起来不错，但其实不匹配目标市场
- 指标看起来能打，但稳定性不足
- 团队在明显不合适的对象上消耗了沟通时间

## 推荐做法

先完成标准化评估，再决定是：

- 继续联系
- 放入监控
- 直接排除

## 为什么这对 ROI 有帮助

因为它把“要不要继续投入”这个决策提前了，而不是等到商务动作已经开始才发现对象不合适。
