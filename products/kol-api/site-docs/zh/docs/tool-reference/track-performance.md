# 表现监控

**当前状态：Beta**

Track Performance 面向“持续观察”场景，用来跟踪达人或内容在一段时间内的表现变化。

## 适合什么场景

- 你已经有 shortlist，想继续观察
- 你已经开始合作，想跟踪表现变化
- 你需要做阶段性复盘，而不是只看一次静态结果

## 当前公开说明

监控能力已经是当前产品线的核心方向之一，但其最终公开形态仍在收敛中。阅读本页时，请把它理解为“公开监控能力说明”，而不是一份底层接口文档。

## 你应该期待什么

- 监控对象建立后的持续观察
- 对变化、异常和趋势的周期性查看
- 支持 shortlist、campaign 或复盘决策

## 当前边界

- 不替代 Discover 或 Analyze
- 不承诺所有监控类细节都已经对外稳定开放
- 具体公开范围以后续产品发布说明为准
