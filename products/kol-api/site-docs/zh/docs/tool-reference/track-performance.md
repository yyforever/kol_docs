# 表现监控

**当前状态：Available**

Track Performance 当前已经作为公开的项目化视频监控能力提供，用来持续跟踪合作内容和阶段性表现变化。

## 适合什么场景

- 你已经进入合作执行，想持续跟踪视频表现
- 你需要做阶段性复盘，而不是只看一次静态结果
- 你想看异常变化、趋势和项目级汇总

## 当前公开工作流

- 查看监控项目列表
- 创建监控项目
- 把一个或多个视频 URL 添加为任务
- 查看任务列表和项目 summary

## 你应该期待什么

- 项目化的监控对象
- 视频级状态和表现快照
- 项目级汇总与平台拆分信息

## 当前边界

- 不替代 Discover 或 Analyze
- 当前公开形态是 video-monitor 项目闭环，不是泛化的长期 creator watchlist
- 还不覆盖导出、协作者、CRM 式协同等完整 campaign 运营层
