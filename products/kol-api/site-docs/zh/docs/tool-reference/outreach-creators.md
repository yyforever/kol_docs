# Outreach Creators

**当前状态：Planned**

Outreach Creators 目录用于承载未来公开的达人触达能力说明。

## 预期适用场景

- 你已经完成 shortlist 和尽调
- 你准备进入联系、沟通或商务动作
- 你需要把达人联系方式和 campaign 信息结合起来使用

## 当前不能假设什么

- 不能假设该能力已经作为完整公开 Tool 稳定可用
- 不能把本页理解成当前开放的外呼自动化承诺
- 不能跳过 Analyze 阶段直接把搜索结果视为可联系名单

## 当前建议

在公开能力完全就绪前，先把本页当作路线图目录位。实际可用的对外能力应以当前实现和发布说明为准。
