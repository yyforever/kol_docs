# Negotiate

**当前状态：Planned**

Negotiate 代表未来面向合作协商和谈判流程的公开能力方向。

## 未来适用范围

- 商务沟通节奏管理
- 价格和合作条件协商
- 将前期分析结果带入后续合作判断

## 当前边界

- 本页不是现阶段可直接上线使用的谈判自动化说明
- 不应被当作已开放的执行型能力承诺
- 对外公开前，仍需要更严格的人工确认和风险控制

## 为什么现在保留目录位

公开文档需要从 Day 1 起保留长期信息架构，便于后续在能力开放时平滑补齐内容，而不是重做导航心智。
