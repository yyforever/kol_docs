# CRM

**当前状态：Planned**

CRM 代表未来面向关系管理与后续协同的公开能力方向，适合承接达人筛选、分组和导出之后的长期运营动作。

## 未来适用范围

- 把一批已选中的 creator records 组织成更长期的关系资产
- 支持后续生命周期管理和跨阶段协同
- 把 collection 风格的分组结果延伸到更长期的运营跟进

## 当前边界

- 本页不是“完整 CRM 已经公开可用”的承诺
- CRM 还不是当前公开 NoxInfluencer skill 的主能力集合之一
- 如果你现在需要可用路径，优先依赖活动管理、资源池、导出任务和 creator workflow 主链路
