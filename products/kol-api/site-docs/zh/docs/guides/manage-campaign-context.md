# Manage Campaign Context

随着使用次数增加，团队最需要的不是“再搜一次”，而是“把上次的上下文接上”。

## 为什么上下文管理重要

- 避免每次都重新解释品牌目标
- 让 shortlist、分析结论和监控对象能复用
- 减少 Agent 会话切换导致的信息丢失

## 适合沉淀的内容

- 品牌目标市场和品类
- 已经筛过的 shortlist
- 值得继续观察的对象
- 已经明确排除的对象和原因

## 当前状态说明

Manage Campaigns 是公开信息架构中的正式目录，但其最终公开形态仍在演进中。若你需要立即可用的能力，应优先依赖当前已公开的发现、分析和监控链路。
