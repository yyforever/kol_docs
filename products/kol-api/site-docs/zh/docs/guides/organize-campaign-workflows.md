# 组织活动工作流

当你已经明确 campaign 方向，并希望同一批对象继续流转到 beta marketing ops 页面时，可以按这条最小闭环来组织工作。

## 推荐的最小闭环

1. 先建立一个明确的 campaign 锚点，固定市场、目标和工作范围
2. 把已经 review 过的达人放进 collection，而不是散落在不同会话里
3. 当你需要按组操作或复用对象集合时，围绕这个 collection 继续工作
4. 当你需要交接或下载结果时，发起 export
5. 再把新的结果回流到同一个 campaign 上下文中

## 需要保持稳定的内容

- campaign 目标和目标市场
- 你决定保留的 creator identities
- 你纳入或排除对象的逻辑
- 这次导出的用途和原因

## 当前边界

- 这是一条 beta workflow bridge，不是完整的 CRM 操作手册
- 对核心 creator 判断，你仍然会继续依赖发现、分析、触达准备和监控主链路
