# 触达前评估达人

在进入联络和合作前，建议先完成一次标准化评估，避免把时间花在明显不合适的对象上。

## 推荐优先级

1. 先看整体可信度和异常信号
2. 再看受众是否匹配你的目标市场
3. 最后判断是否值得进入联系或长期监控

## 你要回答的核心问题

- 这个达人是否值得继续投入时间
- 其受众是否与目标市场一致
- 是否存在明显的合作风险或不稳定信号

## 什么时候不该直接进入 Outreach

- 你还没有确认受众匹配度
- 表现数据与合作预期明显不一致
- 存在明显争议、风险或可疑信号

## 推荐下一步

- 如果评估结果好：进入联系方式或合作准备流程
- 如果结果一般：先保留在观察列表
- 如果结果不稳定：转去监控而不是立刻触达
