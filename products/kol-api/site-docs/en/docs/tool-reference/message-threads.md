# Message Threads

**Current status: Planned**

Message Threads represents the future public direction for thread-based communication workflows after creator selection.

## Intended role

- Keep message history attached to one stable collaboration thread
- Preserve labels, cooperation status, and workflow state across later operations
- Reduce context loss when communication spans multiple sessions or teammates

## What this does not mean today

- This is not a public promise that NoxInfluencer will run messaging channels for you right now
- Message Threads is not yet part of the main public NoxInfluencer skill workflow
- You should not treat this page as a ready-to-run outreach execution surface

## Use instead today

- Use [Outreach Creators](outreach-creators.md) when you need contact retrieval
- Use [Manage Campaigns](manage-campaigns.md) and [Collections](collections.md) when you need workflow continuity
