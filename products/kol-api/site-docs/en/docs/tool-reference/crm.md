# CRM

**Current status: Planned**

CRM represents the future public direction for relationship-oriented coordination after creator selection, grouping, and export preparation.

## Intended role

- Keep grouped creator records organized beyond one shortlist
- Support later lifecycle coordination around selected creators
- Connect collection-style grouping with longer-running operational follow-up

## Current boundary

- This is not a public promise that a full CRM workflow is available today
- CRM is not yet part of the main public NoxInfluencer skill capability set
- If you need a usable workflow now, rely on campaigns, collections, exports, and the creator workflow pages first
