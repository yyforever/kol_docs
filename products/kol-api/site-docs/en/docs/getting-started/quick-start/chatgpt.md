# Quick Start for ChatGPT

## Recommended path

1. Confirm your account, plan, and capability access
2. Complete the ChatGPT-side connection flow
3. Start with a clear creator discovery task
4. Use the returned shortlist to continue into analysis

## A good first prompt

> Find creators for the US skincare market and give me a shortlist I can review

The point is not the exact wording. The point is to include the platform, market, category, and business goal in one request.

## Success signals

- You get structured candidate results instead of generic help text
- You can continue from discovery into analysis
- Errors point to an understandable next step

## If the first attempt fails

- Check whether the right account is connected
- Check whether the current plan includes the capability you are trying to use
- Review the recovery paths in [Error Codes](../../resources/error-codes.md)
