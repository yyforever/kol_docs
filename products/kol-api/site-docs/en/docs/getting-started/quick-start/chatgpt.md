# ChatGPT Support Status

## Current status

ChatGPT is **not currently a supported public install path** for this skill.

The current public `skills` repository does not publish a ChatGPT install flow for `noxinfluencer`.

## What to use instead

- Start from the public GitHub repository: [https://github.com/NoxInfluencer/skills/tree/main](https://github.com/NoxInfluencer/skills/tree/main)
- If you use OpenClaw, use the public store page: [https://clawhub.ai/noxinfluencer/noxinfluencer](https://clawhub.ai/noxinfluencer/noxinfluencer)

## Why this page still exists

This route is kept as a status page so users who look for ChatGPT support can see the current public availability instead of landing on outdated setup instructions.

## If public ChatGPT support appears later

This page should be updated only after the public repository or product release notes publish a real ChatGPT install path.
