# Set Up Performance Monitoring

Monitoring works best after you already have a candidate, collaborator, or watchlist target.

## Best-fit scenarios

- Ongoing shortlist observation
- Collaboration follow-up
- Campaign review and trend watching

## Current note

Track Performance is a core public direction, but the final public surface is still being standardized. Treat this page as capability guidance, not as a low-level API spec.
