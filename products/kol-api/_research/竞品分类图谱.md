| # | 核心结论 | 置信度 |

## 三、数据获取方式分层模型

```
┌─────────────────────────────────────────────┐
│  Layer 4: Agent 智能层                       │
│  Stormy, Aha, GRIN Gia, kol-agent           │
├─────────────────────────────────────────────┤
│  Layer 3: 达人数据API (加工数据)    ← kol-api│
│  Modash, CreatorDB, HypeAuditor              │
├─────────────────────────────────────────────┤
│  Layer 2: 社交数据基础设施 (结构化爬取)      │
│  Data365, EnsembleData, ScrapeCreators       │
├─────────────────────────────────────────────┤
│  Layer 1: 原始数据基础设施 (代理+爬虫)       │
│  Bright Data, Oxylabs, Apify, ScrapingBee   │
├─────────────────────────────────────────────┤
│  Layer 0: 平台官方API / Creator OAuth        │
│  TikTok/IG/YT官方, Phyllo (授权数据)        │
└─────────────────────────────────────────────┘
```

kol-api 在 **Layer 3** — 加工数据层：
- 数据经过清洗、标准化、富化（假粉检测、受众画像、定价估算）
- 比 Layer 1-2 贵但省去开发者自建管道的成本
- 比 Layer 4 灵活，开发者可以自由组合构建自己的产品

## 四、各品类详细分析

### 品类 1: 达人数据 API-First 平台

> 核心特征: 以开发者为主要客户，提供自助 API 访问达人搜索/分析数据
> 与 kol-api 关系: **直接竞品**

| 公司 | 定价 | 数据规模 | 平台覆盖 | DX 评分 | 亮点 | 弱点 |
|------|------|----------|----------|---------|------|------|
| **Modash** | Discovery $16,200/yr, Raw $10,000/yr | 350M+ | IG/TikTok/YT | ⭐⭐⭐⭐⭐ 文档最佳 | OpenAPI spec, Postman, Best Practices 指南 | 无 SDK, 无 MCP, 需 Sales |
| **CreatorDB** | 平台 $79+/mo, API 另计 | 5M+ | YT/IG/TikTok | ⭐⭐⭐⭐ 试用最友好 | 有 MCP(31 tools), 14 天试用+5000 credits | MCP Glama F 级不可用, 无 SDK |
| **HypeAuditor** | 平台 $399+/mo, API $2,999+/yr | 207.8M+ | IG/TikTok/YT/Twitch/X | ⭐⭐⭐ DX 最差 | 假粉检测行业标杆 | 认证非标准(双 header), 无 SDK/Postman/沙盒 |
| **influencers.club** | $249+/mo | 340M, 40+ 平台 | 40+ 平台 | — | 唯一大规模验证联系方式 | 偏外联数据, 非分析数据 |
| **Inbeat** | 公开定价 | IG+TikTok | IG/TikTok | — | 自助注册, 简单易用 | 平台少, 数据深度有限 |

> DX 评分来源: `influencer-api-dx-comparison.md`

**关键洞察**:
- Modash 是 DX 标杆但定价是 kol-api 的 7-10x，且无 MCP
- CreatorDB 试用最友好但 MCP 因缺 LICENSE 在 Glama 上完全隐形
- **全部竞品都无官方 Python/Node SDK** — kol-api 第一个做就是巨大 DX 优势
- **全部竞品都需要 Sales call** — PLG 自助是空白

### 品类 10: MCP 生态成功标杆（跨行业对标）

> 核心问题: 一个非知名品牌，能不能靠"提供优质 MCP 服务"获得新客户和收入？
> 与 kol-api 关系: **模式验证** — 非达人营销行业，但验证"垂直数据 API + MCP 分发 + PLG"模型

#### 筛选标准

- 垂直数据/服务 API（非通用工具）
- MCP 原生分发（用户通过 MCP 目录发现）
- PLG 自助（免费层→自助付费）
- **发布 MCP 前不是知名品牌**（排除 Supabase/Neon/E2B 等已成功公司）
- **增长可归因于 MCP 渠道**

#### 标杆公司

| 公司 | 领域 | MCP 前状态 | MCP 后成果 | 模式 |
|------|------|-----------|-----------|------|
| **Firecrawl** | 网页数据 API | 2024 从 Mendable 转型，YC 孵化但无人知晓 | **$1.5M 收入**（10 人），350K+ 开发者，78K Stars，$16.2M 融资 | 垂直数据 API + MCP + PLG |
| **Tavily** | AI 搜索 API | 2023 开源 GPT Researcher，无商业收入 | **$2.8M 收入**，1M+ 开发者，$275M 被 Nebius 收购（2026.02） | 垂直数据 API + MCP + PLG |
| **Context7**（Upstash） | 文档上下文 API | 母公司 Upstash 仅 $1M ARR，Context7 本身不存在 | **45.2K Stars**，396K 周 NPM 下载，MCP 生态月搜索量第 4 | MCP 原生免费产品 → 付费平台导流 |
| **21st.dev** | UI 组件生成 | 个人开发者项目，完全不存在 | **~£400/月 MRR**，4.2K Stars，4000 beta 用户（2 周内） | 纯 MCP 原生付费 |

#### 各标杆详细分析

**Firecrawl — 最接近 kol-api 的对标**

Mendable（YC S22）原做文档 AI 聊天，2024 年发现"给 AI 获取干净网页数据"是更大需求，转型为 Firecrawl。10 人团队 2024 年做到 $1.5M 收入，核心增长来自 MCP 目录分发 + 开源社区。2025.08 完成 $14.5M Series A（Shopify CEO 参投）。MCP Server 仓库 5.2K Stars，已接入 Claude/Cursor/ChatGPT。

与 kol-api 的相似度：垂直数据 API → MCP 分发 → PLG 定价 → 小团队。区别是网页抓取赛道竞争激烈，达人营销 MCP 品类 99%+ 空白。

> 来源: [GetLatka](https://getlatka.com/companies/firecrawl.dev), [YC Profile](https://www.ycombinator.com/companies/firecrawl), [GitHub](https://github.com/firecrawl/firecrawl-mcp-server)

**Tavily — 从开源到 $275M 收购的 MCP 时代传奇**

2023 年 Rotem Weiss 创建开源 GPT Researcher（25K Stars），2024 年正式成立 Tavily 做商业搜索 API。成为 Claude Desktop 默认搜索工具后增长爆发，2025 年收入达 $2.8M，开发者社区超 100 万。2025.08 获 Insight Partners $20M Series A，2026.02.10 被 Nebius 以 $275M 收购（含里程碑最高 $400M）。NPM 周下载 818K，MCP 生态中非 Anthropic 官方最高之一。

与 kol-api 的相似度：从开源项目起步 → 垂直 API → MCP 集成成为核心增长杠杆。Tavily 证明了 MCP 品类定义者可以在 2-3 年内实现巨额退出。

> 来源: [TechCrunch](https://techcrunch.com/2025/08/06/tavily-raises-25m-to-connect-ai-agents-to-the-web/), [Nebius 公告](https://nebius.com/newsroom/nebius-announces-agreement-to-acquire-tavily-to-add-agentic-search-to-its-ai-cloud-platform), [GetLatka](https://getlatka.com/companies/tavily.com)

**Context7 — MCP 原生产品的采用量天花板**

Upstash（$11.9M 融资，a16z 领投）2024 年仅 $1M ARR，是一家不起眼的 Serverless Redis 公司。2025 年推出 Context7 — 专为 MCP 生态设计的实时文档上下文工具。上线后迅速成为 MCP 生态第 4 大热门（月搜索量 13,000），45.2K GitHub Stars，396K 周 NPM 下载。证明 MCP 原生产品可以从零获得巨大采用量，即使母公司不知名。

局限：Context7 本身免费，通过导流到 Upstash 付费平台间接变现，不直接验证付费模型。

> 来源: [GitHub](https://github.com/upstash/context7), [NPM](https://www.npmjs.com/package/@upstash/context7-mcp), [TechCrunch](https://techcrunch.com/2024/02/08/upstashs-serverless-data-platform-hits-arr-of-1m-just-two-years-after-seed-funding/)

**21st.dev — 唯一纯 MCP 原生付费案例**

个人开发者 Serafim 创建的 AI UI 组件生成工具，完全依托 MCP 生态分发。免费 5 次 → $20/月订阅，组件发布者获 50% 收入分成。2 周内获得 4000 beta 用户，目前 ~£400/月 MRR。这是整个 MCP 生态中**唯一被广泛引用的独立开发者付费成功案例** — 证明模型可行，但规模仍小。

> 来源: [Cline Blog](https://cline.bot/blog/building-the-mcp-economy-lessons-from-21st-dev-and-the-future-of-plugin-monetization), [GitHub](https://github.com/21st-dev/magic-mcp)

#### 对 kol-api 的关键启示

| 启示 | 来源 | 对 kol-api 的意义 |
|------|------|-----------------|
| MCP 能给无名公司带来巨大**采用量** | Context7（396K 下载）、Firecrawl（350K 注册） | kol-api 可以靠 MCP 冷启动 |
| **垂直数据 API + MCP + PLG** 是被验证的最佳组合 | Firecrawl（$1.5M）、Tavily（$2.8M） | kol-api 完全符合这个模型 |
| 品类先发者拿到巨大红利 | Tavily = AI 搜索品类定义者 → $275M 收购 | 达人营销品类 99%+ 空白，先发优势更大 |
| MCP 目前更像**增长杠杆**而非唯一渠道 | 所有成功案例都有多渠道 | kol-api 不应只靠 MCP，REST API + 开源 + 社区并行 |
| 纯 MCP 付费的规模仍小 | 21st.dev 仅 ~£400/月 | 短期收入靠 API 订阅，MCP 主要是获客渠道 |
