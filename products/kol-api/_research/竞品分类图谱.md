## 一、竞品分类总览

kol-api 的竞争环境可分为 **9大类**，按与 kol-api 的竞争关系从强到弱排列：

| # | 品类 | 与kol-api关系 | 代表公司数 |
|---|------|--------------|-----------|
| 1 | 达人数据 API-First 平台 | **直接竞品** | 5+ |
| 2 | MCP / Agent 原生 | **最直接新兴竞品** | 10 |
| 3 | AI Agent 平台 (封闭) | **品类替代威胁** | 4 |
| 4 | 社交数据基础设施 | 上游供应商 / 间接竞品 | 12+ |
| 5 | 企业级网红营销 SaaS | 潜在大客户 / 间接竞品 | 9+ |
| 6 | 单平台专精 API | 细分竞品 | 5+ |
| 7 | 统一社交媒体 API | 相邻品类 | 4+ |
| 8 | 受众智能 / 社交聆听 | 互补品类 | 6+ |
| 9 | 亚洲 / 中国市场玩家 | 区域竞品 | 5+ |

## 二、数据获取方式分层模型

```
┌─────────────────────────────────────────────┐
│  Layer 4: Agent 智能层                       │
│  Stormy, Aha, GRIN Gia, kol-agent           │
├─────────────────────────────────────────────┤
│  Layer 3: 达人数据API (加工数据)    ← kol-api│
│  Modash, CreatorDB, HypeAuditor              │
├─────────────────────────────────────────────┤
│  Layer 2: 社交数据基础设施 (结构化爬取)      │
│  Data365, EnsembleData, ScrapeCreators       │
├─────────────────────────────────────────────┤
│  Layer 1: 原始数据基础设施 (代理+爬虫)       │
│  Bright Data, Oxylabs, Apify, ScrapingBee   │
├─────────────────────────────────────────────┤
│  Layer 0: 平台官方API / Creator OAuth        │
│  TikTok/IG/YT官方, Phyllo (授权数据)        │
└─────────────────────────────────────────────┘
```

kol-api 在 **Layer 3** — 加工数据层：
- 数据经过清洗、标准化、富化 (假粉检测、受众画像、定价估算)
- 比 Layer 1-2 贵但省去开发者自建管道的成本
- 比 Layer 4 灵活，开发者可以自由组合构建自己的产品

## 三、各品类详细分析

### 品类1: 达人数据 API-First 平台

> 核心特征: 以开发者为主要客户，提供自助API访问达人搜索/分析数据
> 与 kol-api 关系: **直接竞品**

| 公司 | 定价 | 数据规模 | 平台覆盖 | 亮点 | 弱点 |
|------|------|----------|----------|------|------|
| **Modash** | Discovery API $16,200/yr, Raw API $10,000/yr | 350M+ creators | IG/TikTok/YT | DX最佳, OpenAPI spec, 14天试用 | 无SDK, 无MCP, 定价高, 要Sales |
| **CreatorDB** | 平台$79+/mo, API另计 | 5M+ profiles, 130+数据点 | YT/IG/TikTok/FB/Threads | 有MCP(31 tools), AI/NLP专利, $4.7M Series A | 体量小, MCP未经安全测试 |
| **HypeAuditor** | 平台$399+/mo, API $2,999+/yr | 207.8M+ profiles | IG/TikTok/YT/Twitch/X | 假粉检测行业标杆 | DX最差, 无自助API, 需Sales |
| **influencers.club** | $249+/mo | 340M creators, 40+平台 | 40+平台 | 唯一大规模验证联系方式(邮箱+电话) | 偏外联数据, 非分析数据 |
| **Inbeat** | 公开定价 | IG+TikTok | IG/TikTok | 自助注册, 简单易用 | 平台少, 数据深度有限 |

**关键洞察**:
- Modash 是 DX 标杆但定价是 kol-api 的 7-10x
- CreatorDB 是最接近的竞品 (有MCP + 较低定价), 但数据量远小于聚星
- 全部竞品都需要 Sales call 或高门槛, **PLG自助是空白**

### 品类2: MCP / Agent 原生

> 核心特征: AI Agent 协议原生支持，为 LLM/Agent 提供工具
> 与 kol-api 关系: **最直接的新兴竞品**, 也是 kol-api 的差异化定位

| # | 名称 | 类型 | 工具数 | 开源 | 平台覆盖 | 定价 | 成熟度 |
|---|------|------|--------|------|----------|------|--------|
| 1 | **AnyTag MCP** | 全功能 | 搜索+Campaign+数据 | 闭源 | 8平台, 45+国, 1M+ KOL | 企业询价 | Production (2025.7) |
| 2 | **CreatorDB MCP** | 数据API | 31 tools | MIT | IG/YT/TikTok | ~$79+/mo | Beta/社区 |
| 3 | **Apify Agent** | 发现 | AI匹配+评分 | 部分开源(Crawlee) | TikTok为主 | $100/1K profiles | Early (19月活) |
| 4 | **HypeBridge** (Apify) | 发现 | 多平台发现+评估 | 闭源 | IG/TT/YT/LI/TW | 按用量 | Beta |
| 5 | **Zapier x IMAI** | 桥接 | InfluencerMarketing.ai动作 | 闭源 | IG/YT/TT | Zapier计划 | Production |
| 6 | **Zapier x influencers.club** | 富化 | 邮件/Handle查询+相似推荐 | 闭源 | 多平台 | Zapier计划 | Production |
| 7 | **MCPFactory pitch** | 外联 | 达人外联+内容生成 | 开源 | 非平台特定 | BYOK | Early |
| 8 | **Refersion** (Pipedream) | 联盟 | 创建/查询/积分 affiliate | 开源 | 电商 | ~$99+/mo | Production |
| 9 | **Xpoz** | 社交数据 | 搜索+情感+影响力评分 | 闭源 | TW/IG/TT/Reddit | $0-200/mo | Production |
| 10 | **Audiense Insights** | 受众 | 受众画像+达人亲和度 | 开源(GitHub) | Twitter/X | 企业 | Production |

**关键洞察**:
- **AnyTag MCP** 是唯一生产级达人营销MCP, 但面向企业, 闭源, SEA为主
- **CreatorDB MCP** 31 tools 最完整开源方案, 但社区质量, F级安全评分
- **达人营销MCP空白度仍极高** (~99.96%), 先发窗口 12-18个月
- 大平台 (Modash/NeoReach/Upfluence/CreatorIQ/Favikon/Kolr) **全部有REST API但无MCP**

### 品类3: AI Agent 平台 (封闭生态, 无MCP)

> 核心特征: 端到端AI Agent平台, 自动化达人营销全流程, 但不对外开放
> 与 kol-api 关系: **品类替代威胁** — 如果Agent够强, 开发者不需要API

| 公司 | 定价 | 创始背景 | 平台覆盖 | 关键能力 |
|------|------|----------|----------|----------|
| **Stormy AI** | $0-170/mo | YC-backed, 5/5评分(100+ reviews) | YT/TT/LI/Substack等8+ | 5个AI Agent 24/7, 自动外联+谈判+支付 |
| **Aha** (aha.inc) | $120/mo | 企业级case study | 140+平台, 5M+ creators | 多Agent团队, 动态定价算法, 合同+托管 |
| **partnrUP** | 企业询价 | gen.video + Lionize合并 | IG/TikTok | AI经理"Lilly", 收购Novel(shoppable video) |
| **GRIN Gia** | 企业询价 | 10年数据训练 | 全平台 | 首个Agentic AI for creator marketing (2025.5) |

**关键洞察**:
- Stormy 是最危险的间接竞品 — YC-backed, PLG定价, 5/5评分
- 但这些都是**封闭平台**, 不暴露API/MCP, 无法被其他开发者集成
- kol-api 的 "基础设施" 定位与这些 "应用层" 互补, kol-agent 本身就类似这个品类

## 四、定价对标矩阵

| 竞品 | 最低入门 | API年费 | 对比 kol-api ($0→$588→$1,788→$2,388+/yr) |
|------|----------|---------|------------------------------------------|
| Modash API | 需Sales | $10,000-16,200/yr | kol-api 便宜 5-7x |
| HypeAuditor API | 需Sales | $2,999+/yr | kol-api 便宜 1-5x |
| CreatorDB | $79/mo | ~$948+/yr | 最接近的价格带 |

## 五、战略启示

### 1. kol-api 的独特定位

```
kol-api = API-First + MCP原生 + PLG自助 + 亚洲数据深度 + 开发者DX
```

**目前没有任何单一竞品同时具备这5个特征。**

### 2. 最需警惕的竞品

| 竞品 | 威胁维度 | 威胁等级 | 应对策略 |
|------|----------|----------|----------|
| **AnyTag MCP** | MCP先发 + 8平台 + 生产级 | 高 | 开源+PLG差异化, 开发者社区 |
| **CreatorDB** | MCP + 低定价 + AI专利 | 中高 | 数据量(1亿 vs 5M)+NoxGPT AI搜索 |
| **Modash** | DX标杆 + 350M数据 | 中 | PLG定价颠覆(便宜7x)+MCP原生 |
| **Stormy AI** | YC-backed + PLG + Agent | 中 | kol-api是基础设施, Stormy是应用 |
| **Xpoz** | MCP原生 + $0起 + 1.5B帖子 | 中低 | Xpoz是社交数据, kol-api是达人数据 |

### 3. 潜在合作/整合对象

| 对象 | 合作模式 |
|------|----------|
| **Phyllo** | 整合其Creator OAuth获取收入/私有数据 |
| **Apify** | 在Actor市场上架kol-api wrapper |
| **Zapier** | 通过Zapier MCP桥接快速获得分发 |
| **企业SaaS** (GRIN/Impact等) | B2B白标数据供应 |
| **Xpoz** | 互补: 社交帖子数据 + 达人profile数据 |

### 4. 有REST API但无MCP的平台 (包装机会)

如果这些平台未来12个月不推MCP, kol-api 的MCP先发优势将持续扩大:

| 平台 | 数据规模 | MCP状态 |
|------|----------|---------|
| Modash | 350M+ | 无MCP |
| NeoReach | 企业级, 100+端点 | 无MCP |
| Upfluence | 12M+ | 无MCP |
| CreatorIQ | 企业级 | 无MCP |
| Favikon | 多平台, AI搜索 | 无MCP |
| Kolr | 3M+, 1BN+数据 | 无MCP |
