# 海外发布中心市场分析

> 基于 6 个超级 Agent 商店 + 13 个 MCP/Skill 平台的综合分析
> 调研日期: 2026-02-11
> 数据来源: 各平台详细报告见 `_research/发布中心/` 目录（01-13）

## 一、执行摘要

### 核心发现

**达人营销工具在整个 AI Agent 生态中存在 100% 的供给真空。** 在我们调研的 6 个超级 Agent 商店（ChatGPT/Claude Code/Copilot/Manus/ClawHub/Google）+ 13 个 MCP/Skill 平台、合计索引超过 30 万个工具/技能中，超级 Agent 商店中达人营销工具为 **零**，MCP 目录中仅有 6 个且全部有缺陷。kol-api 有 6-12 个月的黄金窗口成为该品类的事实标准。

### 五大战略结论

| # | 结论 | 置信度 |
|---|------|--------|
| 1 | 达人营销 MCP/Skill 供给缺口 > 99%，是整个生态中最大的品类空白之一 | 高 |
| 2 | 唯一的专业竞品 CreatorDB MCP 因合规问题（Glama F 评分）实际不可用 | 高 |
| 3 | 开发者获取工具的主要渠道正在从 "搜索 GitHub" 转向 "MCP 目录 + IDE 一键安装" | 高 |
| 4 | kol-api 应以 "原生 MCP Server + 多平台分发" 为核心策略，Apify 为变现补充 | 高 |
| 5 | 定价应参考 Apify PPE 模型（$0.005-0.50/事件），入门层对标 Xpoz（$20/月） | 中 |

### 3.2 已有竞品真实表现

#### 竞品矩阵

| 竞品 | 平台分布 | 工具数 | 数据源 | 评分/状态 | 实际可用性 |
|------|----------|--------|--------|----------|-----------|
| **CreatorDB MCP** | Glama | 31 tools | 官方 API | **F/F/-**（Glama） | **不可用** — 缺 LICENSE，无法安装 |
| **SociaVault MCP** | Glama/多平台 | 11 tools | 爬虫 | A/A/A | 可用但浅 — 周下载仅 2 |
| **Audiense Insights** | Glama | 8 tools | Twitter API | A/A/A | **已弃用** — 迁移到 remote |
| **Xpoz** | Smithery/mcp.so | 数个 | 1.5B 帖子索引 | — | 可用 — 但偏内容非达人 |
| **Influencer Discovery Agent** | Apify | AI Agent | 爬虫 | — | 可用 — 但单次≤50人，不稳定 |
| **influencers.club** | Zapier MCP | 3 actions | 340M 邮箱 | — | 可用 — 仅数据充实 |
| **Influencer Hero** | Zapier MCP | 13 tools | 混合 | — | 可用 — 偏 CRM 非数据 |

#### 竞品问题总结

1. **CreatorDB MCP** — 功能最全（31 tools），但因缺少 LICENSE 文件导致 Glama F 评分，标注"无法安装"。零使用量，等于隐形。
2. **SociaVault MCP** — 合规评分全 A，但周下载仅 2，功能停留在基础 profile 获取。
3. **Audiense Insights** — 曾是优质工具（全 A 评分），但已弃用。
4. **Xpoz** — 以帖子/内容搜索为核心，非达人画像数据，无 ROI/商业价值分析。
5. **Apify 系列** — 多为爬虫型，数据不稳定，单次限 50 人，无营销决策层数据。
6. **Zapier 系列** — 功能极简（2-13 actions），偏 CRM/邮箱充实，非数据分析。

**核心结论**: 达人营销品类表面上有 6 个竞品，但实际上**没有一个提供完整的、生产级的达人营销数据 API**。kol-api 的真正竞品数量为零。

### 4.2 工具发现行为画像

**开发者（构建营销工具的工程师）**:
1. 首先在 MCP 目录（Smithery / Glama / mcp.so）搜索
2. 查看 GitHub Stars、安全评分、周下载量
3. 通过 Cursor `cursor://` deep link 一键安装
4. 试用 free tier，评估后决定付费

**非开发者（营销人员/运营）**:
1. 通过 Zapier 发现可用的营销 Actions
2. 在 AI 聊天界面（Claude/ChatGPT）自然语言调用
3. 关注"无代码"、"一键配置"

**AI Agent（自动发现）**:
1. 通过 MCP Registry API 搜索可用工具
2. 基于工具描述（description）语义匹配
3. 调用 PulseMCP API 评估工具质量和流量

### 5.2 各平台具体定位

#### Smithery — "达人营销品类第一名"

- **定位**: 100K+ tools 中的达人营销标杆工具
- **差异化**: 覆盖中国平台（抖音/小红书/B站），40+ tools 超越 CreatorDB 的 31 tools
- **关键动作**: 利用 AI 语义搜索占位 "influencer"/"KOL"/"creator" 等关键词
- **冷启动**: 慷慨免费层 + Playground 在线试用

#### Glama — "全 A 评分的企业级达人数据工具"

- **定位**: Marketing 品类（仅 9 个 server）的头部，以合规碾压 CreatorDB（F 级）
- **差异化**: Day 1 获得 Security A / License A / Quality A 三 A 评分
- **关键动作**: LICENSE (MIT) + README + glama.json + GitHub Release + Claim Ownership
- **冷启动**: 引导用户通过 Glama 使用以积累 Recent Usage

#### Cursor Directory — "一键安装的达人数据引擎"

- **定位**: 295K 精准开发者流量中的达人营销独占者
- **差异化**: cursor:// deep link 零配置安装，配套 Cursor Rule 引导
- **关键动作**: 精简至 8-12 核心工具（Cursor 40 工具上限），总 token < 3,000
- **冷启动**: 配合发布一条 Cursor Rule（额外曝光渠道）

#### Apify Store — "可变现的达人营销决策引擎"

- **定位**: 营销决策层（非爬虫层）的首个专业 Actor
- **差异化**: 官方数据源（非爬虫）+ ROI 预测 + 竞品对标
- **关键动作**: PPE 定价（$0.005-0.50/事件），先 MVP 后智能 Agent
- **冷启动**: 利用 Apify MCP 原生支持 + Store SEO

#### Zapier MCP — "营销人员的 AI 达人助手"

- **定位**: 3.4M 非开发者用户中的达人营销自动化工具
- **差异化**: 无代码集成，与 Google Sheets/CRM 联动
- **关键动作**: 5-8 个精选 Actions + 10+ Zap Templates
- **注意**: 定位为获客渠道，非核心分发（1 call = 2 tasks 成本高）

#### Agent Skill 生态 — "达人营销知识层标杆"

- **定位**: 160K+ Skills 中唯一的达人营销数据 Skill
- **差异化**: MCP Server（工具层）+ Skill（知识层）组合拳
- **关键动作**: 独立 GitHub 仓库 → Skills.sh/SkillsMP 自动收录 → Anthropic Partner Skill PR
- **注意**: Skill 为辅助引流，MCP Server 为核心

### 6.2 kol-api 建议定价框架

#### 原生 MCP Server（主线）

| Tier | 月费 | 请求数 | 目标用户 |
|------|------|--------|---------|
| **Free** | $0 | 200/月 | 试用+冷启动（对标 Xpoz 免费层） |
| **Starter** | $19/月 | 5,000/月 | 独立开发者/小团队 |
| **Pro** | $49/月 | 20,000/月 | 中型营销团队 |
| **Enterprise** | 定制 | 无限 | 企业 API 接入 |

**定价逻辑**:
- Free 层要足够慷慨（200 次/月），因为 MCP 生态用户期望"即用即走"
- 入门价 $19/月 低于 CreatorDB（$79/月），形成价格优势
- 对标 Xpoz（$20/月）但提供更深度的达人画像数据

#### Apify Actor（变现线）

| 事件 | 单价 | 说明 |
|------|------|------|
| `search` | $0.01/次 | 每次搜索查询 |
| `profile` | $0.005/份 | 基础达人档案 |
| `analytics` | $0.02/份 | 深度分析（含受众画像） |
| `discovery` | $0.10/次 | AI 智能达人匹配 |
| `report` | $0.50/份 | 完整营销报告 |

**PPE 优势**: 用户按使用付费，无门槛试用；创作者获得 80% 收入（Apify 分成模型）。
