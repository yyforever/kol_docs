## 二、各维度详细评分

### 1. 痛点强度：4.0/5

**评分理由**：
- 达人机会发现难是真实痛点，70%达人被动等DM
- 价格不透明问题严重，达人不知道合理报价
- 佣金敏感度高，45%达人因15%佣金不用平台
- 支付风险存在，品牌逾期、延迟支付

**证据**：

| 证据 | 来源 |
|------|------|
| 市场规模$32.55B(2025)，CAGR 23.3% | 行业报告 |
| 70%品牌计划主要使用微/纳米网红 | Collabstr 2024 |
| 45%达人因佣金太高不用平台 | 用户调研 |
| DM回复率<5% | 品牌调研 |
| Micro达人占市场75.9% | 行业报告 |

### 3. 聚星优势：2.5/5

**评分理由**：
- C2网红数据库可用于达人供给，但需产品整合
- C1品牌客户可作为任务供给，但与主业定位有冲突
- 主业是品牌侧工具，达人侧是新方向，需独立验证PMF
- **核心风险**：与主业偏离，资源分散

**可利用的聚星资源**：

| 资源 | 是否可用 | 如何利用 |
|------|---------|---------|
| C1 付费客户 | ⚠️ 部分 | 品牌可发任务，但需重新定位 |
| C2 网红数据库 | ✅ 是 | 导入达人供给侧 |
| C3 品牌方数据 | ⚠️ 部分 | 品牌需求洞察 |
| C4 网红发现算法 | ✅ 是 | AI匹配推荐 |
| C5 大规模建联技术 | ❌ 否 | 达人侧不需要批量建联 |
| C6 AI Agent技术 | ✅ 是 | 智能客服、自动匹配 |
| B1-B4 其他 | ⚠️ 部分 | 需评估 |

### 4. PLG可行性：3.5/5

**评分理由**：
- 竞品已验证PLG模式可行（Collabstr、JoinBrands都是自助模式）
- 达人侧可免费注册+低佣金，品牌侧订阅收费
- **主要挑战**：双边市场冷启动极难

**参考同类产品**：

| 产品 | 获客方式 | PLG程度 | 冷启动策略 |
|------|---------|---------|-----------|
| Collabstr | 自助+内容营销 | 高 | 达人先行 |
| JoinBrands | 自助+社媒推广 | 高 | UGC需求驱动 |
| Afluencer | 自助+免费版 | 高 | 0佣金吸引达人 |

**建议PLG策略**：
```
达人侧：
├─ 免费注册+0-5%佣金（远低于竞品15-20%）
├─ 增值服务付费（认证、数据分析）
└─ 邀请返利机制

品牌侧：
├─ 免费版：5个邀请/月
├─ Pro（$49/月）：50个邀请+筛选器
├─ Growth（$99/月）：100个邀请+分析
└─ Enterprise：定制
```

## 三、结论

### 核心优势

1. ✅ **市场规模大**：$32.55B (2025)，CAGR 23.3%
2. ✅ **中文差异化**：所有国际竞品无中文，服务出海达人
3. ✅ **协同机会**：可与02A CRM形成闭环
4. ✅ **趋势确定**：微达人崛起，70%品牌偏好

### 主要风险

1. ❌ **与主业偏离**：聚星核心是品牌侧，达人侧需独立验证
2. ❌ **双边冷启动极难**：需同时吸引品牌和达人
3. ❌ **竞争激烈**：JoinBrands已有0佣金+250K达人
4. ❌ **2C获客成本高**：达人分散，获客效率低

### 建议

**短期（不推荐独立做）**：
- 优先做02A达人CRM（4.05分），从品牌侧切入
- 在02A中增加"任务发布"功能，逐步延伸到达人侧

**中长期（如果决定做）**：
1. **定位**：中国出海达人专属平台
2. **差异化**：中文支持+AI匹配+0佣金
3. **冷启动**：导入C2数据库+种子任务补贴
4. **协同**：与02A CRM深度整合，共享品牌资源
