## 二、核心竞品深度分析

### 2.1 Euka AI（最强新进入者）

**基本信息**：
- 官网：euka.ai
- 定位：首个端到端TikTok Shop生态解决方案
- 成立：2024年
- 融资：获Creator Ventures、NYX Ventures投资
- 营收：$2.1M（截至2025年11月）
- 客户：2000+品牌，200+头部品牌

**核心功能（9大工具集成）**：
1. AI达人发现（350万+库）
2. 自动外联（1000 DM/天 + 2500邀请/天）
3. 达人CRM
4. 内容管理
5. 样品管理
6. 佣金管理
7. 数据分析
8. AI内容优化
9. 自动支付

**竞争优势**：
- TikTok官方API早期访问权
- 端到端完整覆盖
- AI驱动自动化程度最高
- 平均提升36%销售

**定价**：
- 定价不公开，需联系销售
- 每个TikTok Shop店铺需单独许可证
- 推测：企业级定价（$199+/月）

**标杆案例**：
- SharkNinja：GMV 171万→1816万（120天内）

### 2.6 TikTok Shop官方工具（Affiliate Center / Manage Creators）

**官方工具功能**：

| 功能 | Affiliate Center | Manage Creators |
|------|-----------------|-----------------|
| 数据查看 | GMV、订单、佣金 | 达人90天表现 |
| 标签管理 | - | 最多100个标签 |
| 批量邀请 | - | 一次50-100人 |
| 批量消息 | - | 每人限5条 |
| 数据导出 | ✅ | - |

**核心限制**：
1. **数据时限**：Manage Creators仅90天历史
2. **批量上限**：一次最多邀请100人
3. **消息限制**：每人5条，需等回复才能继续
4. **标签上限**：100个标签，每人最多10个
5. **无法真正移除**：移除只是隐藏，不能解除合作
6. **无AI发现**：没有智能搜索和匹配

**API开放情况**：
- TikTok已开放Affiliate API（OAuth 2.0）
- 有速率限制（1分钟滑动窗口）
- 早期合作伙伴：Euka、Social Snowball、Superfiliate、Canal

**官方认证合作伙伴**：
- 广告管理：Flywheel、Helium10、Pacvue、Quartile、Teikametrics
- 达人发现：Grin、Aspire、Upfluence
- 物流返品：ReBound

**为什么需要第三方工具**：
1. 官方无AI发现能力
2. 数据留存期太短
3. 批量操作能力有限
4. 缺少高级分析报告

## 六、关键洞察

### 6.1 市场趋势

1. **AI驱动升级**：所有新进入者都强调AI能力
2. **端到端整合**：从单点工具→完整解决方案
3. **多平台扩展**：Cruva计划支持Instagram/YouTube
4. **中英市场分裂**：达密统治中文，Euka/Cruva主攻英文
5. **定价透明化**：FastMoss/Kalodata定价清晰，管理工具仍需咨询

### 6.2 聚星的机会

1. **全平台数据**：聚星75M达人（vs Euka 3.5M、达密8M）
2. **中国出海客户**：40万中国出海客户
3. **TikTok Shop官方关系**：待确认API权限
4. **AI Agent技术**：可用于自动化建联

### 6.3 风险点

1. **Cruva/Euka快速增长**：市场窗口正在关闭
2. **TikTok政策不确定性**：美国禁令风险
3. **达密中文市场垄断**：需要差异化突破
