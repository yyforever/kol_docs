## 重点竞品详情

### 1. GRIN（美国，高端）

| 维度 | 详情 |
|------|------|
| **定价** | ~$2,500/月起，强制年付 |
| **数据库** | 1.9亿达人 |
| **核心功能** | Shopify集成、产品gifting、付款结算、affiliate追踪 |
| **优点** | 功能最全、电商集成最强 |
| **缺点** | 太贵、学习曲线陡、邮箱数据不准、强制年付 |
| **适合** | 大型DTC品牌 |

---

### 2. Aspire（美国，高端）

| 维度 | 详情 |
|------|------|
| **定价** | ~$2,000/月起，需年付 |
| **数据库** | 1.7亿达人 |
| **核心功能** | 达人CRM、UGC权限管理、创作者市场 |
| **优点** | CRM体验好、界面友好 |
| **缺点** | 核心功能锁高级版、加载慢、无月付 |
| **适合** | 中大型品牌 |

---

### 3. Modash（爱沙尼亚，中端）⭐性价比

| 维度 | 详情 |
|------|------|
| **定价** | $199/月起（年付），比GRIN便宜85% |
| **数据库** | 3.5亿达人 |
| **核心功能** | AI视觉搜索、批量外联、Shopify集成 |
| **优点** | 透明定价、无需年合同、14天免费试用 |
| **缺点** | 无内置外联工具、CRM基础、小城市数据弱 |
| **评分** | G2 4.5-4.9/5 |

---

### 4. Favikon（法国，轻量）⭐B2B首选

| 维度 | 详情 |
|------|------|
| **定价** | $69-299/月（年付）|
| **数据库** | 10-15M达人 |
| **核心功能** | AI Influence Score、LinkedIn覆盖最强、9平台 |
| **优点** | 定价低、B2B/LinkedIn独特、透明定价 |
| **缺点** | 无电商集成、无礼品/联盟功能、聚焦发现不做执行 |
| **适合** | B2B/SaaS公司 |

---

### 5. 新榜小豆芽（中国，本土）

| 维度 | 详情 |
|------|------|
| **定价** | 免费版20账号、VIP ¥59/月、SVIP ¥79/月 |
| **支持平台** | 抖音、小红书、快手、B站、视频号（50+平台）|
| **核心功能** | 多账号管理、一键分发、指纹浏览器防关联、AI创作 |
| **优点** | 新榜生态可接广告、AI创作丰富 |
| **缺点** | **不支持海外平台**、2024年涨价后用户吐槽 |

---

### 6. 蝉妈妈（中国，数据分析）

| 维度 | 详情 |
|------|------|
| **定价** | 年费1.2万（专业版）|
| **核心功能** | 达人数据、带货数据、选品分析 |
| **优点** | 数据全面 |
| **缺点** | 数据延迟>2小时、自动扣费投诉多、数据造假嫌疑 |
| **投诉** | 2025年电诉宝排名第8，退款问题占66.6% |

---

### 7. Stormy AI（美国，新进入者）⭐AI Agent

| 维度 | 详情 |
|------|------|
| **定价** | $350/月 |
| **融资** | $500K（YC支持）|
| **核心功能** | 全自主AI Agent：发现→外联→谈判→签约 |
| **创新点** | 自然语言搜索、AI自动发DM/邮件、协助谈判 |
| **上线** | 2025年8月（Product Hunt）|

---

### 8. BrandChamp（美国，Ambassador专用）

| 维度 | 详情 |
|------|------|
| **定价** | ~$195/月起（4档套餐）|

| **优点** | 中小品牌友好、Ambassador专精 |
