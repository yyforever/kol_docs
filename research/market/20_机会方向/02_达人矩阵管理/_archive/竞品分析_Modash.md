# Modash 竞品分析

> 调研时间：2026-02-04
> 定价验证时间：2026年2月（官网）

---

## 核心结论

**Modash是综合红人营销平台，$199/月起，CRM是功能之一但非核心。**

- ✅ 综合平台：达人发现+管理+追踪+付款
- ✅ 有基础CRM功能
- ❌ 无矩阵管理
- ⚠️ $199/月这个价格带不是空白

---

| 维度 | 信息 |

## 三、功能范围

### 达人发现 ✅ 强
- 350M+ 数据库
- AI搜索 + 37种筛选器

### CRM功能 ⭐⭐⭐ 中等
| 功能 | 支持 |
|------|------|
| 列表管理 | ✅ |
| 状态追踪 | ✅ |
| 标签 | ✅ |
| 批量操作 | ✅ |
| **Pipeline可视化** | ⚠️ 基础 |
| **矩阵管理** | ❌ 无 |
| **高级工作流** | ❌ 缺少 |

### 批量建联 ✅ 有
- 每天100封邮件
- 自动序列

### 内容追踪 ✅ 强
- 自动内容收集（无需达人注册）
- CPM/ROAS追踪

### 支付 ✅ 有
- 180国36货币全球付款

### 电商集成 ✅ 强
- Shopify一键集成
- 产品赠送/联盟/销售归因
