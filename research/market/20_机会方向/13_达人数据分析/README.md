# 13号方向：TikTok Shop达人合作机会平台

> 调研完成时间：2026-02-05（第5轮）
> 评分：**4.28/5** ✅ 推荐
> 定位：帮达人发现高佣金品牌合作机会，用真实交易数据做匹配

---

## 产品定义

**产品名**：TikTok Shop Creator Marketplace（达人合作机会平台）

**一句话描述**：
> 帮TikTok Shop带货达人发现高佣金品牌合作机会，用真实交易数据做匹配。

**核心功能**：

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **品牌机会浏览** | 显示正在找达人的品牌、佣金率、要求 | P0 |
| **智能匹配** | 根据达人粉丝画像推荐适合的品牌 | P0 |
| **一键申请** | 直接申请品牌合作 | P0 |
| **佣金对比** | 显示该品牌 vs 行业平均佣金率（聚星差异化）| P0 |
| **合作历史** | 达人与品牌的合作记录（CRM模块）| P1 |

---

## 为什么是这个产品

| 对比 | Aspire/Collabstr | **本产品** |
|------|-----------------|-----------|
| 覆盖平台 | 全平台通用 | **TikTok Shop专用** |
| 数据来源 | 品牌自己填写 | **真实交易数据** |
| 佣金信息 | 品牌声称的佣金率 | **历史实际支付佣金率** |
| 市场状态 | 有成熟产品 | **TikTok Shop专用空白** |

> ⚠️ **关键洞察**：通用creator marketplace很多，但**TikTok Shop专用的反向市场完全空白**。聚星10%交易数据是核心差异化。

---

## 快速摘要

**结论**：✅ 推荐作为独立产品方向

**核心原因**：
1. **竞争空白最大** — TikTok Shop专用的达人找品牌平台空白
2. **聚星优势最强** — 10%交易数据提供真实佣金率（竞品无法复制）
3. **商业模式成熟** — 达人免费+品牌付费模式已验证（Aspire）
4. **获客优势** — 触达几乎全部有价值达人，响应率约10%

**待验证**：
- 聚星能否提供"正在找达人的品牌"列表
- 品牌历史佣金率数据能否用于产品展示
- 达人愿意通过平台申请还是直接联系

## 目标用户

**核心用户**：中腰部TikTok Shop带货达人

| 特征 | 描述 |
|------|------|
| 粉丝量 | 50K-500K |
| 月佣金收入 | $2K-30K |
| 合作品牌数 | 5-20个 |
| 核心诉求 | 找到更多高佣金品牌合作机会 |

**市场规模**：约15,000-20,000目标用户

---

## 聚星差异化

| 聚星资源 | 对产品价值 |
|---------|-----------|
| **10%市场交易数据** | 显示品牌真实历史佣金率（核心差异化）|
| **达人触达渠道** | 主动推送品牌机会给达人 |
| **品牌方数据** | 知道哪些品牌正在找达人 |

**竞品无法复制**：Aspire等通用平台没有TikTok Shop交易数据，无法提供真实佣金率。
