# 13号方向：TikTok Shop达人收益仪表盘

> 调研完成时间：2026-02-05（第4轮）
> 评分：**4.12/5** ✅ 推荐
> 定位：帮达人追踪多品牌佣金收入，对比行业基准

---

## 产品定义

**产品名**：TikTok Shop Creator Earnings（达人收益仪表盘）

**一句话描述**：
> 帮助TikTok Shop带货达人追踪多品牌佣金收入，对比行业基准，为谈判提供数据支撑。

**核心功能**：

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **佣金汇总面板** | 多品牌佣金一目了然 | P0 |
| **净收入计算** | 扣除退款退货，显示真实收入 | P0 |
| **佣金基准对比** | "你 vs 同级别达人平均"（聚星差异化）| P0 |
| **到账预测** | 预测待到账金额和时间 | P1 |
| **变化预警** | 佣金率下调、保护期到期提醒 | P1 |

---

## 快速摘要

**结论**：✅ 推荐作为独立产品方向

**核心原因**：
1. **竞争空白完全确认** — 达人收益追踪无工具
2. **聚星差异化明确** — 10%市场数据提供佣金基准（竞品无法复制）
3. **获客优势** — 触达几乎全部有价值达人，响应率约10%
4. **产品定义清晰** — 可直接进入开发

**待验证**：
- TikTok Affiliate API是否开放佣金数据
- 聚星10%数据是否足够形成基准

## 聚星差异化

|---------|-----------|
