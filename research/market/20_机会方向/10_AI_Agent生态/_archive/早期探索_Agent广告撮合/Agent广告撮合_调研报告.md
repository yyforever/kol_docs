## C方向定义（R2修正）

原始定义(R1)："达人GEO"——帮品牌通过达人内容在AI生态中获得推荐

**R2重大修正**：R1核心因果链存在概念偷换。

| R1假设 | R2验证 |
|--------|--------|
| "affiliate content被AI引用=达人内容影响AI" | 🔴 **概念混淆**。AI引用的affiliate content主要是Wirecutter/Forbes Advisor等专业编辑评测站，不是达人个人内容 |
| "89%引用来自earned media=达人机会" | 🟡 实际82-85%，且earned media≠达人内容（earned media=第三方媒体报道）|
| "40%含联盟内容" | 🔴 Evertune CEO self-serving claim，定义模糊，不可独立验证 |
| "40万客户交叉销售" | 🔴 GEO买家是SEO经理，达人营销买家是品牌经理，**几乎不重叠** |

**AI实际引用排名**（First Page Sage, 36,127条购买意图查询）：
1. 专业评测站（Wirecutter/Tom's Guide）: 7,642次
2. 消费者评论平台: 5,983次
3. 传统媒体: 4,581次
5. **YouTube视频评测: 3,211次** ← 唯一达人相关高引用渠道
11. 达人/创作者网站: **928次**（仅为评测站的1/8）

**R2修正后定义**：待步骤1-8完成后重新定义

## 一、场景画像

### 1.1 GEO市场全景

| 指标 | 数据 |
|------|------|
| 2025市场规模 | $8.5亿 |
| 2034预测 | $337亿（CAGR 50%） |
| GEO工具融资(2024-2025) | $77M+（Profound $58.5M/Evertune $19M/AirOps $40M等） |
| 品牌认知度 | 85% CMO视GEO为关键优先级 |
| 实际投入 | 仅10-15%品牌系统性投入 |
| 预算来源 | 60-70%从SEO转移，达人预算仅5-10%（刚萌芽）|

### 1.2 R2修正：达人GEO的前提假设被部分推翻

**R1三个信号的R2验证**：
1. ✅ Profound创始人来自达人营销（Kyra）——事实成立
2. 🔴 "AI引用源40%+是联盟内容" → **self-serving claim**，"联盟内容"主要指Wirecutter等专业媒体，不是达人
3. 🟡 UGC/社区内容占AI引用约50% → Reddit部分成立，但达人个人内容在AI引用中仅排#11

**R2新发现**：
- AI搜索流量绝对值仍小（~0.2-1%），但增长527% YoY
- AI搜索转化率是Google的4.4倍
- GEO实际买家是SEO经理（非达人营销经理），买家画像和聚星客户不匹配
- $0-199/月PLG可行：单用户成本$1.25-2.00/月，毛利85%+
- 没有PLG模式的GEO工具成功案例——整个品类都是销售驱动

### 1.3 AI购物推荐生态

| 平台 | 机制 | 达人维度 | 商业模式 |
|------|------|---------|---------|
| ChatGPT Instant Checkout | 商家提交产品Feed→有机排名→对话内结账 | ❌ 无标准化达人通道 | 4%交易费 |
| Perplexity Shopping | Shopify/Amazon实时数据→即买 | ❌ 无 | 暂不收佣 |
| Google AI Shopping | Gemini+Shopping Graph(500亿商品) | ❌ 无 | 有Sponsored Listings |

> **"达人内容真空"**：达人是消费者最信任的推荐源，但在AI Agent Commerce中没有标准化通道。

### 1.4 品牌AI可见度需求

| 维度 | 现状 |
|------|------|
| 追踪工具 | 20+专用GEO工具已上线 |
| 品牌认知 | 不再太新（2025下半年转折），但术语未标准化(GEO/AEO/AIO/LLMO) |
| Agency | 50+家卖GEO服务，方法论同质化，**无人做达人路径** |
| "达人→AI"认知 | ⭐ **极低**——AI抓取达人内容已被确认，但几乎无人作为策略系统执行 |

**市场三层结构**：监测(拥挤) → 优化(拥挤) → **影响源(空白)**。聚星定位在第三层。

### 1.5 "达人GEO"产品形态

| 层 | 功能 | 差异化 |
|----|------|--------|
| Monitor | 追踪达人内容在AI回答中的出现频率 | 低（GEO工具可扩展）|
| **Match** | 帮品牌选"AI友好"的达人（**AI引用力**新指标）| **最高** |
| Optimize | 生成AI友好型达人Brief | 中 |
| Attribute | 追踪"达人→AI引用→购买"全链路 | 高 |

**定价参考**：$300-700/月(Monitor) → $4,000-7,000/月(Enterprise Pro)

### 1.6 已有案例

- **Brooklinen**：已付费让达人创作内容，知道AI会爬取，但靠直觉没有工具
- **Evertune × impact.com**：2025年12月刚宣布合作，最接近但是B2B partnership非独立产品
- **MikMak**：明确提出达人campaign增加UGC→AI引用，但只是概念没做产品

> **窗口期：12-18个月。**

## 七、JTBD分析

**Core Job**: When 品牌营销经理看到AI搜索流量增长 → I want to 通过达人内容在AI推荐中可见 → So I can 抢占这个新流量入口

**Top 3 Underserved Outcomes**:
1. 达人内容→AI引用的精准归因
2. 达人"AI引用力"评分（新指标）
3. 从发现缺口→达人执行的闭环

**Switch Trigger**: ChatGPT Instant Checkout上线 = AI可见度直接等于营收

### R2 JTBD修正

R1 Core Job **不成立**："品牌通过达人内容在AI推荐中可见"建立在被推翻的假设上。

修正为两个独立Job：
- **Job A（品牌/SEO团队）**：监测AI可见度+识别影响它的off-site内容源
- **Job B（创作者）**：看到自己内容在AI世界的存在感，用于品牌谈判 — **完全未验证**

### R2最终建议：三路径评估

| 路径 | 判定 | 理由 |
|------|------|------|
| A. 独立品牌SaaS | 🔴 **NO-GO** | 两个根基假设被推翻，市场太小 |
| B. 聚星增值模块 | 🟡 有条件 | 低风险低回报，作为数据积累 |
| **C. Creator AI Citation Power** | 🟡/🟢 **推荐验证** | 1人3周+$100验证核心赌注 |

**推荐策略**：
1. 先花3周验证路径C——做一个免费"YouTube AI引用力检测"工具，核心指标是创作者分享率是否>10%
2. 中了→全力推进创作者飞轮
3. 没中→退回路径B，"AI引用力"作为A方向API的第三个独特数据维度

### C方向跨方向复用价值

即使C独立产品No-Go，R2研究有高价值复用：
1. **"AI Citation Power Score"** → A方向API的第三个独特维度（全球零竞品）
2. **免费检测工具** → A方向API+聚星主站引流
3. **GEO工具数据供应商** → Profound/Evertune的达人数据API
