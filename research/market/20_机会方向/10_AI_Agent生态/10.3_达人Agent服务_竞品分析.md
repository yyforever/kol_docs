## 九、战略决策

**10.3应聚焦海外市场**（TikTok/YouTube/Instagram），暂不做中国市场达人端数据分析类产品。

如果做中国市场，仅限：
1. AI内容创作工具（不涉及平台数据）
2. 服务出海达人（帮中国达人做海外）

### 10.4 混合模式案例

**Beacons for Brands（2025年4月）**：
- 700万+达人用户基础
- 先聚焦达人端3年，才推品牌端
- 100%佣金策略（达人），品牌按效果付费
- 成功因素：第一方数据优势

- 7000万+用户，市占79.95%

1. 需要**百万级达人**才有品牌吸引力
2. 第一方数据是核心竞争力
3. 100%佣金保持达人粘性
4. Affiliate→Brand Deal自然过渡

## 十一、R2竞品深挖（步骤2）⭐新增

### 11.1 Agentio（$340M估值）

| 维度 | 详情 |
|------|------|
| **商业模式** | 品牌付**~20%平台费**，创作者**完全免费** |
| **核心价值** | Google Ads式程序化购买创作者广告 |
| **AI差异化** | 匹配45天→**<1天**，全流程自动化 |
| **客户** | 100+企业品牌（Uber/DoorDash/CashApp） |
| **效果案例** | Bombas ROAS提升**5.3x** |

### 11.2 Beacons for Brands

| 维度 | 详情 |
|------|------|
| **商业模式** | Affiliate Marketplace**免费接入**+CPA抽佣 |
| **增长路径** | Link-in-Bio免费（700万达人）→ 200+ MCN → 品牌端 |
| **独特优势** | **双向透明**（唯一让双方都能看完整漏斗） |
| **病毒循环** | 用户看到链接→5次操作创建→放入Bio→循环 |

### 11.3 Modash（中端定位）

| 维度 | 详情 |
|------|------|
| **定价** | $199-499/月（比Aspire便宜5-10倍） |
| **数据库** | 350M+创作者（业内最大），爬虫抓取公开数据 |
| **聚焦** | Shopify电商生态 |
| **CEO预测** | 18个月后三大玩家：CreatorIQ(Fortune 100)、Modash(电商)、垂直玩家 |

### 11.4 冷启动策略研究

**关键发现**：
- **最小可行规模**：50-100个高质量创作者（10K+粉丝）可吸引第一个品牌
- **核心原则**：密度>数量，100个高契合创作者 > 10,000个泛泛创作者
- **Beacons增长秘诀**：病毒循环 + 搭乘TikTok红利 + 极简Onboarding

**分阶段增长建议**：
| 阶段 | 创作者数量 | 品牌策略 |
|------|------------|----------|
| 1 | 20-50人 | 手工匹配，亲自介绍 |
| 2 | 50-200人 | 自助Discovery |
| 3 | 200-1000人 | 品类扩展 |
| 4 | 1000+人 | 全面marketplace |
