### 12.1 Ghost AI（onghost.com）— 最纯粹的达人Agent

#### 产品定位

Ghost AI是目前市场上**最明确站在达人立场**的AI经纪人产品。定位："AI Talent Agent to Close More Deals"。公司主体为Ghost Fitness, Inc.（特拉华州注册），总部洛杉矶，团队规模1-10人。

#### 功能清单（一手来源）

| 功能模块 | 详细描述 | 自动化程度 |
|---------|---------|:----------:|
| **Gmail邮件Agent** | 在Gmail收件箱中（桌面+移动端）自动草拟对品牌的回复 | AI草稿→人类点击发送 |
| **谈判战术引擎** | 训练了专业谈判策略：反驳低报价、构建价值框架、建议还价方案 | AI建议→人类决定 |
| **品牌外联** | 帮达人识别"梦想品牌"，生成个性化外联邮件（非群发模板） | AI生成→人类发送 |
| **邮件Campaign** | AI根据达人品牌调性生成邮件内容，自动检查遗漏（如缺少按钮链接） | AI检查+建议改进 |
| **AI推荐优化** | 悬浮"!"图标提示可改进项目 | 自动提醒 |
| **日程管理** | 保持排期有序 | 辅助型 |

#### "AI草稿+人类确认"的具体流程

```
1. 品牌邮件进入达人Gmail收件箱
   ↓
2. Ghost AI自动识别品牌询价/合作邀约
   ↓
3. Ghost分析品牌信息+达人历史报价+市场数据
   ↓
4. AI在Gmail中生成回复草稿（嵌入达人的语气和调性）
   ↓
5. 如果品牌报价偏低：AI自动草拟还价邮件
   - 策略包括：反驳低报价、构建达人价值论据、建议具体还价数字
   ↓
6. 达人审阅草稿（桌面或移动端）
   ↓
7. 达人点击"发送" —— 这是唯一需要人类操作的步骤
   ↓
8. 如果品牌继续还价：重复步骤3-7
```

**核心设计原则**：All you have to do is hit send（你只需要点击发送）

#### 定价模式

| 维度 | 详情 |
|------|------|
| 付费模式 | 月付制，无合同、无承诺 |
| 佣金 | **0%**（不抽成交，区别于传统经纪人的10-25%） |
| 具体价格 | **未公开**（网站不展示具体金额，需直接联系或注册后查看） |
| 已知层级 | Free plan + Starter plan（有品牌水印），更高层级去除水印 |

#### 用户规模与融资

| 维度 | 数据 |
|------|------|
| 用户规模 | "数千创作者"（thousands of creators）— 具体数字未公开 |
| 融资 | **$500K**（Wellfound数据） |
| 团队 | 1-10人 |
| TikTok活跃度 | 有活跃的达人分享使用体验 |
| 性能声称 | 达人收入提高40-60%，成交速度2-3x |

#### 与品牌Agent的对接

**当前状态**：Ghost AI**没有标准化的Agent间接口**。

- Ghost通过**邮件**与品牌沟通，不是API对API
- 品牌侧是否用AI Agent不影响Ghost的工作——Ghost只处理邮件文本
- 没有公开的API、Webhook、或Agent Card暴露机制
- **本质是"邮件层面的半自动化"，非"Agent层面的协议交互"**

#### 关键洞察

1. Ghost证明了**达人愿意为AI经纪人付费**，且核心场景是**谈判和外联**
2. "AI草稿+人类发送"是当前唯一被验证的达人信任模式
3. **局限性**：仅限邮件渠道，不覆盖DM/微信等渠道；无标准化Agent接口
4. **竞争壁垒薄弱**：核心能力是LLM prompt engineering，无数据壁垒

### 12.2 Retrograde（Emma AutoPilot）— 最像传统经纪人的AI Agent

#### 产品定位

Retrograde是爱尔兰都柏林初创公司，由Grace Beverley（TALA创始人）、Jake Browne、Gary Meehan于2024年创立。定位：**用AI AutoPilot替代传统经纪人**。

#### 核心产品：Emma

| 维度 | 详情 |
|------|------|
| **产品名** | Emma（AI AutoPilot） |
| **人格化** | 每个达人配对一个专属AutoPilot，有独立名字、头像和声音 |
| **覆盖渠道** | 邮件（对品牌）+ SMS/WhatsApp（对达人） |
| **核心流程** | 管理收件箱→识别品牌合作→谈判合同→追付款→发发票 |
| **自动化程度** | 24/7运行，达人通过一键确认/拒绝控制决策 |
| **达人控制权** | 最终审批权在达人——不满意则AI重新谈判或拒绝 |

#### 功能清单

| 功能 | 描述 |
|------|------|
| 收件箱管理 | 自动分类品牌邮件、初筛合作机会 |
| 合同谈判 | AI与品牌通过邮件谈判条款和价格 |
| 创意Brief管理 | 处理品牌Brief，确认交付要求 |
| 内容审批追踪 | 管理内容审批流程 |
| 发票+追款 | 自动生成发票并追踪付款 |
| 实时仪表盘 | Retrograde 2.0新增：绩效追踪、Deal状态 |
| 短信/WhatsApp通知 | 重要Deal节点实时通知达人 |

#### 定价与商业模式

| 维度 | 详情 |
|------|------|
| **佣金** | **10%**（传统经纪人20%+的一半） |
| **预付费** | 无预付费用 |
| **合同** | 无合同承诺 |
| **推荐奖励** | 推荐新达人可获5%佣金分成 |
| **首笔Deal** | **完全免费**（2025.04起，任何人30秒注册即可使用） |

#### 融资与规模

| 维度 | 数据 |
|------|------|
| 融资 | **EUR 1.8M pre-seed**（Elkstone领投 + 天使投资人含Wayflyer联合创始人Jack Pierse） |

#### 关键洞察

## 十七、R3竞品深挖（步骤2）⭐新增

> 调研时间：2026-02-07

### 17.1 GRIN Gia深挖结论

- **优势**：先发（2025.05发布）、十年专有数据、品牌侧工作流自动化完整。
- **弱点**：高价格（市场口径$999-2500+/月）、用户反馈有稳定性与体验问题、部分平台数据能力受限。
- **对10.3启示**：不建议正面同质化竞争，应以更低门槛、更稳定体验、更清晰达人价值主张切入。

### 17.2 Beacons vs Agentio商业模式对比

| 维度 | Beacons | Agentio |
|------|---------|---------|
| 模式 | Freemium工具平台 | 封闭双边市场 |
| 获客 | 低门槛规模化（700万+创作者） | 精选高质量供给+企业品牌 |
| AI形态 | 辅助型（Copilot） | 自动化撮合（Agent级） |
| 变现 | 订阅+交易费 | 平台撮合费（品牌侧） |

**结论**：二者并非替代关系，而是两条路径。10.3更适合“先低门槛工具获客，再叠加高价值撮合层”的分阶段策略。

### 17.3 Ghost AI vs Retrograde（达人侧）

- **Retrograde**：更接近“AI经纪人”定位（AutoPilot + 10%佣金）。
- **Ghost**：当前公开信息更多体现为邮件/外联辅助能力，完整经纪闭环能力边界更窄。
- **结论**：近期直接对手更应按“Retrograde类产品”定义，重点比较其佣金模式与成交效率。

### 17.4 新进入者战略（2025-2026）

- 融资热度高，AI原生玩家密集进入。
- 共性策略：**AI原生架构 + 垂直深耕 + 自动化优先 + ROI可视化**。
- 冷启动共性：先打垂直场景、先做可量化结果、再扩平台边界。

### 17.5 步骤2综合判断

1. 红海不是“有没有竞品”，而是“有没有结构性差异化”。
2. 10.3的差异化应落在：
   - 达人侧立场（而非品牌侧效率）
   - 合同/信用/风控等高信任环节
   - 跨平台统一商务工作流
3. 进入步骤3时，痛点验证要聚焦“达人真实损失（钱/时间/机会）”，避免只做功能对比。
