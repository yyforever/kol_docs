## 核心竞品深挖

### 1. Modash（最直接竞品）

**基础信息**：
- 公司：爱沙尼亚，2019年成立
- 融资：未公开大额融资
- 客户：Agency、SaaS构建者

**API能力**：

| API类型 | 功能 | 定价 |
|---------|------|------|
| **Discovery API** | 搜索/筛选达人数据库 | $16,200/年起 |
| **Raw API** | 实时拉取单个达人数据 | $10,000/年起 |
| **Collaborations API** | 品牌合作历史 | 包含在上述定价 |

**定价模式**：
- 年度承诺必须
- Credit制（不同endpoint消耗不同）
- 超额有overage费用

**技术特点**：
- 文档质量高（docs.modash.io）
- 30分钟内可完成首次调用
- 支持AI搜索endpoint

**弱点**：
- 只覆盖3平台（IG/TikTok/YT）
- 无MCP支持
- 定价门槛高，无PLG
- 年度承诺对小团队风险大

### 2. HypeAuditor

**基础信息**：
- 定位：欺诈检测+分析为主
- 数据库：219M+达人

**API能力**：
- Discovery API
- Fraud Detection API
- Audience Analytics API

**定价**：~$30,000+/年（企业定制）

**特点**：
- 欺诈检测能力强
- 5平台覆盖（含X和Twitch）
- 无公开文档（需Sales）

**弱点**：
- 定价极高
- 无自助接入
- 无MCP

### 3. Phyllo（不同定位）

**定位**：认证数据API（Creator OAuth）

**特点**：
- 需Creator主动授权
- 数据准确度高（一手数据）
- 20+平台覆盖

**定价**：自定义

**适用场景**：
- Creator自愿提供数据的平台
- 不适用于Discovery/冷启动场景

**与10.1的差异**：
- Phyllo = 认证数据（需Creator配合）
- 10.1 = 公开数据Discovery（无需Creator配合）

### 4. CreatorDB（有MCP）

**基础信息**：
- 数据库：10M+达人
- 有开源MCP Server

**MCP能力**：
- 31个工具
- IG/YT/TikTok搜索分析
- MIT开源

**弱点**：
- 数据库规模小（10M vs Modash 350M）
- 社区项目，非商业级

### 5. Apify Influencer MCP

**定位**：通用爬虫层的达人发现

**定价**：$100/1000 profiles

**特点**：
- Apify官方维护
- 多平台支持
- 评分2.4/5（较低）

**弱点**：
- 通用爬虫，非专业达人数据
- 数据质量不如专业竞品
- 无达人分析能力

## 竞争空白分析

### 已确认的空白

| 空白 | 描述 | 验证来源 |
|------|------|---------|
| **PLG定价** | $0-199/月自助订阅几乎不存在 | 全部竞品需Sales或年付 |
| **专业MCP** | Modash/HypeAuditor级别的MCP为零 | MCP目录搜索 |
| **自助文档** | 多数竞品需Sales才能看文档 | 竞品官网验证 |

### 可能的空白（需验证）

| 可能空白 | 假设 | 验证方式 |
|---------|------|---------|
| 内容DNA/多模态分析 | 现有API只提供静态数据，无内容理解 | 深挖竞品功能 |
| AI Agent原生支持 | 现有API未针对Agent场景优化 | 竞品文档分析 |
| 实时+结构化组合 | 要么静态Discovery，要么实时Raw，无智能组合 | 用户访谈 |

## 关键发现

1. **市场成熟但Enterprise化**：主流竞品都走Enterprise Sales路线
2. **PLG是真空**：$0-199/月定价几乎不存在
3. **MCP是新战场**：专业级达人数据MCP为空白
4. **数据质量>覆盖**：用户更看重数据质量而非数据库大小
5. **自助体验差**：大多数竞品需Sales conversation才能评估

### Modash API详细功能

**Discovery API功能**：

| 端点 | 功能 | Credit消耗 |
|------|------|-----------|
| `/search` | 按关键词/过滤器搜索达人 | 0.01/结果 |
| `/ai-search` | 自然语言搜索（Beta，仅IG 10K+粉丝）| 未公开 |
| `/profile` | 获取达人详细资料 | 1 credit |
| `/audience` | 获取受众人口统计 | 5 credits |
| `/collaborations` | 获取品牌合作历史 | 未公开 |

**Raw API功能**：

| 端点 | 功能 | 请求消耗 |
|------|------|---------|
| `/profile` | 实时获取达人资料 | 1 request |
| `/posts` | 获取最新帖子 | 1 request |
| `/followers` | 获取粉丝列表 | 1 request |
| `/mentions` | 追踪品牌提及 | 1 request |

**技术特点**：
- Postman集合可下载
- Bearer Token认证
- 详细错误码文档
- 30分钟可完成首次调用

**定价详情**：
- Discovery: $16,200/年 = 3,000 credits/月
- Raw: $10,000/年 = 40,000 requests/月
- 超额费用: 未公开（需询问Sales）

**局限**：
- 只覆盖3平台（IG/TikTok/YT）
- AI搜索仅限IG 10K+粉丝
- 无MCP支持
- 必须年度承诺
