### R2定位调整说明

**原定位B"Agent高权重数据源"的问题**：
- 消费者Agent场景：达人内容仅占AI引用**8%**，被Shopping Graph(42%)和品牌官网(44%)碾压
- 聚星品牌认知低：品牌搜索量是Agent信任的最强因素(0.334)，NoxInfluencer在英文世界认知不足
- 假设过于乐观：没有专门"达人数据通道"被主流Agent采用

**新定位C"营销Agent专业数据源"的依据**：
- 营销Agent场景（选达人、外联、效果监控）= 达人数据**刚需**
- AnyTag已发布MCP（2025年7月），partnrup.ai实现全自动化闭环——市场已启动
- 聚星API结构可用(4/5)，改造成本可控
- B2B场景品牌认知要求低于B2C

### 1.3 购买行为

| 维度 | 发现 |
|------|------|
| **价格范围** | API: $10,000-30,000+/年（年付为主）|
| **定价模式** | 年订阅 + Credit/调用次数制 |
| **决策流程** | 需Sales conversation，少有自助购买 |
| **价格敏感度** | 对API质量敏感 > 对价格敏感 |
| **核心关注** | 数据覆盖 > 数据新鲜度 > 文档质量 > 价格 |

**主要竞品定价**：

| 竞品 | 定价 | 模式 | 备注 |
|------|------|------|------|
| Modash Discovery API | $16,200/年起 | 年付 + Credit | 需年度承诺 |
| Modash Raw API | $10,000/年起 | 年付 + Credit | 实时数据 |
| HypeAuditor API | ~$30,000+/年 | 企业定制 | 需Sales conversation |
| Phyllo | 自定义 | 需Creator认证 | 非Discovery场景 |
| Influencers.Club | Credit制 | 无公开定价 | 需Sales |
| CreatorIQ | $28,000+/年 | 年付 | Enterprise only |
| Apify MCP | $100/1000 profiles | 按量付费 | 通用爬虫层 |

**关键发现**：
- 几乎所有API都需要Sales conversation
- 年度承诺是行业标准
- $0-199/月的PLG定价在市场上几乎不存在 → **潜在空白**

### 1.4 现有做法

| 方案 | 优点 | 缺点 | 使用比例 |
|------|------|------|---------|
| **购买第三方API**（Modash/HypeAuditor）| 数据质量高、覆盖广 | 贵、需年度承诺、无自助 | 30% |
| **自建爬虫** | 完全控制、无外部依赖 | 维护成本高、合规风险 | 25% |
| **官方平台API** | 合规、准确 | 覆盖有限、需Creator授权 | 20% |
| **手动收集** | 免费 | 不可扩展 | 15% |
| **Phyllo等认证API** | 数据准确、合规 | 需Creator opt-in | 10% |

**用户不满**：
1. "Modash和HypeAuditor太贵了" — Reddit用户
2. "需要Sales conversation才能看到定价和文档" — 开发者
3. "Credit制定价难以预测，容易超支" — 多个评价
4. "年度承诺对小团队风险太大" — 开发者

**换方案的触发条件**：
1. 找到更便宜的替代品（主要）
2. 需要更好的文档/开发体验
3. 需要MCP/Agent支持（新兴需求）
4. 合规要求变化

| 维度 | 评分 | 现状 | 改造需求 |
|------|:----:|------|---------|

| **假粉检测** | 2/5 | ❌ 无 | **必须开发**（竞品都有）|
| **实时性** | 4/5 | ✅ 可实时更新 | 需增加Webhook |

**聚星独特优势（R2确认）**：
1. **NoxGPT**：自然语言达人搜索能力，可直接暴露为MCP工具
2. **中国出海品牌数据**：40万+中国客户的合作数据，对中国营销Agent高价值
3. **NaverBlog支持**：韩国市场独特覆盖

**聚星关键缺口**：
1. **无假粉检测**：HypeAuditor有53个ML模式，聚星没有——Agent场景必须补上
2. **无MCP支持**：需开发MCP Server
3. **品牌认知低**：品牌搜索量是Agent信任最强因素(0.334)

### 4.4 优势评分

**R1评分：2.5/5.0** → **R2评分：3.0/5.0**（小幅上调）

上调理由：
- NoxGPT是独特能力，可复用于MCP
- 中国出海场景有数据优势
- 定位调整为"营销Agent"后，品牌认知要求降低

仍需解决：
- 必须开发假粉检测
- 必须开发MCP Server
- 需新品牌绕过信任问题
