## 一、场景画像

### 1.1 行业背景

- 2025年全球网红营销市场规模 **$32.55B**，同比增长10%
- **60.2%** 品牌方已使用AI进行达人发现和campaign优化
- AnyMind Group 成为全球首批将 **MCP协议** 接入网红营销平台（AnyTag）的公司
- AMT 获 $3.5M 种子轮，其AI Agent "Lyra" 可在 **5分钟** 完成传统9小时的达人签约流程

### 1.2 典型使用场景

| # | 场景 | 触发条件 | 频率 | 规模 |
|---|------|---------|------|------|
| 1 | **新品/Campaign达人发现** | 新品推广、大促、进入新市场 | 月频1-4次 | 20-300个达人 |
| 2 | **竞品达人策略监控** | 竞品曝光增加、季度分析 | 周频-月频 | 5-50品牌×100+达人 |
| 3 | **单个达人快速尽调** | 达人主动联系、BD推荐 | **高频日常**（最高频场景） | 单个达人 |
| 4 | **达人自身竞品分析** | 制定策略、准备商务提案 | 月频-季频 | 10-50达人 |
| 5 | **投资/战略达人生态洞察** | VC尽调、CMO年度预算 | 季频-年频 | 100-1000+达人 |

#### Agent场景 vs 传统SaaS的本质差异

| 维度 | 传统SaaS | Agent + Skill |
|------|---------|---------------|
| **入口** | 独立平台，需登录切换 | 嵌入对话/工作流，零切换成本 |
| **交互** | GUI点击、筛选器 | 自然语言，支持模糊需求 |
| **上下文** | 无记忆，每次从零开始 | 记住品牌偏好、历史、排除名单 |
| **输出** | 数据表格/仪表盘 | 直接生成洞察、建议、可执行方案 |
| **组合性** | 单一功能 | 可串联：查数据→写Brief→生成邮件→发送外联 |
| **付费模式** | 年费$数千-数万 | 按需调用，低门槛 |

> **核心洞察**：达人数据Skill的最大价值不是"查数据"本身，而是成为**更大工作流的一个环节**。用户不是来"用达人平台"的，而是来"做营销"的，达人数据只是过程中被自动调用的一个能力。**从"达人数据库"变成"营销Agent的眼睛"。**

### 1.4 购买行为

| 维度 | 发现 |
|------|------|
| **主流定价模式** | 混合模式（基础订阅+用量计费），61%的SaaS已采用 |
| **Agent专属定价** | 按执行/对话计费（Salesforce $2/对话）、按结果计费（$20/转化线索）|

| **GPT Store变现** | 尚未成熟，无创作者直接付费系统 |
| **ClawHub变现** | 完全免费开放，无内置变现机制 |
| **MCP变现** | 刚起步，usage-based metering工具正在出现 |

**定价建议**：Freemium + 按调用计费 + 月度封顶，核心定价锚定 **$19-$99/月**

### R2修正（步骤0）

#### 场景修正
- ✅ **确认**：达人数据是嵌入MarTech栈的数据层，不是独立产品 → Skill/API/MCP定位正确
- ✅ **新增场景**：CRM Enrichment（把达人数据灌入品牌现有CRM）、White-label Data（OEM给其他平台）
- ⚠️ **降级**：竞品达人情报（不是API场景）、匹配推荐（子能力非核心场景）
- 🔴 **发现**：已有3-5个初期达人MCP Server（CreatorDB 31工具最完整），但专业级=0

#### 用户画像修正
- 🔴 **重大修正**：PLG目标从"海外品牌营销团队"→ **"为品牌服务的开发者和平台构建者"（B2D）**
  - 品牌营销团队能用API的不到5%
  - 达人数据API的真实买家是Pietra（a16z投）这类平台公司，不是品牌本身
  - MCP用户几乎100%是开发者，品牌方<5%
- ⚠️ **信任障碍**：TikTok被罚€5.3亿先例下，中国公司做海外数据服务获客成本可能是非中国公司的3-5倍

#### 定价修正
- R1: $29-199/月 → **R2: $9-299/月（Credit制）**
- Starter从$29-49降至**$9-19/月**（MCP首个付费案例Ref.tools仅$9/月）
- **Credit-based是最优模式**（纯订阅在MCP场景下Starter tier毛利率可达-206%）
- 竞品API定价：Modash $16,200/yr起、HypeAuditor ~$30K+/yr → **PLG低门槛有巨大空间**

#### MCP生态修正
- R1: 80%标准概率 → **R2: 95%+**（OpenAI+Google全面采纳+捐赠Linux Foundation）
- MCP Server总数5,500+，下载量13个月80x增长
- **专业达人数据MCP Server = 0（空白确认）**
- 但：独立付费MCP Server商业模式仍在验证期

#### R3场景发散
**非传统场景（扩展TAM）**：

**应用层延伸（突破API天花板）**：
| 层级 | ARR天花板 | 倍数 |
|------|----------|------|
| 纯API | $3-10M | 1x |
| 应用层 | $50-100M+ | 10-30x |
| 建议路径 | API验证 → 对话式AI助手(低门槛) → 达人+电商归因(高价值) | |

> **Twilio/Plaid共同规律**：5-10年从API切入后向应用层延伸，TAM扩大10倍+

**客户端反推**：
- 40万+客户最缺"达人营销全流程操作系统"，不是更多数据
- API买家是5-10%客户但代表40-60%收入潜力
- 中国出海品牌对AI工具接受度90%+，但对MCP概念认知极低
- **正确打法：AI嵌入工作流，不让用户学概念**

**MCP生态位**：
- 一个远程MCP Server = 全平台覆盖（Claude+ChatGPT+Gemini+Cursor）
- 官方MCP Registry(2025.9上线) + 17+第三方目录 = 发现渠道
- 广泛采用关键：**数据独特性** > 响应速度 > DX
- ⚠️ 聚星数据（YT/IG/TT）与Modash相同平台+更小规模 → 数据独特性分低

### 1.5 现有做法与空白

| 方式 | 代表 | 现状 |
|------|------|------|
| ① 直接问LLM | ChatGPT/Claude | 🔴 幻觉严重，编造达人名字，数据过时 |

## 二、竞品分析

> 详见：[达人数据Skill_竞品分析.md](达人数据Skill_竞品分析.md)

### 竞品格局一句话

> **"达人数据Skill"是全新品类，零直接竞品。** AnyTag MCP是最接近的（全球唯一），但定位企业级(¥10K+/月)。营销Agent赛道2025年融资$1.5亿+，但无人专做"数据层"。

### 核心竞品定位图

```
                执行深度
                  ↑
    AMT/Agentio   │   Aha Lab
    ($3.5M-$56M)  │   ($5.7M)
    全流程Agent    │   全流程Agent
                  │
    ─────────────────────────── → 数据广度
                  │
    AnyTag MCP    │   HypeAgent
```

### 关键发现
1. **AnyTag已覆盖抖音+小红书**（小红书一级合作伙伴），但数据量仅100万（聚星100x）

4. **Aha Lab vs 聚星 = Agent-first vs Tool-first的代际差异**

## 四、聚星优势

### SWOT矩阵

| | 正面 | 负面 |
|--|------|------|
| **内部** | **S**: 40万出海客户(需求垄断) + 独占交易数据 + 建联支付闭环 | **W**: 数据依赖爬虫(合规风险) + MCP落后6月 + 国际品牌弱 |
| **外部** | **O**: Agent爆发+MCP标准化 + 出海预算增长 + 竞品无执行层 | **T**: 平台封杀爬虫 + AnyTag先发 + 大平台自建匹配 |

### 绝对优势（竞品5年内难追）

| 维度 | 能力 | 为什么不可复制 |
|------|------|--------------|
| **C1 客户基数** | 40万+出海企业 | 需求侧垄断，竞品从零获客中国出海市场成本极高 |
| **C3 交易数据** | 真实报价/效果/合作历史 | 只有撮合平台才有，爬虫**永远拿不到** |
| **C5 执行闭环** | 邀约→合同→支付 | MCP可触发行动，从"信息工具"升级为"执行工具" |

### ⚠️ 聚星2025现实状况

**R1调研修正（Owner反馈）**：
- 聚星在中国出海圈是**龙头地位且保持增长**，只是在海外不太宣传
- 英文评价平台（G2/Capterra）数据少不代表业务差——核心客群是中国出海品牌
- 数据库规模多源不一致，待内部确认

| 维度 | 公开可查 | Owner反馈 |
|------|---------|----------|
| AI/Agent | 无公开动作 | — |
| MCP | 无 | — |
| API | ✅ 有（只读数据查询）| — |
| 业务状况 | 英文渠道信息少 | **中国出海圈龙头，持续增长** |

> 聚星的AI/MCP空白是事实。但业务基本面稳健——做达人数据Skill既紧迫也有基础。

### 交易数据优势修正（Owner反馈）

- 交易数据（报价/效果/合作记录）**不算独有** — 所有一站式平台都有，只是多以邮件/沟通记录形式存在，未结构化挖掘
- 闭环平台（建联+支付）也有一些，不算聚星独占
- 聚星交易数据的**量级和质量**待内部确认

> 修正后的优势判断：交易数据是**可挖掘的差异化方向**（如果结构化+AI增值），但不是"只有聚星有"的绝对壁垒。

### 推荐定位

> **❌** "通用达人数据API"（和Modash/Phyllo正面竞争）
> **✅** "达人数据Skill" — 面向**海外品牌**（新客群），PLG自助获客，不依赖BD
> ℹ️ "TikTok电商智能数据层"（Action MCP）作为A1子方向单独调研

| 层级 | 价格 | 能力 |
|------|------|------|
| Explorer | 免费/$29/月 | 搜索+画像（抢HypeAgent市场）|
| Pro | $199/月 | + 交易智能 + 匹配推荐 |
| Enterprise | ¥5K-10K/月 | + 执行闭环 + 批量建联 + 支付 |

## 五、PLG可行性

### 总评分：2.5/5 — 纯PLG不可行，需混合模式

| 维度 | 评分 | 关键发现 |
|------|------|---------|
| 自然病毒传播 | ⭐⭐ | MCP是"后端组件"，无Slack式viral loop |
| **Time-to-Value** | **⭐⭐⭐⭐** | **最大优势** — 自然语言查达人秒级返回，vs传统SaaS 20分钟onboarding |
| Free→Paid转化 | ⭐⭐⭐ | "搜到完美达人但看不到联系方式"是强转化触发器 |
| 增长飞轮 | ⭐⭐⭐ | 数据网络效应存在但冷启动慢 |
| **PLG障碍** | **⭐⭐** | **最大风险** — MCP营销用户全球<5000人，60-70%目标用户无法自助用MCP |

### 推荐模式：PLG底座 + Sales上层

```
PLG自助（Free/$29/$199）→ 30%收入、70%客户数（获客漏斗）
Sales-Assisted（Enterprise）→ 70%收入、30%客户数（核心营收）
```

> 用免费层+Agent平台筑低成本获客漏斗，用Sales转化Enterprise贡献核心收入。**不要期待PLG自己跑通。**

## 七、JTBD分析

### 核心Job

| 层面 | Job |
|------|-----|
| **Functional** | 在AI工作流中即时获取准确的实时达人数据，不离开Agent环境 |
| **Emotional** | 消除"花钱打水漂"的不确定性焦虑，获得确定感和掌控感 |
| **Social** | 成为"数据驱动的AI时代营销领袖" |

### 功能优先级

| 优先级 | 阶段 | 功能 |
|--------|------|------|
| **P0** | 搜索+评估 | 自然语言达人搜索 + 结构化档案 + 按量付费 |
| **P1** | 决策+执行 | 跨平台查询、假粉检测、对比报告 |
| **P2** | 监控+复盘 | 竞品监控、ROI预测、效果追踪 |

### Switch分析

促进力(Push+Pull)=10 > 阻碍力(Anxiety+Habit)=7 → **切换可能但需助推**

- **最强触发**：现有SaaS涨价/续费到期、大促前赶时间、被达人骗过
- **最大阻碍**：数据准确性焦虑 → **化解信任是打开切换闸门的钥匙**

### Top 3 Underserved Outcomes

1. **Agent内即时获取数据**（19/20）— 目前0个成熟方案
2. **10倍价格降低**（18/20）— $0.05/次 vs $300+/月
