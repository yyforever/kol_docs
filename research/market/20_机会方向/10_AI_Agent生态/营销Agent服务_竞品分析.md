## 五、核心竞品深挖（步骤2）

### 5.1 Stormy AI — YC S24 PLG标杆

| 维度 | 详情 |
|------|------|
| **创始人** | Robert Lukoszko（波兰裔，连续创业者，前Fixkey AI被收购）|
| **YC** | **S24**（非W25），Pre-Seed $500K |
| **ARR** | ~$550K（2025.9，5人团队）|
| **定价** | Basic $100 / Starter $140 / Agent $350（Credit制）|
| **流程** | 自然语言搜索→一键获取联系方式→AI外联+自动谈判→成交追踪 |
| **杀手功能** | 预算护栏内AI自动砍价（案例：$3500→$2000，4封邮件）|
| **覆盖** | YouTube, TikTok, Twitch, Substack, Podcasts (1.6B+ IG档案) |
| **验证** | Product Hunt #1（465 upvotes），Futurepedia 4.6/5 |
| **技术** | 推测GPT-4/Azure OpenAI，Gmail+TikTok集成 |

**Stormy vs AMT**：Stormy = 品牌侧自助工具（PLG，已有收入）；AMT = 双边平台基础设施（融资更多，叙事更大）

**与聚星的关系**：Stormy是最直接的PLG竞品参照。$100起步+自然语言+自动外联 = 聚星如果做B方向的对标产品。

### 5.2 Agentio — 程序化达人广告交易所（不同品类）

| 维度 | 详情 |
|------|------|
| **定位** | "达人广告的The Trade Desk" — 程序化购买达人内容 |
| **融资** | $56M累计，$3.4亿估值（Forerunner+Benchmark+Craft）|
| **增长** | 5x+ YoY，100+企业客户（Uber/DoorDash/CashApp），数千万$GMV |
| **团队** | 35人→2026扩至100+ |
| **模式** | 品牌输入预算+目标→AI自动匹配+出价+签约+审核+追踪 |
| **抽佣** | ~20% take rate（仅向品牌收，达人零费用）|
| **定价** | 动态定价（基于30天交付量），90天归因窗口 |
| **AI** | LLM多模态内容分析，"非显而易见匹配"（健康品牌→户外工作者达人）|

**与B方向的关系**：**不同品类**。Agentio是垂直交易平台（双边marketplace），不是通用Agent工具。但验证了两点：
1. AI程序化匹配达人是可行的 → 技术路线成立
2. 20% take rate在市场可接受 → 商业模式有参考价值


### 5.3 Scrumball — 中国出海SaaS，$39/月极低价切入

| 维度 | 详情 |
|------|------|
| **创始人** | 朱正扬(Alex Chu)，深圳/Mountain View双城，前Nanigans GM |
| **公司** | 北京铨能科技，40-50人，深圳/北京/上海/杭州/纽约/台北 |
| **融资** | 未公开 |
| **定价** | Free → $39/月（行业最低入门价）|
| **AI Agent** | 3个人格化Agent(Lillian品牌分析/Hunter红人发现/Charlie外联) + 2个功能模块 |
| **数据库** | 120M+（YouTube/TikTok/IG），数据增长快但一致性存疑 |
| **客户** | UR、Banggood(棒谷)、Meitu(美图)、Dreame(追觅)、MomMed等出海品牌 |
| **验证** | PH 604票/121评论，但Shopify 0评价，无第三方独立评价 |

**⚠️ 与聚星高度重叠**：
- 核心客群相同（中国品牌出海）
- 但模式不同：Scrumball = 自助SaaS工具，聚星 = 全链路服务
- $39/月可能抢走中小客户（"先试试自己做"）

**对B方向的启示**：
- 验证了中国出海品牌对AI Agent工具的需求
- 但也说明**直接面向中国出海品牌的赛道已有玩家**
- 如果聚星做B方向，面向**海外原生品牌（PLG）**可能是更好的差异化

### 5.4 GRIN Gia — 传统龙头的教科书级Agent化转型 ⭐

| 维度 | 详情 |
|------|------|
| **公司** | GRIN Technologies，2014年创立，7轮共$145M融资 |
| **领导层** | 2025.1 Ryan Debenham接任CEO（原联合创始人Brandon Brown已离开另创Search Party）|
| **Gia定位** | 从零构建的"purely agentic-powered operating system" — **不是在旧GRIN上加AI** |
| **客户** | SKIMS, Rhode, GoPro, Warby Parker, Allbirds |

**Phase路线图**：
| Phase | 状态 | 能力 |
|-------|------|------|
| Phase 1 | ✅ 已上线 | 自然语言搜索+AI视觉分析（能看图片/视频识别达人特征）|
| Phase 2 | ✅ 已上线 | 全流程沟通自动化（399封跟进邮件一键生成）|
| Phase 3 | 🔄 开发中 | **目标驱动自主执行**："月GMV $5K"→Gia自主完成全部 |

**2026.1定价革命**：
| 旧模式 | 新模式（2026.1.27起）|
|--------|---------------------|
| $999-2,500/月，年付锁定，无试用 | **月付月结+30天免费+信用卡即开** |
| 需Sales-led Demo | **即时自助注册** |

**用户评价**：G2 4.5-4.6/5(481条)，Capterra 4.7/5(147条)，但Gia专门评价尚少

> **CEO原话**："We are not layering this on top of GRIN. We started over, built Gia from the ground up as a purely agentic-powered operating system."

**⚠️ 对聚星的关键警示**：
1. GRIN是**行业龙头做Agent化转型**的标杆——有$145M融资+10年数据壁垒+G2 4.5/5的品牌
2. 正在**从Enterprise走向PLG**（月付+免费试用）→ 直接进入聚星想做的价位区间
3. Phase 3如果实现（按效果收费），商业模式会根本改变
4. **聚星做B方向 = 和GRIN正面竞争**

## 六、传统SaaS AI化进展

### 三种分化路径

| 路径 | 代表 | Agent深度 |
|------|------|----------|
| 🟢 **Agentic重构** | GRIN Gia | Agent替代80%人工，Phase 3目标：设目标→AI自主执行 |
| 🟡 **AI功能嵌入** | Upfluence Jace / SARAL SIA / Aspire / Modash | 搜索+邮件AI辅助 |
| 🔵 **AI基础设施** | CreatorIQ SafeIQ | 多模态品牌安全AI |

### 关键发现

1. **只有GRIN在做真正的Agent** — CEO拒绝"AI hype"，从零重构Agentic系统
2. **没有人因AI涨价** — 所有平台都将AI视为"应有之义"
3. **GRIN反而降低门槛**：从年付锁定→月付+免费试用（最聪明的定价变化）
4. **传统SaaS的真正威胁**是partnrUP这类Day 1按Agent范式设计、$0起步的AI-native玩家
