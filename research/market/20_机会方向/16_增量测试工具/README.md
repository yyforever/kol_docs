# 16号机会：增量测试工具（Incrementality Testing）

> 调研时间：2026-02-05
> 状态：✅ 调研完成 | 评分：**3.13分** | 结论：❌ 不推荐

---

## 方向概述

**一句话描述**：帮助品牌衡量网红营销真实增量效果的测试工具，解决Cookie淘汰后的归因难题。

**核心问题**：网红营销到底带来了多少**真实的增量销售**？（而非自然转化）

**市场信号**：
- Cookie淘汰后增量测试成为"金标准"
- Measured获$21M融资
- 73%营销领导者认为增量测试必不可少（2025年）
- 52%品牌/代理商已在使用增量测试

**契合度**：🔴 极高 → ❌ 实际契合度有限

## 核心结论

### 不推荐原因

1. **聚星资源契合度有限**（2.5分）
   - 增量测试需要Geo实验能力
   - 网红营销天然无法做地理隔离（竞品也承认这是盲区）
   - 需要全新的数据科学团队

2. **开发难度极高**（2.0分）
   - 需要因果推断、合成控制法专家
   - 6-18个月开发周期
   - $500K-1M+团队成本

3. **网红场景方法论不成熟**
   - 内容覆盖全网，无法Geo隔离
   - 传统增量测试方法不适用

### 建议方向

**整合到02A CRM作为增值功能**：
- 优惠码防泄漏（已从19号迁移）
- 独占链接追踪
- 简化归因报告
- ROI估算仪表盘

## 关键发现

### 问题验证结果

| 问题 | 答案 |
|------|------|
| 增量测试技术门槛有多高？ | **极高**：需数据科学团队、因果推断专家 |
| 现有竞品的定价区间？ | $2,000-50,000+/月（SMB: $2K起） |
| SMB客户是否有需求？ | 有需求但Stella已在$2K/月覆盖 |
| 可以做轻量版吗？ | ✅ 可以做"轻量归因"而非严格增量测试 |
| 聚星数据能否支撑？ | 数据有价值但缺乏Geo实验能力 |

### 竞品格局

```
企业级（$10K+/月）：Measured、Lifesight、Recast
         ↓
成长型（$2K-10K/月）：Haus、Stella、INCRMNTAL ← 已有竞品
         ↓
入门级（免费）：平台原生工具、开源方案
         ↓
SMB市场（$99-499/月）：❓ 空白但需求存疑 ← 不建议进入
```

### 替代方案

**"轻量归因"比"严格增量测试"更适合网红场景**：
1. 优惠码防泄漏 → 已整合到02A CRM
2. 独占链接追踪 → UTM + 短链
3. 第一方数据匹配 → 曝光vs转化交叉
4. 增量估算模型 → 历史数据建模
