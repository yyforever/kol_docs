## 核心竞品详细分析

### 1. Euka AI (TikTok Shop专用)

| 维度 | 详情 |
|------|------|
| **定价** | $159/月(Starter) → $479/月(Growth) |
| **官方合作** | TikTok Shop App Store上架 |
| **外联能力** | 每天1000 DM + 2500邀请 |
| **AI匹配** | ⚠️ 主要靠数据驱动（GMV/粉丝等），多模态AI尚未上线 |
| **用户反馈** | "Blazi beats them on affiliate discovery" |
| **Trustpilot** | 3.2/5（样本少）|
| **融资** | 未融资 |

**启示**：发送量大≠效果好，匹配精准度是核心痛点

## 关键洞察

### 1. 市场空白

| 空白 | 说明 |
|------|------|
| **$100-200/月高性价比工具** | Modash太贵，Aha质量不稳定 |
| **AI匹配精准度** | 所有工具都被吐槽"推荐不精准" |
| **内置外联+匹配一体化** | Modash无外联，LumiTok无深度分析 |
| **垂直细分领域** | 通用工具在crypto/tech等垂直领域效果差 |

### 2. 用户核心痛点（从竞品评价提炼）

| 痛点 | 证据 |
|------|------|
| **匹配不精准** | "Blazi beats them on affiliate discovery" |
| **消息触达率低** | "Most messages sent via tools are never seen" |
| **价格贵** | "Modash and HypeAuditor are too expensive" |
| **数据质量差** | "Disappointed with data quality, especially for EU" |

### 3. 竞争格局结论

- **Blazi** = 搜索发现最强
- **Cruva** = 多平台扩展野心
- **Euka** = 创作者市场差异化
- **Aha** = AI Agent概念领先但执行不成熟
- **CeMeta** = 中国本土小红书场景最强
