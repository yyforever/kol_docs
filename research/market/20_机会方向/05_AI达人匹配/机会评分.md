## 二、三轮调研核心发现

### 第1轮：基础调研
- ✅ 痛点真实强烈（"discovery is the most painful"）
- ✅ $100-300价位确实空白
- ⚠️ "AI匹配可能是伪命题" → 第2轮推翻

### 第2轮：深度调研
- 🔥 **Blazi为什么最强**：不是数据库大，而是**AI分析视频内容提取非结构化特征**
  - 肤色、内容风格、语言、出镜方式等
  - 传统工具只能筛选GMV/粉丝数
- ✅ $149理想用户：月GMV $30K-$100K跨境卖家
- ✅ 存量客户10-15%转化率（4-6万客户可期）

### 第3轮：技术可行性
- 💰 **复制Blazi完整能力**：$150-300万 + 14-19人月
- ⚠️ 核心问题："技术可行，但值不值得？"
- ✅ 更务实路径存在：LLM增强元数据 → 字幕分析 → 视频分析（分阶段）

## 三、各维度详细评分

### 1. 痛点强度：4.5/5 ✅ 维持

| 痛点 | 严重度 | 验证来源 |
|------|:------:|---------|
| 发现过程极度耗时 | ⭐5 | Reddit、G2、中文社区 |
| AI推荐不精准 | ⭐5 | 所有竞品都被吐槽 |
| 回复率<15% | ⭐5 | 中国跨境卖家调研 |
| 工具太贵$300+/月 | ⭐5 | 用户原话"stupidly expensive" |
| 功能割裂 | ⭐4 | "需要4-5个订阅" |

**结论**：痛点真实且强烈，付费意愿明确。

---

### 2. 竞争空白：3.0/5 ⬇️ 下调0.5

**下调理由**：

| 发现 | 影响 |
|------|------|
| Blazi已实现AI视频内容分析 | 空白比预期小 |
| Blazi定价$199起，覆盖目标价位 | 价格空白被部分填补 |
| 技术壁垒比预期高 | 不是简单的"做就行" |

**空白仍存在的地方**：
- Blazi只做TikTok Shop（聚星全平台）
- Blazi库1M vs 聚星75M
- 中国跨境卖家服务（中文+支付宝）

---

### 3. 聚星优势：3.0/5 ⬇️ 下调0.5

**下调理由**：

| 资源 | 第1轮评估 | 第3轮修正 |
|------|----------|----------|
| 1亿网红库 | 核心优势 | ⚠️ 只有元数据，**没有视频内容分析** |
| 推荐算法 | 可复用 | ⚠️ 不支持非结构化特征搜索 |
| NoxGPT | 可用于搜索 | ❌ 是游戏助手，**不是达人搜索AI** |
| 40万客户 | 分发优势 | ✅ 维持，但需要新产品才能转化 |

**要达到Blazi水平需要**：
- 💰 $150-300万（一次性）
- 👥 14-19人月
- ⏰ 6-12个月

**聚星真正的优势**：
1. 75M全平台数据（Blazi只有TikTok 1M）
2. 40万中国出海客户
3. 已有邀约功能（可形成闭环）

---

| PLG因素 | 评估 |
|---------|------|

### 5. 开发难度：3.0/5 ⬇️ 下调1.0

**下调理由**：

| 场景 | 原评估 | 修正评估 |
|------|--------|----------|
| 简单版（LLM增强搜索）| 6人月 | ✅ 可行 |
| Blazi水平（视频AI分析）| — | 14-19人月 + $150-300万 |

**分阶段开发路径**：

| 阶段 | 内容 | 投入 | 效果 |
|------|------|------|------|
| Phase 1 | LLM增强元数据搜索 | 2-3周 | 30%提升 |
| Phase 2 | 字幕/文案分析 | 4-6周 | 70%效果 |
| Phase 3 | 视频内容AI分析 | 3-6月 | Blazi水平 |

## 四、策略建议

### 推荐路径：分阶段验证

```
┌─────────────────────────────────────────────────────────┐
│  Phase 1: LLM增强搜索 (2-3周, ~¥20万)                    │
│  ├─ 自然语言查询 → 结构化筛选                            │
│  ├─ 语义扩展（"美妆" → 护肤、彩妆、美容仪...）            │
│  └─ 验证PMF：用户是否愿意为"更聪明的搜索"付费             │
├─────────────────────────────────────────────────────────┤
│  Phase 2: 字幕/文案分析 (4-6周, ~¥50万)                  │
│  ├─ Whisper提取视频语音 → 文本分析                       │
│  ├─ 识别语言、内容主题、说话风格                         │
│  └─ 70%的Blazi效果，10%的成本                           │
├─────────────────────────────────────────────────────────┤
│  Phase 3: 视频内容AI分析 (3-6月, ¥150-300万)            │
│  ├─ 关键帧提取 → VLM分析                                │
│  ├─ 肤色、场景、穿着、出镜方式                          │
│  └─ 只在Phase 2验证成功后投入                           │
└─────────────────────────────────────────────────────────┘
```

### 备选策略：收购/合作Blazi

| 考量 | 分析 |
|------|------|
| Blazi规模 | 小（TikTok Shop垂直），可能是收购目标 |
| 技术互补 | Blazi有视频AI，聚星有数据规模 |
| 风险 | 整合难度、团队稳定性 |
