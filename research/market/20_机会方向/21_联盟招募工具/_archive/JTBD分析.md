## 一、核心Job定义

### 1.1 主Job（Core Job）

**When** 我的品牌/店铺需要通过达人带货扩大销售规模，但自身流量能力有限时，
**I want to** 高效招募并管理一批能持续产出销售的联盟达人，
**So I can** 以可预测的成本获得可衡量的销售增长。

### 1.2 触发场景（Trigger Situations）

| 触发场景 | 用户类型 | 紧迫程度 |
|---------|---------|---------|
| **新店冷启动** | TikTok Shop新卖家 | 高 - 没有达人=没有流量 |
| **增长瓶颈** | 成熟卖家/品牌 | 高 - 现有渠道增长停滞 |
| **大促备战** | 所有卖家 | 高 - 时间窗口压力 |
| **竞品抢人** | 所有卖家 | 中 - 发现达人被竞品抢走 |
| **团队效率低** | 中大型卖家 | 中 - 人工成本攀升 |
| **ROI下降** | 成熟运营 | 中 - 需要更精准的达人匹配 |

### 1.3 Job层级分解

```
主Job：以可预测成本获得可衡量的销售增长
│
├─ 子Job 1：找到合适的达人（Discovery）
│   ├─ 1.1 发现潜在达人
│   ├─ 1.2 评估达人质量
│   └─ 1.3 识别品牌匹配度
│
├─ 子Job 2：成功建立合作（Recruitment）
│   ├─ 2.1 高效触达达人
│   ├─ 2.2 提升回复率
│   └─ 2.3 快速完成签约
│
├─ 子Job 3：追踪转化效果（Tracking）
│   ├─ 3.1 准确归因销售
│   ├─ 3.2 防止数据丢失
│   └─ 3.3 识别欺诈行为
│
├─ 子Job 4：管理达人关系（Management）
│   ├─ 4.1 组织达人信息
│   ├─ 4.2 维护沟通记录
│   └─ 4.3 达人分层运营
│
├─ 子Job 5：准确支付佣金（Payment）
│   ├─ 5.1 自动计算佣金
│   ├─ 5.2 批量发放支付
│   └─ 5.3 处理争议退款
│
└─ 子Job 6：优化投资回报（Optimization）
    ├─ 6.1 分析达人表现
    ├─ 6.2 预测合作效果
    └─ 6.3 持续迭代策略
```

### 2.2 功能性Job详解与量化目标

| Job | 用户期望 | 量化目标 | 当前痛点 |
|-----|---------|---------|---------|
| **发现达人** | 快速找到符合要求的达人 | 10分钟内筛选出50个潜在达人 | 逐个浏览，每个达人需2-3分钟 |
| **评估质量** | 了解达人真实带货能力 | 看到GMV、退货率、复购率 | 只能看到粉丝数和互动率 |
| **批量触达** | 高效联系大量达人 | 日触达500-3000人 | 手动每天50-100人 |
| **提高回复率** | 获得更多达人响应 | 回复率>15% | 当前<5%回复率 |
| **准确追踪** | 每一单都能正确归因 | 追踪准确率>95% | Cookie失效导致20-30%丢失 |
| **防止欺诈** | 识别刷单和假流量 | 欺诈检测率>90% | 55%存在假互动问题 |
| **自动支付** | 无需手动处理佣金 | 一键完成批量支付 | 手动计算、逐个转账 |
| **ROI分析** | 清晰看到投入产出比 | 实时ROAS仪表盘 | Excel手动统计 |

## 三、情感性需求（Emotional Jobs）

### 3.1 用户想要的感受

| 情感需求 | 描述 | 触发场景 |
|---------|------|---------|
| **掌控感** | 我清楚地知道达人招募进展和效果 | 每天看仪表盘，清晰了解漏斗各阶段 |
| **信心** | 我确信找到了合适的达人 | 数据证明达人与品牌匹配、带货能力真实 |
| **效率感** | 我的时间花在刀刃上，不做重复劳动 | 自动化处理繁琐任务，聚焦决策 |
| **安全感** | 我不会因为欺诈或追踪失败损失钱 | 追踪准确、欺诈预警及时 |
| **成就感** | 看到招募的达人持续产出业绩 | 达人带来稳定GMV增长 |

### 3.2 用户想要避免的负面情绪

| 负面情绪 | 描述 | 触发场景 | 影响 |
|---------|------|---------|------|
| **焦虑** | 不知道达人找得对不对 | 投入时间联系后达人不出单 | 质疑整个策略 |
| **挫败感** | 大量触达但几乎无人回复 | <5%回复率，被达人"无视" | 丧失信心 |
| **失控感** | 不知道钱花在哪了 | 追踪不准确，ROI无法计算 | 不敢继续投入 |
| **被欺骗感** | 发现达人刷单或数据造假 | 支付佣金后发现退货率极高 | 信任危机 |
| **疲惫感** | 重复劳动占据大量时间 | 手动复制粘贴、逐个发消息 | 效率低下，想放弃 |
| **尴尬** | 同一个达人被重复联系 | 缺乏系统记录，重复触达 | 品牌形象受损 |

### 5.2 各竞品详细分析

#### Impact.com

**定位**：企业级全渠道联盟管理平台

| 维度 | 表现 | 评价 |
|------|------|------|
| **Functional Jobs** | 功能全面，追踪和支付能力强 | 覆盖全流程 |
| **Emotional Jobs** | 复杂界面带来焦虑，学习曲线陡峭 | 安全感高但掌控感差 |
| **Social Jobs** | 企业级定位，用它显得专业 | 但中小企业用不起 |

**优势**：追踪准确、支付自动化、企业级安全
**劣势**：价格高（$500+/月）、设置复杂、不适合TikTok Shop场景

**用户评价**：
> "界面对初学者来说令人不知所措，功能、选项和设置太多"

---

#### Social Snowball

**定位**：Shopify电商联盟自动化工具

| 维度 | 表现 | 评价 |
|------|------|------|
| **Functional Jobs** | Shopify集成强，自动化程度高 | 客户自动转联盟做得好 |
| **Emotional Jobs** | 简单易用，效率感强 | 但仅限Shopify |
| **Social Jobs** | DTC品牌圈子认可度高 | TikTok Shop卖家不太适用 |

**优势**：易用性好、客户转联盟自动化、Shopify深度集成
**劣势**：仅支持Shopify、批量触达能力弱、不支持TikTok Shop

**用户评价**：
> "把付费客户自动转化为affiliate的体验很棒"

---

#### 达秘

**定位**：TikTok Shop批量建联工具

| 维度 | 表现 | 评价 |
|------|------|------|
| **Functional Jobs** | 批量触达能力极强（日触达3000+） | 但缺乏追踪和支付 |
| **Emotional Jobs** | 效率感爆棚，但安全感差（合规风险） | 用户既爱又怕 |
| **Social Jobs** | 卖家圈子广泛使用，"同行都在用" | 但有灰色地带担忧 |

**优势**：批量效率高、中文界面、价格亲民（2999元/年）
**劣势**：无CRM、无追踪、无支付、合规风险高

**用户评价**：
> "效率确实高，但总担心账号被封"

---

#### TikTok Shop官方

**定位**：平台原生联盟功能

| 维度 | 表现 | 评价 |
|------|------|------|
| **Functional Jobs** | 基础功能具备，追踪可靠 | 但批量能力受限、CRM缺失 |
| **Emotional Jobs** | 安全感最高（官方工具） | 但效率感差、功能单一 |
| **Social Jobs** | 新手入门必用 | 但难以差异化 |

**优势**：官方数据准确、免费、追踪可靠
**劣势**：批量触达受GMV配额限制、无CRM、无自动化

**用户评价**：
> "数据准确，但每周只能联系这么多人，效率太低"

## 八、总结

### 8.1 核心洞察

1. **核心Job清晰**：用户雇佣联盟招募工具的根本目的是"以可预测成本获得可衡量的销售增长"

2. **最大机会空白**：TikTok Shop场景下，"批量触达效率"和"追踪归因准确"两个关键Job无法被同一工具满足

3. **情感需求被忽视**：现有工具过于关注功能，忽视了用户对"掌控感"和"安全感"的情感需求

4. **中国卖家机会**：跨境场景的中文界面、本土化支持存在明显市场空白

5. **竞品各有短板**：
   - Impact.com：功能强但复杂贵
   - Social Snowball：好用但不支持TikTok Shop
   - 达秘：效率高但无追踪无安全保障
   - TikTok官方：安全但效率低
