## 一、公司概况

### 基本信息

| 项目 | 详情 |
|------|------|
| 成立时间 | 2023年初 |
| 总部 | 美国 |
| 融资情况 | $20M Series A（2024年11月，Volition Capital领投） |
| 预计ARR | $3M+（12个月内达成） |
| 平台GMV | $2.86亿/年（基于30天数据推算） |
| 创作者规模 | 7,000-50,000+（不同统计口径） |
| 覆盖市场 | 美国、英国、加拿大、法国、德国（西班牙/意大利即将上线） |

### 核心定位

Levanta是Amazon专用联盟营销平台，定位于连接Amazon卖家与创作者的桥梁：
- **核心价值**：帮助卖家建立自有联盟计划，招募创作者驱动外部流量
- **技术优势**：与Amazon Attribution API深度集成
- **差异化**：14天归因窗口（vs Amazon Associates的24小时）

## 四、API功能与集成能力

### API端点

| 端点 | 功能 |
|------|------|
| `/partners` | 发现活跃品牌和ID |
| `/products` | 产品目录（支持ASIN、品牌ID、佣金、库存筛选） |
| `/reports` | 性能报告（点击、转化、预估销售、预估佣金） |
| `/links` | 程序化创建和管理追踪链接 |

### 核心能力

- **流线化集成**：连接现有技术栈，解锁数十万促销机会
- **定制报告**：点击、转化、预估销售、预估佣金的定制视图
- **简化发现**：丰富的筛选选项加速联盟机会发现
- **程序化链接**：按ASIN/来源批量创建追踪链接

### 申请方式

- 需联系Levanta申请API访问权限
- 目前主要面向Publishers/创作者开放

## 六、差异化竞争优势

### vs Amazon Associates

| 维度 | Levanta | Amazon Associates |
|------|---------|-------------------|
| 归因窗口 | 14天 | 24小时 |
| 平均佣金 | 10-20% | 3-5% |
| 归因透明度 | 完全可见 | 黑盒 |
| EPC（每点击收益） | 2-4倍 | 基准 |
| 品牌控制 | 完全控制 | 无 |

### vs Archer Affiliates

| 维度 | Levanta | Archer |
|------|---------|--------|
| 创作者池 | 50,000+ | 2,500+ |
| 国际市场 | 5个市场 | 仅美国 |
| 定价模式 | 固定月费+% | 按销售定价 |
| 适合 | 规模化运营 | 新手/小预算 |

### 独特能力

1. **CPC模式**（2024年8月推出）
   - 行业首创Cost-Per-Click campaign
   - 适用于新品推广和复活老品
   - 降低创作者风险，更易达成合作

2. **Walmart扩展**（2025年6月）
   - 早期访问计划
   - 成为唯一同时支持Amazon + Walmart的平台

3. **Brand Referral Bonus集成**
   - 平均10%返点
   - 与Levanta归因无缝对接
   - 返点计入2个月后的推荐费用

## 七、中国卖家服务评估

### 现状分析

**未发现明确证据显示Levanta专门服务中国卖家**

| 维度 | 现状 |
|------|------|
| 语言支持 | 仅英文界面 |
| 市场覆盖 | 美/英/加/法/德（非中国） |
| 官方沟通 | 无中文内容 |
| 案例展示 | 全部为美国本土品牌 |

### 潜在障碍

1. **语言障碍**：无中文界面和客服
2. **支付挑战**：美元结算，中国卖家可能面临跨境收款问题
3. **文化差异**：创作者资源主要为欧美创作者
4. **合规要求**：1099税务申报针对美国主体

### 可能的变化

- Walmart扩展后可能触达更多跨境卖家
- 中国大卖家可设立美国公司使用
- 未来可能推出多语言支持

## 九、对聚星的威胁评估

### 威胁等级：中等偏高

### 威胁维度分析

| 维度 | 威胁程度 | 说明 |
|------|---------|------|
| **平台专精** | 高 | Amazon/Walmart双平台领先 |
| **资本实力** | 高 | $20M融资支持快速扩张 |
| **产品成熟度** | 高 | Product Seeding、API、CPC等功能完善 |
| **大客户背书** | 高 | HexClad、MANSCAPED等知名品牌 |
| **中国市场渗透** | 低 | 无中文支持，无明确中国策略 |
| **TikTok能力** | 低 | 专注Amazon/Walmart，不涉及TikTok |

### 直接竞争场景

| 场景 | 竞争强度 | 聚星应对 |
|------|---------|---------|
| Amazon大卖家联盟管理 | **高** | 需差异化定位 |
| 跨平台联盟整合 | 中 | 强化TikTok优势 |
| 中国跨境卖家 | 低 | 中文+本地化服务 |
| TikTok Shop联盟 | **无** | 核心优势领域 |

### 防守策略建议

1. **强化TikTok专精**
   - Levanta完全不涉及TikTok Shop
   - 这是聚星的核心护城河

2. **中国卖家本地化**
   - 中文界面和客服
   - 人民币结算
   - 本地化合规支持

3. **跨平台差异化**
   - TikTok + Amazon + 独立站整合
   - 单一平台工具无法提供

4. **价格竞争力**
   - Levanta Gold $750/月 + 3.5%
   - 存在价格下探空间

### 关注重点

| 监控项 | 频率 | 触发信号 |
|--------|------|---------|
| Levanta中文化 | 季度 | 官网新增中文 |
| TikTok整合 | 月度 | 产品公告 |
| 中国客户案例 | 季度 | 案例发布 |
| 定价调整 | 季度 | 价格变化 |

## 十、关键洞察总结

### Levanta核心优势

1. **Amazon Attribution API深度集成** - 14天归因窗口
2. **成熟的Product Seeding** - 简化产品寄样流程
3. **CPC + CPS双模式** - 灵活的合作模式
4. **大客户背书** - HexClad、MANSCAPED等验证
5. **$20M资本** - 支持快速产品迭代

### Levanta核心劣势

1. **仅限Amazon/Walmart** - 不支持TikTok Shop
2. **无中文支持** - 中国卖家使用门槛高
3. **价格较高** - $750/月起步
4. **创作者质量参差** - 部分用户反馈负面

### 聚星机会点

| 机会 | 可行性 | 优先级 |
|------|--------|--------|
| TikTok Shop专精深耕 | 高 | P0 |
| 中国卖家本地化服务 | 高 | P0 |
| 多平台整合（TikTok+Amazon） | 中 | P1 |
| 产品寄样功能借鉴 | 高 | P1 |
| CPC模式探索 | 中 | P2 |
