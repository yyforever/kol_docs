## 背景

基于27个方向的调研（25个已评分），结合对Agent时代趋势的判断，确定以下三条战略路线。三条路线**共享一套底层能力**，以不同产品包装面向不同用户群体。

---

## 三条战略路线

### 路线1：网红营销API（MCP/Skill）⭐ 最优先

| 维度 | 内容 |
|------|------|
| **定位** | 对外开放网红营销能力API，类似Brave Search API |
| **目标客户** | 自建超级Agent的公司/开发者（调用网红营销能力而非自建） |
| **产品形态** | MCP Server / Agent Skill / REST API |
| **能力范围** | 基础数据（达人搜索/画像）+ 增值服务（策略推荐/洞察/匹配） |
| **商业模式** | 限量API调用，按量计费或订阅 |
| **防竞品策略** | 严格限量，防止竞品通过API获取数据 |
| **优先级** | **最高**（最简单，最快落地） |

**对应调研**：
- 10.1 达人数据智能层（3.625）— 核心对应
- 10.6 A2A撮合基础设施（3.55）— 远期延伸
- [MCP/Skill空白分析](../20_机会方向/10_AI_Agent生态/达人营销MCP_Agent_Skills清单与空白分析.md)

**关键发现（来自调研）**：
- 专业达人数据MCP全球=0，窗口6-12个月（10.1调研）
- AnyTag 2025.7首发MCP，老玩家未跟进
- Phyllo证明B2D API大概率$10-30M/yr封顶，必须向应用层延伸
- 中国平台（小红书/抖音/B站）达人数据API完全空白

**⚠️ 评分偏低（10.1=3.625）的补充考量**：
- 聚星当前客户100%国内（品牌方），API/MCP模式可**首次切入海外客户**
- 以前无法切海外因为模式重销售，API纯线上小额交易绕过这个瓶颈
- 相对海外竞品（Phyllo等）具备**成本优势**（中国团队+中国数据源）
- 单看TAM可能不大，但作为**海外客户入口**的战略价值超过评分体现

### 路线2：网红营销Agent/数字员工 — 品牌侧

| 维度 | 内容 |
|------|------|
| **定位** | 产品化的网红营销/整合营销专用Agent |
| **目标客户** | 品牌方/广告主（先做品牌侧包装） |
| **产品形态** | 云端SaaS Agent + 私有化部署（客户服务器/电脑） |
| **底层** | 复用路线1的API能力，上层包装成完整工作流 |
| **商业模式** | 订阅制（$0-199/月PLG tier）+ 私有化部署费 |
| **优先级** | 次高（与路线1相辅相成） |

**对应调研**：
- **10.8 网红营销Agent（3.95）** — 直接对应，✅ R4完成，强烈推荐
- 10.3 达人Agent服务（3.90）— 品牌侧执行能力
- 24 数字员工全流程替代 — 待调研
- 02A 达人CRM（4.05）— Agent化的CRM
- 05 AI达人匹配（3.53）— Agent内置能力

**✅ 10.8 R4调研结论（2026-02-10）**：

定位从"全栈编排Agent"调整为"网红营销全流程Agent"后重新评估，评分3.125→**3.95**（+26%），当前所有方向最高分。

| 维度 | 结论 |
|------|------|
| **竞品格局** | 6个真Agent竞品2025年爆发（Stormy/Aha/AMT/图灵/Scrumball/CrowdCore），都极早期 |
| **差异化** | 数据飞轮(10年合作数据) + 客户分发(40万) + 私有化(竞品0覆盖) |
| **赢面** | vs Stormy 80%, vs AMT 75%, vs 图灵 70%, vs Aha 65% |
| **空白** | $80-169价位完全空白；履约管理/关系维护/归因无竞品覆盖 |
| **定价** | $0/29/99/199 四层PLG + 可选效果提成 |
| **MVP** | 7-8个月，8-10人团队，需新招3-4人（2 LLM工程师+1 AI PM+1数据工程师）|
| **PLG** | 72/100分，外联邮件带品牌露出=天然获客引擎 |
| **核心风险** | 品牌污名→必须新品牌；AI谈判→MVP只做辅助；零海外客户→Line 1先破局 |
| **TAM** | $8-10B（2025）→ $18-22B（2028）|
| **SOM** | 3年$25-50M ARR |

详细文档：`10.8_R4_最终评分.md` + 7个步骤文档（场景/竞品发现/竞品分析/痛点/优势/空白/PLG-难度-定价）

**与路线1的关系**：
- 路线1是基础设施层，路线2是应用层
- 如果超级Agent成为主流 → 路线1价值更大（被集成）
- 如果专用Agent成为主流 → 路线2价值更大（直接卖）
- 两条路对冲Agent形态不确定性

### 路线3：网红经纪人 — 达人侧

| 维度 | 内容 |
|------|------|
| **定位** | 帮网红更好地赚钱（聚星做经纪人角色） |
| **目标客户** | 创作者/网红（2C） |
| **产品形态** | 大概率是路线2的达人侧产品包装 |
| **商业模式** | 前期：免费+订阅；后期：平台抽佣 |
| **品牌** | 可能独立品牌（非聚星），择机决定 |
| **优先级** | 第三（大概率是路线2的另一种包装） |

**对应调研**：
- **13 帮达人赚更多钱（4.17）** — 核心方向之一，但具体产品定位需重新全面调研
- 14 Amazon网红生态（4.10）— 垂直场景
- 10.3 达人Agent服务（3.90）— Agent化的经纪人能力
- 12 达人接单撮合（2.88）— 需求存在但伪需求风险
- 11 达人媒体Kit（3.30）— 达人工具之一

**⚠️ 注意**：路线3的产品定位不等于13号方向（达人版Jungle Scout）。13号是调研中的一个可能形态，但路线3的产品定位需要基于"帮网红赚钱"这个大方向重新全面调研，不预设具体产品形态。

**关键发现（来自调研）**：
- 达人侧(2C)比品牌侧(2B)更有机会 — Top 2都是2C方向
- 46.8%创作者年收入<$500，工具需求强烈
- NoxInfluencer品牌在Reddit有scam关联 → 必须新品牌
- Beacons增长秘诀：病毒循环+5次点击onboarding
