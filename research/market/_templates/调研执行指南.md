| 工具 | 用途 | 示例 |

| `web_search` | 搜索竞品、市场数据、新闻 | `web_search "influencer CRM pricing"` |

## 二、单个方向评分流程

### Step 1: 竞品扫描（10-15分钟）

```
搜索关键词组合：
- "[方向] software/tool/platform"
- "[方向] pricing"
- "[方向] alternatives"
- "[方向] vs [竞品]"
```

**输出**：竞品清单（名称、定价、定位、融资）

### Step 2: 用户痛点挖掘（10分钟）

```
Reddit搜索：
- site:reddit.com "[方向] frustrating"
- site:reddit.com "[方向] hate"
- site:reddit.com "[方向] wish there was"
- site:reddit.com "[方向] alternative to [竞品]"
```

**输出**：3-5个真实用户痛点

### Step 3: JTBD分析（5分钟）

```
JOB: "当 [情境] 时，我想要 [动机]，这样我就能 [结果]"

📋 功能性：要完成什么任务？
💜 情感性：想要什么感受？
👥 社会性：想被如何看待？

ALTERNATIVES: 用户现在怎么解决？
UNDERSERVED: 哪部分没被满足？
```

### Step 4: 第一性原理检验（5分钟）

```
假设识别：我们在假设什么？
→ 挑战：为什么可能是错的？

基本真相：不可再分解的事实是什么？

重建方案：只基于基本真相，最简方案是什么？
```

### Step 5: 填写评分（5分钟）

打开对应目录的 `机会评分.md`，填写：
- 5维度分数（1-5）
- 理由和证据
- 聚星资源勾选
- 一句话摘要

## 三、5维度评分标准

### 1. 痛点强度（25%）

| 分数 | 标准 |
|------|------|
| 5 | 用户主动寻找方案，愿意付费，痛点频繁被提及 |
| 4 | 明确痛点，有付费意愿，但不紧急 |
| 3 | 有痛点但可忍受，"nice to have" |
| 2 | 痛点模糊，用户没意识到 |
| 1 | 伪需求，用户不关心 |

### 2. 竞争空白（30%）

| 分数 | 标准 |
|------|------|
| 5 | $0-199价格带完全空白，或现有方案极差 |
| 4 | 有1-2个竞品但明显不足 |
| 3 | 有竞品但差异化空间大 |
| 2 | 竞争激烈，需强差异化 |
| 1 | 红海，巨头主导 |

### 3. 聚星优势（25%）

| 分数 | 标准 |
|------|------|
| 5 | 直接利用C1-C3核心资源，竞品无法复制 |
| 4 | 利用C4-C6技术优势 |
| 3 | 可利用B1-B4资源 |
| 2 | 有间接关联 |
| 1 | 无明显优势 |

**聚星资源代号**：
- C1 付费客户（万级）
- C2 网红数据库（千万级）
- C3 品牌方数据
- C4 网红发现算法
- C5 大规模建联技术
- C6 AI Agent技术
- B1-B4 联盟/支付/CRM/物流（有但无优势）

### 4. PLG可行性（15%）

| 分数 | 标准 |
|------|------|
| 5 | 天然PLG，免费试用→自助付费，病毒传播 |
| 4 | 可PLG，需要一定销售辅助 |
| 3 | 混合模式，PLG+销售 |
| 2 | 主要靠销售，PLG困难 |
| 1 | 纯销售驱动 |

### 5. 开发难度（5%，反向计分）

| 分数 | 标准 |
|------|------|
| 5 | <2人月，聚星已有模块可复用 |
| 4 | 2-4人月，技术可控 |
| 3 | 4-6人月，有挑战 |
| 2 | 6-12人月，技术风险 |
| 1 | >12人月或技术不确定 |

## 四、批量执行策略

### 方式A：串行深度（推荐先做2-3个验证流程）

逐个方向完整执行，每个30-40分钟

### 方式B：并行广度

用 `sessions_spawn` 同时启动3-5个子Agent：
- 每个子Agent负责1个方向
- 完成后汇总到07文档

### 方式C：分类批处理

按相似性分组：
- 品牌侧工具：样品、CRM、竞品情报
- 达人侧工具：媒体Kit、接单、数据分析
- 特定市场：东南亚、跨境卖家
- 技术驱动：AI匹配、AI翻译

---

## 五、输出检查清单

每个方向完成后检查：

- [ ] 竞品清单有3-5个，含定价
- [ ] 用户痛点有证据来源
- [ ] JTBD三维度都填了
- [ ] 5维度都有分数和理由
- [ ] 聚星资源勾选了
- [ ] 一句话摘要写了
- [ ] 更新07汇总表
