# 聚星网红营销平台 - 产品文档中心

本仓库是聚星网红营销平台的产品文档中心，用于存放产品相关的各类文档资料。

## 目录结构

```
kol_docs/
├── research/                 # 调研文档
│   ├── market/              # 市场调研
│   ├── competitor/          # 竞品分析
│   └── user/                # 用户调研
├── prd/                      # 产品需求文档
│   ├── features/            # 功能需求
│   └── iterations/          # 版本迭代
├── design/                   # 产品设计文档
│   ├── interaction/         # 交互设计
│   ├── information-arch/    # 信息架构
│   └── flowcharts/          # 流程图
├── specs/                    # 规格说明
│   ├── api/                 # API 规格
│   ├── data/                # 数据字典
│   └── business-rules/      # 业务规则
├── strategy/                 # 产品策略
│   ├── roadmap/             # 产品路线图
│   ├── okr/                 # OKR 目标
│   └── positioning/         # 产品定位
├── operations/               # 运营相关
│   ├── campaigns/           # 营销活动方案
│   ├── kol-management/      # 网红管理策略
│   └── metrics/             # 运营指标
├── meeting-notes/            # 会议纪要
└── archive/                  # 归档文档
```

## 文档类型说明

### 调研文档 (research/)
- **市场调研**：网红营销行业趋势、市场规模、增长预测
- **竞品分析**：主要竞争对手功能对比、差异化分析
- **用户调研**：品牌方需求、网红痛点、用户画像

### 产品需求文档 (prd/)
- **功能需求**：各模块详细 PRD
- **版本迭代**：每个版本的需求汇总和变更记录

### 产品设计文档 (design/)
- **交互设计**：页面交互逻辑、操作流程
- **信息架构**：产品结构、导航设计
- **流程图**：业务流程、状态流转

### 规格说明 (specs/)
- **API 规格**：接口文档、数据格式
- **数据字典**：核心数据模型定义
- **业务规则**：结算规则、匹配算法、审核标准

### 产品策略 (strategy/)
- **产品路线图**：短期/中期/长期规划
- **OKR 目标**：季度目标与关键结果
- **产品定位**：核心价值主张、目标用户群

### 运营相关 (operations/)
- **营销活动方案**：平台活动策划
- **网红管理策略**：网红入驻、等级体系、激励机制
- **运营指标**：关键指标定义、数据看板

## 文档命名规范

- 使用日期前缀：`YYYY-MM-DD-文档名称.md`
- PRD 使用版本号：`PRD-模块名-v1.0.md`
- 调研报告：`调研类型-主题-日期.md`

## 贡献指南

1. 新建文档请放入对应目录
2. 更新文档需在文档头部记录修改日期和修改人
3. 重要决策需在会议纪要中留存记录
